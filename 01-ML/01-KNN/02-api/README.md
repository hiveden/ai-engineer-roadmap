### KNN 分类 API

```python
sklearn.neighbors.KNeighborsClassifier(n_neighbors=5)
```

n_neighbors : int, 可选（默认 = 5）

### KNN 回归 API

```python
sklearn.neighbors.KNeighborsRegressor(n_neighbors=5)
```
