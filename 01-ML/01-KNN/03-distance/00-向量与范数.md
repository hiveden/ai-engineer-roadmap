# 前置：向量与范数

> 本章后续的 L2 / L1 / L∞ / Lp 范数概念建立在这两个基础上。如果你熟悉，可跳过。

## 向量 vector

**向量** 是 **有大小和方向的量**。在直角坐标系中用一组**有序实数**表示：

- 一维：`[3]` —— 数轴上从原点到 3 的有向线段
- 二维：`[3, 4]` —— 平面上从原点指向 (3, 4) 的有向线段
- N 维：`[a₁, a₂, ..., aₙ]` —— N 维空间一个有方向的"箭头"

### 两种等价视角

| 视角 | 解读 | 例：`[3, 4]` |
|---|---|---|
| **点视角** | 向量 = 空间里一个具体的点 | 平面上 (3, 4) 这个点 |
| **箭头视角** | 向量 = 从原点指向某点的有向线段 | 从 (0, 0) 指向 (3, 4) 的箭头 |

→ 两种视角完全等价，按场景方便取用。

> ML 里"**特征向量** feature vector" = 一个样本的所有特征数字组成的向量。

## 范数 norm

衡量向量"大小"的方法 —— **给一个向量，算出一个非负实数**（代表它的"长度" / "模 modulus"）。

最简单的例子：**1 维数轴上 $\lvert x \rvert$（绝对值）就是范数** —— 告诉你 $x$ 离原点多远。

到 2 维 / N 维向量，有 **多种** 衡量"大小"的方式：

| 范数 | 公式 | 几何意义 | 对应距离 |
|---|---|---|---|
| **L2 范数** | $\sqrt{\sum x_i^2}$ | 直线长度（勾股定理）| 欧氏 |
| **L1 范数** | $\sum \lvert x_i \rvert$ | 各维度绝对值之和 | 曼哈顿 |
| **L∞ 范数** | $\max \lvert x_i \rvert$ | 最大维度的绝对值 | 切比雪夫 |
| **Lp 范数** | $(\sum \lvert x_i \rvert^p)^{1/p}$ | 上面三种的统一参数化 | 闵可夫斯基 |

## 图形化：二维例 `[3, 4]`

### 向量本身：点 = 箭头

```
 y
  4 ┤             ●(3, 4)
    │           ╱
    │         ╱
  3 ┤       ╱
    │     ╱
  2 ┤   ╱
    │ ╱
  1 ┤╱
    │
  0 ●─────────────► x
    0  1  2  3

  ●(3, 4) = 点视角
  ╱       = 箭头视角（从原点出发的有向线段）
```

→ **同一个向量 `[3, 4]`，既是平面上的点 (3, 4)，也是从原点指向它的箭头**。

### 三种范数的几何意义

向量 `[3, 4]` 在 3 种范数下"长度"不同：

```
 L2（欧氏 / 直线）：       L1（曼哈顿 / 网格）：
                                              
   ●(3, 4)                    ●(3, 4)
  ╱                           │
 ╱  √(3² + 4²) = 5            │ 竖 4
╱                             │
●(0, 0)                       ●─────●(3, 0)
                              0  横 3
                              
                              |3| + |4| = 7

 L∞（切比雪夫 / 最大维度）：
                                              
   ●(3, 4)
   ▲
   │ 取较大的那一段
   │ max(3, 4) = 4
   │
   ●(0, 0)
```

| 范数 | 含义 | `[3, 4]` 的值 |
|---|---|---|
| **L2** | 拉直线（勾股）| $\sqrt{3^2+4^2} = 5$ |
| **L1** | 走网格（横+竖）| $\lvert 3 \rvert + \lvert 4 \rvert = 7$ |
| **L∞** | 取最大维度 | $\max(3, 4) = 4$ |

→ **同一个向量，3 种范数算出 3 个不同"长度"** —— 这就是"换范数 = 换距离公式"的根源。

## 距离 = 两点差的范数

$$d(a, b) = \|a - b\|$$

→ 换范数 = 换距离公式。"L2 距离 = 欧氏距离 / L1 距离 = 曼哈顿距离"的本质来源。
