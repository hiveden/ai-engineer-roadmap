# 欧氏距离 Euclidean Distance

> ★ 掌握级别

## 底稿

欧氏距离 Euclidean Distance

---

## KNN 视角：默认怎么找邻居

KNN 找最近 K 个邻居，**默认就是用欧氏距离**——两点之间的直线距离（勾股定理 N 维推广）。

延续 [02 接近程度](../01-intro/02-接近程度.md) 豆瓣场景：流浪地球 3 (8.5, 9.5) 找邻居时，sklearn 默认用的就是这把尺子。

## 公式

$$d(a, b) = \sqrt{\sum_{i=1}^{n}(a_i - b_i)^2}$$

每多一维就加一项平方，最后开方。

## 实算

流浪地球 3 (8.5, 9.5) 到流浪地球 2 (8.3, 9.0)：

$$d = \sqrt{(8.5-8.3)^2 + (9.5-9.0)^2} = \sqrt{0.04 + 0.25} \approx 0.54$$

### 几何示意：KNN 找邻居用直线

```
       主演
       9.5 ┤  ●(8.5, 9.5) 流浪地球 3
           │   ╲
           │    ╲ ← 直线距离 ≈ 0.54
           │     ╲
       9.0 ┤      ●(8.3, 9.0) 流浪地球 2
           │
           └─────────────────► 评分
              8.3   8.5
```

→ KNN 算两点间距离时，欧氏 = "拉一根直线"。

## KNN 何时用欧氏

- **KNN 默认**：sklearn 通过 `metric='minkowski', p=2` 实现
- **特征量纲对齐 + 各维度同等重要** 时最合适
- **典型应用**：图像相似度（像素向量）、文本 embedding 相似度、推荐系统

## 关键启示

KNN 默认用欧氏，**绝大多数场景用它就够**。其他距离公式是特殊场景的优化。
