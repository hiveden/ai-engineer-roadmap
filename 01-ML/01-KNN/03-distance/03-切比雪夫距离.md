# 切比雪夫距离 Chebyshev Distance

> ○ 了解级别

## 课程原文

切比雪夫距离 Chebyshev Distance

---

## KNN 视角：某一维度差距独裁

如果你想让 KNN 的"近邻"判断 **只看差距最大的那个维度**，其他维度不算 —— 用切比雪夫。比较少见。

例如"色阶相似" 取 RGB 三通道差的最大值，决定两个颜色像不像。

## 公式

$$d(a, b) = \max_{i} \lvert a_i - b_i \rvert$$

取所有维度差的**最大值**（不求和、不开方）。

## 直觉：国王棋盘走法

国际象棋"国王"每步可走 8 个方向（横/竖/斜），一步覆盖 = 横向距离和纵向距离的较大者：

$$\text{国王从 A 到 B 最少几步} = \max(\lvert x_A - x_B \rvert, \lvert y_A - y_B \rvert)$$

也叫 **L∞ 距离 / 棋盘距离**。

### 几何示意：取最大维度差

延续豆瓣两点示例：流浪地球 3 (8.5, 9.5) 到流浪地球 2 (8.3, 9.0)：

```
       主演
           │  ●(8.5, 9.5)
           │  │ ← 维度 2 差: 0.5  ★ 取最大
           │  │
           │  ●━━━━━●(8.3, 9.0)
           │  ← 维度 1 差: 0.2
           └─────────────────► 评分

距离 = max(0.2, 0.5) = 0.5
```

→ 三种距离同两点对比：欧氏 0.54、曼哈顿 0.7、**切比雪夫 0.5**。

## KNN 何时用

- **任一维度差距决定相似性**（如色阶 / 仓储坐标）
- 别名 **L∞ 距离**

## 关键启示

了解即可。**KNN 工业场景比欧氏 / 曼哈顿少见很多**。知道存在 + 公式形态，以后看到能认出。
