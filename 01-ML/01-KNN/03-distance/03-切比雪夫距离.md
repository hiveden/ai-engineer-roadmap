# 切比雪夫距离 Chebyshev Distance

> **○ 了解级别**

## 课程原文

切比雪夫距离 Chebyshev Distance

---

## 公式

$$d(a, b) = \max_{i} \lvert a_i - b_i \rvert$$

**取所有维度差的最大值**（不求和、不开方）。

## 直觉：国王棋盘走法

国际象棋里"国王"每步可走 8 个方向（横/竖/斜），一步覆盖距离 = 横向距离和纵向距离的较大者：

$$\text{国王从 A 到 B 最少几步} = \max(\lvert x_A - x_B \rvert, \lvert y_A - y_B \rvert)$$

## 几何形态：单位圆是正方形

切比雪夫距离下，方程 $\max(\lvert x \rvert, \lvert y \rvert) = 1$ —— **边长 2 的正方形**：

```
(-1,1)●———————————●(1,1)
      |           |
      |           |
      |     +     |
      |           |
      |           |
(-1,-1)●———————————●(1,-1)
```

**为什么是正方形?** 只要某一维度差达到 1，距离就是 1，不管其他维度是 0 还是 0.99 —— **最大维度差独裁**。

## L∞ 范数视角

切比雪夫距离也叫 **L∞ 距离**，对应 **L∞ 范数**：

$$\|x\|_\infty = \max_i \lvert x_i \rvert$$

## 何时用

- **任一维度差距决定相似性** 时：例如"色阶相似"取 RGB 三通道的最大差
- **棋盘 / 仓储 / 物流**：平台允许斜向移动时
- 别名 **L∞ 距离 / 棋盘距离**

## 与前两种的关系

| 距离 | 公式核心 | 范数 |
|---|---|---|
| 欧氏 | 平方求和再开方 | L2 |
| 曼哈顿 | 绝对值求和 | L1 |
| **切比雪夫** | 绝对值的 **最大值** | **L∞** |

## 关键启示

了解即可。**实际工业场景比欧氏 / 曼哈顿少见很多**。知道存在 + 公式形态，以后看到能认出。
