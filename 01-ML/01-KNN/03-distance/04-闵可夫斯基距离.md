# 闵可夫斯基距离 Minkowski Distance

> ○ 了解级别

## 底稿

闵可夫斯基距离 Minkowski Distance

---

## KNN 视角：sklearn 实际默认就是它

sklearn KNN 的 `metric` 参数默认值就是 `'minkowski'`，通过参数 `p` 控制具体形态：

```python
KNeighborsClassifier(n_neighbors=5, metric='minkowski', p=2)  # 默认 = 欧氏
KNeighborsClassifier(n_neighbors=5, metric='minkowski', p=1)  # = 曼哈顿
```

**所以你以为在用欧氏，其实 sklearn 是在用闵可夫斯基（p=2 退化）**。

## 公式

$$d(a, b) = \left( \sum_{i=1}^{n} \lvert a_i - b_i \rvert^p \right)^{1/p}$$

通过参数 $p$ 统一前面三种距离。

## p 的退化形式

| $p$ | 退化为 |
|---|---|
| **p=1** | **曼哈顿距离** |
| **p=2** | **欧氏距离** |
| **p=∞** | **切比雪夫距离** |

→ 闵可夫斯基不是"第 4 种新距离"，是 **欧氏 / 曼哈顿 / 切比雪夫的统一参数化**。

### 同两点 3 种 p 对比

豆瓣两点 (8.5, 9.5) 到 (8.4, 9.0)：

| `p` | 计算方式 | 距离 |
|---|---|---|
| **p=1**（曼哈顿）| $\lvert 0.1 \rvert + \lvert 0.5 \rvert$ | **0.6** |
| **p=2**（欧氏）| $\sqrt{0.1^2 + 0.5^2}$ | **0.51** |
| **p=∞**（切比雪夫）| $\max(0.1, 0.5)$ | **0.5** |

→ 同样两点，不同 p 给出不同距离。**改 p 参数 = 换距离公式**。

## 关键启示

KNN 实际用时**直接调 `p` 参数**即可——不用记四个 metric 名字。sklearn 默认 `p=2`（欧氏），改 `p=1` 就变曼哈顿。
