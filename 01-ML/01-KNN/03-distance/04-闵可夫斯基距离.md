# 闵可夫斯基距离 Minkowski Distance

> **○ 了解级别**

## 课程原文

闵可夫斯基距离 Minkowski Distance

---

## 公式

$$d(a, b) = \left( \sum_{i=1}^{n} \lvert a_i - b_i \rvert^p \right)^{1/p}$$

**通用参数化形式** —— 通过参数 $p$ 统一前面三种距离。

## 关键性质：参数 p 决定退化形式

| $p$ | 退化为 | 公式形态 |
|---|---|---|
| **p=1** | **曼哈顿距离 L1** | 绝对值求和 |
| **p=2** | **欧氏距离 L2** | 平方求和再开方 |
| **p=∞** | **切比雪夫距离 L∞** | 取最大维度差 |

→ 闵可夫斯基不是新距离，是 **欧氏 / 曼哈顿 / 切比雪夫的统一参数化**。

## 几何形态：形态光谱

随着 $p$ 从 1 增到 ∞，单位圆的形状 **连续变形**：

```
p = 1 (L1)        p = 2 (L2)       p = ∞ (L∞)

    ◇                ○                □

  菱形             圆形             正方形
  (顶点贴轴)       (饱满)           (顶点贴角)
```

中间值的形态：

- p < 1：单位圆 **凹陷**（向轴线收缩，星形）—— 严格意义上不再是范数（违反三角不等式）
- 1 < p < 2：菱形向圆"鼓出"（顶点钝化）
- p > 2：圆向正方形"挤压"（顶点贴角）

→ **Lp 单位圆是连续的形态光谱**，p 是"形状光谱的旋钮"。

## Lp 范数

闵可夫斯基距离对应的范数是 **Lp 范数**：

$$\|x\|_p = \left( \sum_i \lvert x_i \rvert^p \right)^{1/p}$$

L1 / L2 / L∞ 都是 Lp 在不同 $p$ 值下的特例。

## 在 sklearn 中

`metric='minkowski'` 是 sklearn KNN 的实际默认值，通过参数 `p` 控制：

```python
KNeighborsClassifier(n_neighbors=5, metric='minkowski', p=2)  # 默认 = 欧氏
KNeighborsClassifier(n_neighbors=5, metric='minkowski', p=1)  # = 曼哈顿
```

## 关键启示

**闵可夫斯基不是新东西**，是欧氏 / 曼哈顿 / 切比雪夫的统一参数化形式。理解这个关系即可，实际用时直接调 `p` 参数。
