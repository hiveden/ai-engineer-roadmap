**实际 UI**（按代码读，不要凭空想象）：
- **左上图（距离路径）**：A（红圆）+ B（蓝圆）两点 + 三条物理路径
  - 紫色 L 形 = 曼哈顿
  - 绿色直线 = 欧氏
  - 红色虚线对角段 = 切比雪夫（先对角后单走）
- **右上图（等距线）**：以 A 为圆心 4 条等距线
  - 蓝色实线 = 当前 p 的闵可夫斯基等距集
  - 紫绿红淡虚线 = L1（菱形）/L2（圆）/L∞（方）参考
- **底部演化全谱图**：固定半径 1，原点为圆心，p ∈ {0.5, 1, 1.5, 2, 3, 5, 20} 七条曲线 → 形状从星形 → 菱形 → 圆 → 圆角方 → 标准方
- **5 个滑块**：A.x / A.y / B.x / B.y / 闵可夫斯基 p（0.5–10, step=0.1）
- **实时距离卡片**：4 行彩色文字（紫曼/绿欧/红切/蓝闵），每行带计算过程

**关键数字（已 Python 实测）**：A=(2,3), B=(8,7) 默认值时
- 曼哈顿 6+4=10；欧氏 √(36+16)≈7.21；切比雪夫 max(6,4)=6
- 但脚本主线沿用 E01 豆瓣两点 (8.5,9.5)→(8.4,9.0)：曼 0.60 / 欧 0.51 / 切 0.50（演示时把滑块拖到这一对）
