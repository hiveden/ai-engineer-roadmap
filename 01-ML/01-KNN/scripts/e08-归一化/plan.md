## Demo 片段

- `01-KNN/04b-scaling/demos/02-knn-scaling.py`（**部分场景**：MinMax + 异常值开关）

**实际 UI**（按代码，不要凭空想象）：
- **2 个控件区**：
  - 左：模型参数 → k 滑块 (1-7) / 缩放下拉（None / MinMaxScaler / StandardScaler）/ 异常值开关
  - 右：新人特征 → 身高 (160-195) / 体重 (50-130) / 视力 (0.1-2.0) 三滑块
- **预测 callout 卡片**：彩色横条，绿底=健康 / 红底=不健康，显示「健康 X / 不健康 Y · k=N · 缩放方式 · 异常」
- **散点图（matplotlib，1 行 2 列 figsize=(8,3)）**：左=Raw（原始 height vs weight），右=After {缩放方式}。每个图：训练点圆形彩色（绿=健康/红=不健康），新人=蓝星
- **距离明细表**：8 行（异常值开关下 9 行）按距离升序，前 k 行黄色高亮
- **缩放参数表**（accordion 展开）：
  - MinMax：身高/体重/视力的 min / max / 跨度
  - Standard：μ / σ
- **缩放前后对比表**：每个样本原值 vs 缩放后值

**关键场景（Python 实测）**：
- 默认 (173, 80, 1.0)，k=3，None：top-3 都是健康，但距离主导被体重占
- 量纲独裁演示 (175, 85, 1.7)，k=3，None：top-3 进 2 个不健康（被体重独裁）
- 同一新人切 MinMax：top-3 全是健康
- 异常值开关 ON + MinMax：体重列 min=65/max=250/range=185，正常样本被压到 [0, 0.243]

## 砍掉

- 教材 mermaid 钟形分布平移 / 缩放可视化 → 转 Remotion 双格动画（钟形 + 离散点），不读数据

- "考试折算 100 分制" 类比 → 留一句直觉，不展开三科满分
- 公式严格写法 `x'' = x' * (mx - mi) + mi`（非 [0,1] 区间映射）→ 不进视频，默认区间 [0,1]
- 教材"两步缺一不可"对照表 → 用直觉口播代替

## 口吻提示

- 公式 `x' = (x − min) / (max − min)` 念成「减最小值除以跨度」，避免读符号
- "平移 + 缩放两步"用直觉锚：先把起点对齐到 0（减 min），再把跨度压到 1（除 max−min）
- demo 段先演示「不缩放被独裁 → 缩放后翻转」，再演示「异常值绑架」副作用 —— 两个对比强烈的状态切换
- 数字不要密集念，用 demo 静帧停顿
- cta 末句："欢迎在评论区指正。下一期我们看第二种缩放方法：标准化，先讲清楚什么是高斯分布。"
