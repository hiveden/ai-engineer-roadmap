#### Slide 56 · 一元线性回归 – 解析解计算

**PPT 内容**：
```
损失函数：L(k,b) = 145416k² + 852b + ... (对k,b的二元二次方程)
固定 b=-100：
  145416k² - 281671.6k + 136496.32 = 0
  k = -(-281671.6) / (2×145416) = 0.9685
```

**README 状态**：✓ 一致但有差异
- README 给出的公式：

  - $\bar{x}=170.4, \bar{y}=65.1$

- **结论**：数值不同但含义一致——两者都通过解二次方程得到最优参数

**PPT 内容**：
- Slide 83-86：MAE、MSE、RMSE 三种指标的定义、公式、API
  - MAE：$\frac{1}{n}\sum |y_i - \hat{y}_i|$（单位同 y，不敏感异常值）

### 5b. 正则化详解（Slide 111-120）→ 07b-regularization

#### Slide 111 · L1 正则化（Lasso）

**PPT 内容**：
```
L = L(W) + λ · Σ|w_i|
梯度：∂L/∂w_i = ∂L(W)/∂w_i + λ·sign(w_i)
权重更新：w_i ← w_i - α·∂L(W)/∂w_i - α·λ·sign(w_i)
```

**关键点**：
- "等量收缩"：$w_i > 0$ 时减常数，$w_i < 0$ 时加常数
- 一旦 $w_i = 0$，就保持为 0（稀疏解）
- Lasso API：`from sklearn.linear_model import Lasso`

**README 状态**：✓ 完整且更深入
- 07b-regularization/README.md 给出相同的更新规则
- 补充了"几何直觉"：L1 等高线是菱形，与椭圆相切在顶点 → 稀疏解
- 提及 $\alpha$ 越大正则越强（trade-off: 训练误差↑，测试误差先↓再↑）

**评估**：OK

---

#### Slide 112 · L2 正则化（Ridge）

**PPT 内容**：
```
L = L(W) + λ · Σ w_i²
梯度下降更新：w_i ← w_i - α·∂L(W)/∂w_i - 2α·λ·w_i
                  = (1 - 2αλ)w_i - α·∂L(W)/∂w_i
```

**关键点**：
- "按比例收缩"：每次乘以 < 1 的因子，权重逐渐趋于 0 但**不直接归零**
- L2 vs L1 的本质差异：比例衰减 vs 等量衰减
- Ridge API：`from sklearn.linear_model import Ridge`

**README 状态**：✓ 完整
- 给出相同的更新公式
- 注明"不会直接归零"的特性

**评估**：OK
