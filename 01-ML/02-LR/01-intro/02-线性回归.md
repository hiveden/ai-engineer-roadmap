# 线性回归

> 级别:【理解】
> 维度:概念
> 章节底稿全文见 [`README.md`](./README.md)（PPT slide 7）

## 底稿

> 利用回归方程（函数）对**一个或多个特征值和标签之间**关系进行建模的一种分析方式。当这个模型是线性时，称为线性回归。

注意事项：

1. **为什么叫线性模型？** 因为求解的 w 都是 w 的零次幂（常数项）所以叫成线性模型
2. 在线性回归中，从数据中获取的规律其实就是学习权重系数 w
3. 某一个权重值 w 越大，说明这个权重的数据对预测结果影响越大

---

## 直觉

一句话：**y 是 x 的"加权和" + 一个偏置** —— 没有平方、没有乘积、没有 log，只有加和乘常数。

公式：

$$y = w_1 x_1 + w_2 x_2 + \cdots + w_n x_n + b$$

每个特征 $x_i$ 配一个权重 $w_i$，所有项相加再加偏置 b。**这个加法结构就是"线性"的本质**。

## 从"斜率和截距"迁移到"权重和偏置"

你初中已经熟了 $y = kx + b$（一次函数）：

- **k 斜率**：x 每增加 1，y 跟着变 k（直线倾斜得多陡）
- **b 截距**：x 为 0 时 y 等于 b（直线和 y 轴的交点）

LR 就是把这个**升级**到多个 x：

$$y = w_1 \cdot x_1 + w_2 \cdot x_2 + \cdots + w_n \cdot x_n + b$$

### 几何画面：1 元一根斜率 / 多元每个特征一根斜率

```
一元（1 个特征 x）：拟合一根直线
  y │
    │      ╱
    │    ╱   ← k 是这根线的「陡」
    │  ╱
  b ●
    └────────── x
    0

二元（2 个特征 x₁, x₂）：拟合一个平面
  y │
    │   ╱╱╱╱╱╱
    │  ╱平 面 ╱   ← 沿 x₁ 方向的「陡」 = w₁
    │ ╱╱╱╱╱╱   ← 沿 x₂ 方向的「陡」 = w₂
  b ●
    └─── x₁ ↗
  x₂ ↗

三元及以上：4 维空间画不出来，但代数一样——
每个特征都有自己的「陡」(w_i)，加在一起 + 一个共享的偏置 b
```

### 旋钮直觉：参数数量随特征增长

```
一元：    y = k·x + b
              ↑     ↑
            1 个旋钮 1 个旋钮

二元：    y = w₁·x₁ + w₂·x₂ + b
              ↑       ↑       ↑
              2 个旋钮         1 个旋钮

n 元：    y = w₁·x₁ + w₂·x₂ + ... + wₙ·xₙ + b
                  ↑     ↑           ↑       ↑
                  n 个独立旋钮              1 个

→ 权重旋钮每加一个特征就 +1；偏置永远只有 1 个。
```

迁移两条要点：

### ① 一元的 k → 多元的 w 向量

每个特征都有自己的"k"——一元一根斜率，多元变成一组 $(w_1, w_2, \ldots, w_n)$，叫**权重**。

- 一元里 k 表示「x 每变 1，y 变多少」
- 多元里 $w_i$ 表示「**只动第 i 个特征 $x_i$，其他不动**，y 跟着变多少」

→ 每个特征都有一个**独立的"k"**，每个特征对 y 的影响互不干扰，全靠加法叠加。

### ② b 还是 b（含义不变）

- 一元 $y = kx + b$：b 是直线和 y 轴的交点，x = 0 时 y = b
- 多元 $y = w_1 x_1 + \cdots + b$：b 是所有特征都为 0 时 y 的值

含义完全一样：**和具体特征无关、所有样本共享的那一笔**。

只是几何画面变了：一元能画图（线和 y 轴交点），4 维以上画不出来，但代数含义一字不差。

### 为什么 k 要改名叫"权重"，b 要改名叫"偏置"

| 一元（数学课）| 多元（ML）| 改名原因 |
|---|---|---|
| 斜率 k | 权重 $w_i$ | k 是几何概念（直线斜度），多元高维画不出来。借了统计学"加权求和"里的 weight，每个特征的"权" |
| 截距 b | 偏置 b | "y 轴交点"在高维没法看。借了电子学/神经科学的 bias（不依赖输入的固定偏移）|

但 sklearn 属性**仍然叫 `intercept_`**（保留数学传统），不叫 `bias_`。所以同一个 b 你会看到三种称呼：

| 数学课视角 | 工程 / sklearn | ML / 神经网络 |
|---|---|---|
| 斜率 k | 系数 `coef_` | 权重 weight |
| 截距 b | 截距 `intercept_` | 偏置 bias |

→ 三套词指同一组东西，**本笔记交替使用，知道指的是 (k, b) 那一对就行**。

## 工资场景：把 w 和 b 落到具体业务

把多元 LR 翻译到打工算工资：

> 月薪 = **50·加班时数** + **200·签单数** + **100·工龄年** + **5000**

- "加班一小时加 50"、"签单一单加 200"、"工龄一年加 100"——每件事的"汇率"，对应 $w_1, w_2, w_3$
- "底薪 5000"——所有人都先有的那笔基础，对应 b

**三个特征 → 三个权重；不管特征多少，永远只有一个偏置**。

### 调一调这些数字会发生什么（带几何对照）

| 改动 | 工资场景后果 | 几何对照（数学课视角） |
|---|---|---|
| 加班汇率 50 → 500 | 加班价值翻 10 倍，所有人疯狂加班 | k 调大 10 倍，直线变陡 |
| 加班汇率 50 → -50 | 越加班越扣钱，下班准点跑路 | k 由正变负，直线由上倾改为下倾 |
| 底薪 5000 → 0 | 全员从零起步，纯看业绩 | b = 0，直线必过原点 |
| 底薪 5000 → 8000 | 所有人月薪**同步涨 3000**，但**人和人之间的差距没变** | b 加 3000，直线**整体上移**，斜率不变 |

→ **改 w 改的是"陡不陡"（斜率）**，**改 b 改的是"高不高"（截距）**——一元几何的直觉**一字不改**地迁移到多元。

### 一句话区分

- **权重 $w_i$ = 第 i 个特征对 y 的"斜率"**——决定不同样本的差距怎么拉开（每件事多值钱）
- **偏置 b = 所有特征都为 0 时 y 的值**——所有样本共享的起跑线

## "线性"到底说的是谁线性

容易混淆：什么对什么是线性？

| 候选 | 是否线性 |
|---|---|
| **y 对 w**（权重）| ✓ 是的——这才是"线性回归"的本意 |
| **y 对 x**（特征）| ✓ 通常也是，但不是必须 |
| **w 对 w 自己的高次幂**（如 $w^2$）| ✗ 没有这种东西，w 永远只出现一次（一次幂）|

**关键点**：模型对**权重 w**是线性的，所以叫"线性"模型。即使你拿 x 做平方再喂进去（如 $y = w_1 x + w_2 x^2 + b$），它仍是线性回归——因为 $x^2$ 只是一个新特征，w 还是一次幂。这就是 [`03-线性回归分类`](./03-线性回归分类.md) 提到的**多项式线性回归**。

## 权重的工程含义（房价例子）

$y = w_{\text{面积}} \cdot \text{面积} + w_{\text{楼层}} \cdot \text{楼层} + \cdots + b$

- $w_{\text{面积}} = 5000$ → 每多 1 平米，房价增加 5000 元
- $w_{\text{楼层}} = 200$ → 每高 1 层，房价增加 200 元（影响小）
- $w_{\text{朝向}} = -10000$ → 朝向编码 +1 反而降价 1 万（朝向编码方向反了）

**绝对值大 = 该特征影响大**；**符号 = 正/负相关**。这是 LR 模型最大的优势：**可解释性强**。

## 与其它算法的对比

| 算法 | 模型类型 | 训练 | 预测 | 可解释性 |
|---|---|---|---|---|
| **KNN** | 非参数（lazy） | 不算参数，只存数据 | O(N) 算所有距离 | 弱（黑盒） |
| **线性回归** | 参数模型 | 算 n+1 个 w（一次性） | O(n) 内积 | **强**（看 w 大小符号）|
| **决策树** | 参数模型 | 递归找最佳分裂 | O(log N) | 中（看 if-else 树）|
| **神经网络** | 参数模型 | 梯度下降迭代 | O(参数量) | 弱（黑盒） |

LR 是 **简单、快、可解释** 的代表。任何回归问题先用 LR 跑一遍做 baseline，比这条线还差的复杂模型不值得上。

## 公式速查

$$\boxed{\hat{y} = \mathbf{w}^T \mathbf{x} + b}$$

- $\mathbf{x} \in \mathbb{R}^n$：n 个特征组成的列向量
- $\mathbf{w} \in \mathbb{R}^n$：n 个权重组成的列向量（每个特征一个）
- $b$：偏置（标量）
- $\hat{y}$：预测值（标量，回归问题中是连续数）

矩阵化（m 个样本一起预测）：

$$\hat{\mathbf{y}} = X \mathbf{w} + b \mathbf{1}$$

其中 $X \in \mathbb{R}^{m \times n}$ 把 m 个样本的特征堆成矩阵；常用技巧把 b 吸收进 w（X 多加一列全 1）：

$$\hat{\mathbf{y}} = X \mathbf{w}, \quad X \in \mathbb{R}^{m \times (n+1)}, \quad \mathbf{w} \in \mathbb{R}^{n+1}$$

这个矩阵化形式是 [`04a-analytical`](../04a-analytical/) 正规方程的基础。
