> 【实操】线性回归 API 的应用

预测播仔身高。

已知数据：⚠️ 待补充（身高 / 体重对照表）

需求：播仔身高是 176，请预测体重？

```python
"""
   线性回归解释：
        描述一个或多个自变量和因变量之间的关系，基于这种关系的模型，叫线性回归模型
   分类：
        一元线性回归：
            一个自变量和一个因变量之间的关系
        多元线性回归：
            多个自变量和一个因变量之间的关系
   公式：
         一元线性回归：
            y = kx + b (k 是斜率，b 是截距)
        多元线性回归：
            y = w1·x1 + w2·x2 ... + b (w 是权重（回归系数），b 是偏置（截距）)

步骤
1 准备数据集  身高和体重
2 实例化线性回归模型
3 训练，用线性模型拟合训练集数据分布规律  身高 = w*体重 + b
   求得 w 和 b 的最佳值
4 模型预测（播仔体重）

"""
# 导包
from sklearn.linear_model import LinearRegression  # 线性回归

# 1 准备数据集  身高和体重
x = [[160], [166], [172], [174], [180]]   # 训练集的特征列  身高
y = [56.3, 60.6, 65.1, 68.5, 75]          # 训练集的标签列  体重

# 只有一个特征，不做标准化

# 2 实例化线性回归模型
model = LinearRegression()
# 正规方程解法    梯度下降法
# 在这个 API 里面，求解方法为  SVD  奇异值分解法

# 3 训练，用线性模型拟合训练集数据分布规律  身高 = w*体重 + b
# 输入为 训练集特征和标签
model.fit(x, y)

# 打印模型参数
print('w:', model.coef_)        # coefficient 系数 = 斜率 slope = 权重 weight
print('b:', model.intercept_)   # 截距 b

# 求得 w 和 b 的最佳值
# 4 模型预测（播仔身高）
output = model.predict([[176]])

print('预测结果：', output)
```
