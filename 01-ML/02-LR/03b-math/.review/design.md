# 03b-math Demo 设计文档

## 1. 教学目标

学完本 demo，使用者能够：

1. **认识 MSE 损失函数的含义**：理解每个红色方块代表一个样本的平方误差，MSE = 所有方块面积的平均值
2. **理解参数对损失的影响**：通过拖滑块改变 k（斜率）和 b（截距），看到红方块和 loss 值实时变化
3. **建立 1D 直觉**：固定一个参数，看 loss 关于另一参数的抛物线形状，认识到极值的存在
4. **掌握最优模型的概念**：看到绿色钻石（最优参数对应的点）是拟合直线最好的位置
5. **区分欠拟合与过拟合（初级）**：通过预设对比，感受不同参数下的拟合效果差异

## 3. 视图布局（A + B 双视图模式）

```

├─────────────────────────────────────────────────────────┤
│  【预设选择器】【帧滑块】 · 预设方案 + 回放            │

```

### 详细视图说明

**A 视图（左上）**：数据点 + 拟合直线 + 残差方块
- 数据：5-10 个合成点（身高-体重）
- 拟合线：当前 k、b 对应的 y = kx + b，红色粗线
- 最优线：闭式解计算的最优 k\*、b\*，绿色虚线（作参照）
- 残差方块：mark_square，size 编码 (y_true − y_pred)²，红色半透明
- Size 编码 range=[50, 3000]

**B1 视图（右上）**：Loss vs k 抛物线
- 场景：b 固定为当前值 cur_b，k 在 [-2, 6] 范围扫描
- 抛物线：蓝色线 + 当前红点 + 最优绿钻石

**B2 视图（左下）**：Loss vs b 抛物线
- 同 B1 结构，颜色改紫色

### 联动逻辑
单点输入原则：滑块（k、b）= 唯一用户输入源；预设模式时滑块隐藏

## 5. 数据生成

```python
np.random.seed(42)
x_data = np.array([160, 166, 172, 174, 180])  # 身高
w_true, b_true = 0.97, -95
y_data = w_true * x_data + b_true + np.random.normal(0, 1.2, 5)
```

样本量：5（对应 PPT slide 24）

最优参数：

$$k^* = \frac{\sum (x_i - \bar{x})(y_i - \bar{y})}{\sum (x_i - \bar{x})^2}, \quad b^* = \bar{y} - k^* \bar{x}$$

## 6. 预设方案（6 个）

| 标签 | k0 | b0 | 场景 |
|---|---|---|---|
| ✋ 手动 | — | — | 用户用滑块 |
| ✓ 完美 | 0.95 | -90 | loss 极小，方块几乎消失 |
| 📍 欠拟合 k 偏小 | 0.4 | 0 | loss 中等，方块明显 |
| 📍 欠拟合 b 偏大 | 1.0 | 3.0 | 同上 |
| 🎯 远距长征 | -1.5 | 8.0 | loss 巨大，方块遍布 |
| 🚫 反向 k 负值 | -0.8 | 2.0 | 拟合反向 |

## 7. 5 秒测试预演

| 判断点 | 视觉信号 | 5 秒理解 |
|---|---|---|
| 红方块的含义 | A 视图大小不一的红方块 | "代表某种误差量度" ✓ |
| 滑块作用 | 拖 k 滑块拟合线和抛物线立刻变 | "旋钮改变直线和 loss" ✓ |
| 最优解位置 | 绿钻石标记 B 抛物线谷底 | "绿点是特别的最优位置" ✓ |
