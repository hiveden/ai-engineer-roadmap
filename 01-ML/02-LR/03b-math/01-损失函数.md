#### Slide 19 · 引入评判标准 – loss function 损失函数

- **误差**：每个数据点预测值 ŷ − 真实值 y（用来计算损失函数）
- **损失函数**：综合衡量"模型预测结果"和"真实值"之间的整体差异
- 也叫代价函数、成本函数、目标函数

```
  |   误差1│  /
  |        ●预测值
  +────────┴───────┴───────┴─────────→ 身高 X
                  172      174
```

#### Slide 20 · 损失函数数学表达

损失函数综合衡量"预测结果"与"真实值"的整体差异，即评估模型性能。

- **平均绝对误差 MAE**（Mean Absolute Error）：$\text{loss}(w,b) = \dfrac{1}{m}\sum_{i=1}^{m}|\hat y_i - y_i|$
  - 第 i 个误差的绝对值
- **均方误差 MSE**（Mean-Square Error）：$\text{loss}(w,b) = \dfrac{1}{m}\sum_{i=1}^{m}(\hat y_i - y_i)^2$
  - **对误差的平方惩罚！！** 不需要绝对值
  - **不建议用在有异常值的数据**

> **备注**：对误差的平方惩罚——怕模型优化时为了降低 loss 去拟合异常值。

#### Slide 21 · 损失函数（结论）

- 结论：损失函数越小，"模型预测结果"越接近"真实值"，模型性能越好
- 红色直线在每个数据点上有更小的误差、更小的 loss 值，因此它是更好的线性模型，能更好地描述数据分布规律
- **最小二乘**：平方和误差

```
                红线（loss 小，更优）            蓝线（loss 大，差）
  ●───●─●───●         vs                 ●  ●  ●
        红线贴近所有点                     ────────  蓝线偏离
```

#### Slide 22 · 求解最优模型的套路

- 最优模型 ⇔ 最小的损失函数
- 通过损失函数最小化，求解 w、b → 获得最优模型参数
- 最优模型：**y = wx + b**

```
最小化 loss  ──►  解出 (w*, b*)  ──►  最优模型 ŷ = w*x + b*
```

#### Slide 23 · 损失函数计算（简化）

样本数据（5 个：身高 → 体重）：

| 序号 | 身高 x | 真实体重 y |
|---|---|---|
| 1 | 160 | 56.3 |
| 2 | 166 | 60.6 |
| 3 | 172 | 65.1 |
| 4 | 174 | 58.5（PPT 原值，备注与 slide 56 用 **68.5**） |
| 5 | 180 | 56.3（PPT 原值，备注与 slide 56 用 **75**） |

> 注：PPT slide 23/24 正文 5 个 y 值（56.3, 60.6, 65.1, 58.5, 56.3）属录入错误。slide 24 备注、slide 55 平方和、**slide 56 推导用的 56.3, 60.6, 65.1, 68.5, 75 才是与最终系数 281671.6 / 145416 / 852 / 55683.8 / 325.5 自洽的"官方答案"**。

假设 y = kx + b，损失函数：

$$L(k,b) = (160k+b-56.3)^2 + (166k+b-60.6)^2 + (172k+b-65.1)^2 + (174k+b-58.5)^2 + (180k+b-56.3)^2$$

损失函数是关于 k、b 的二元二次方程，展开复杂。**为简化计算，先固定截距 b = -100**：

$$L(k, b{=}{-100}) = (160k-156.3)^2 + (166k-160.6)^2 + (172k-165.1)^2 + (174k-158.5)^2 + (180k-156.3)^2$$

#### Slide 24 · 求损失函数最小值（求最优解）

逐项展开：每个 $(ax - c)^2 = a^2 k^2 - 2ac\,k + c^2$，再求和合并 k² 系数 / k 系数 / 常数项。

```
k² 系数：160² + 166² + 172² + 174² + 180²
       = 25600 + 27556 + 29584 + 30276 + 32400
       = 145416

k 一次项系数（取自 PPT 备注，y 用 156, 160.6, 165.1, 168.5, 175）：
  2·160·156 + 2·166·160.6 + 2·172·165.1 + 2·174·168.5 + 2·180·175
  = 49920 + 53319.2 + 56794.4 + 58638 + 63000
  = 281671.6   →  作为 −281671.6 k

常数项：156² + 160.6² + 165.1² + 168.5² + 175²
       = 136496.32

⇒  L(k, b=-100) = 145416 k² − 281671.6 k + 136496.32
```

二次函数顶点：k* = −b/(2a)，导数为 0 时取得最小值：

$$k^* = \frac{281671.6}{2 \times 145416} = \mathbf{0.9685}$$

截距 b = −100 时，预测身高 176 的小明体重：

$$\hat y = 0.9685 \times 176 - 100 = \mathbf{70.456}$$

> 总结：求最好拟合线 y = kx + b ⇒ 引入损失函数 Loss(k, b) ⇒ 优化方法求最小值得 k 最优解

### 笔记

> 【掌握】损失函数

需要设置一个评判标准。

**误差概念**：用预测值 ŷ – 真实值 y 就是误差。

**损失函数**：衡量每个样本预测值与真实值效果的函数。

"红色直线能更好的拟合所有点" 也就是误差最小，误差和最小。

想求一条直线更好的拟合所有点 y = kx + b：
- 引入损失函数（衡量预测值和真实值效果）Loss(k, b)
- 通过一个优化方法，求损失函数最小值，得到 K 最优解

回归的损失函数：
- **均方误差**（Mean-Square Error, MSE）
- **平均绝对误差**（Mean Absolute Error, MAE）

## ━━━━━━━━ 讲解 ━━━━━━━━

### 直觉

**生活锚——猜体重比赛**：你和朋友街头猜陌生人体重，主持人定规矩——**每猜偏 1 公斤罚 1 块钱**。5 个人猜下来，谁罚款总额最少谁赢。

把这个「罚款规则」搬到 LR 上：

- **罚款规则** = 损失函数 loss function（也叫代价 cost / 目标 objective）
- **每次猜测的偏差** = 单点误差 $\hat y_i - y_i$
- **罚款总额** = 整条线的 Loss
- **赢家的猜法** = 最优参数 $(w^*, b^*)$

**损失函数 = 给「这条线拟合得多差」打一个罚款分数；罚得越少，模型越好。**

模型训练就一句话：在所有可能的 $(w, b)$ 里，挑那组让罚款最少的——这就是 [`04a-analytical/`](../04a-analytical/) 正规方程和 [`04b-gd/`](../04b-gd/) 梯度下降在干的事。

```
体重 y (kg)                                  误差 = ŷ - y
   ↑                              ●(180, 75)    ← 真实
75 │                          ╱   ↕ +5
   │                       ╱      ─ŷ=70 (预测)
70 │                    ╱
   │                ●(172, 65) ← 真实
65 │             ╱  ↕ +1
   │          ╱─────ŷ=66 (预测)
60 │       ●(166, 60) ← 真实
   │     ╱  ↕ +2
   │  ╱──── ŷ=62 (预测)
56 │●(160, 56) ── ŷ=58 (预测) ↕ +2
   └─────────────────────────────────→ 身高 x (cm)
     160    166      172       180

   ●  真实体重   ─  当前模型 ŷ = wx + b   ↕  单点误差（=单次罚款）
   损失函数 = 把所有 ↕ 综合成一个总分
```

> 罚款规则不止一种：偏 1 公斤罚 1 块（线性 → MAE），还是偏 1 公斤罚 1 块、偏 2 公斤罚 4 块（平方 → MSE）？规则不同，赢家可能不一样——这是下面「为什么用平方而不是绝对值」要回答的。

### 误差 vs 损失：单点 vs 整体

PPT 把概念分两层，得讲清楚——容易混淆：

| 层级 | 对象 | 定义 | 例子（上图）|
|---|---|---|---|
| **误差 error** | 单个样本 | $\hat y_i - y_i$（每点一个数）| +2, +2, +1, +5（4 个数）|
| **损失 loss** | 整条线 | 把所有误差综合成一个分数 | 一个标量 L |

> 一个**误差**只描述「这一点偏多少」，无法回答「这条线整体好不好」。**损失函数**就是把 m 个误差汇总成一个**总分**——这样不同的 $(w, b)$ 才能比较「谁更好」。

**别名警告**——一物多名，首次出现必须知道：

```
  损失函数 loss function
  = 代价函数 cost function     （本质都是同一个 L）
  = 成本函数
  = 目标函数 objective function
```

后续看到 sklearn 文档说 `loss=...`、Andrew Ng 说 cost、最优化教材说 objective——指的都是同一个东西，叫法看场景。

### 公式 + 例子

**先扫盲数学符号**（首次出现，下面公式会用到）：

| 符号 | 读法 | 直觉 | 例子 |
|---|---|---|---|
| $\sum_{i=1}^{m}$ | 「sigma 从 i=1 到 m 求和」 | 把 i 从 1 加到 m 的所有项累加 | $\sum_{i=1}^{3} i = 1+2+3 = 6$ |
| $\hat y_i$ | 「y hat 下标 i」 | 第 i 个样本的**预测值**（戴帽子=猜的）| 模型猜的体重 |
| $y_i$ | 「y 下标 i」 | 第 i 个样本的**真实值**（不戴帽子=真的）| 真实体重 |
| $m$ | — | 样本数（5 个人就 m=5）| — |

**MSE**（Mean Squared Error，均方误差）：

$$\text{MSE} = \frac{1}{m} \sum_{i=1}^{m} (\hat y_i - y_i)^2 = \frac{1}{m} \sum_{i=1}^{m} (w x_i + b - y_i)^2$$

「把每个点的误差**平方**，全加起来，再**除以 m 求平均**」——所以叫「均方」。

**MAE**（Mean Absolute Error，平均绝对误差）：

$$\text{MAE} = \frac{1}{m} \sum_{i=1}^{m} |\hat y_i - y_i|$$

「把每个点的误差取**绝对值**，全加起来，再除以 m 求平均」——所以叫「平均绝对」。

**例子**：3 个点，真实 $y = [10, 20, 30]$，预测 $\hat y = [12, 18, 33]$，误差 $[+2, -2, +3]$。

| 指标 | 算法 | 值 |
|---|---|---|
| MSE | $(4 + 4 + 9)/3$ | **5.67** |
| MAE | $(2 + 2 + 3)/3$ | **2.33** |

### 为什么用平方而不是绝对值

**罚款规则曲线**（横轴=单点误差，纵轴=该点贡献的罚款）：

```
   单次罚款 ↑
        16 │              ╱  MSE: 罚款 = 误差²
           │             ╱       (偏 4 → 罚 16，平方放大)
           │            ╱
           │           ╱
         9 │          ╱
           │         ╱
           │        ╱           MAE: 罚款 = |误差|
         4 │       ╱      ___────       (偏 4 → 罚 4，线性)
           │      ╱  ___───
           │     ╱___─
         1 │ __─╱
           │_/
         0 ┼─────┬─────┬─────┬─────┬───→ 误差 |ŷ - y|
           0     1     2     3     4

   关键差异：误差 ≥ 1 时 MSE 罚得比 MAE 狠，且越偏越狠（平方放大）
            误差 < 1 时反而 MSE 罚得更轻（小误差被「轻判」）
```

| 维度 | 平方（MSE）| 绝对值（MAE）|
|---|---|---|
| 可导性 | **处处可导** ✓ | 在 0 处不可导（尖角）✗ |
| 大误差敏感度 | **平方放大** —— 误差 10 算 100 | 线性 —— 误差 10 还是 10 |
| 异常值 outlier | **怕异常值**，会被拽歪 | 抗异常值（鲁棒）|
| 求解 | 有闭式解（正规方程）| 没有闭式解，得迭代 |

**结论**：MSE 数学性质好（光滑可导 → [`04b-gd/`](../04b-gd/) 梯度下降能用 / 可解析求最小 → [`04a-analytical/`](../04a-analytical/) 正规方程能用），但代价是怕 outlier。LR 标准教科书默认用 MSE 就是这个原因。数据有 outlier 就改用 MAE 或 Huber loss。

### 红线 vs 蓝线：Loss 小才是好线

PPT slide 21 的核心论断：**Loss 越小，模型越好**——拿两条候选线打分立马看出来。

```
  体重 y                                      体重 y
   ↑      红线（Loss 小，更优）  ●            ↑      蓝线（Loss 大，差）
   │                       ╱  │              │              ●
   │                    ╱     ↕ ↓1            │            ●  ↕↑5
   │                  ●                       │                ↕↑3
   │               ╱  ↕ ↓0.5                  │       ●━━━━━━━━━━━ ŷ ≈ 65（水平）
   │             ●                            │     ●
   │           ╱   ↕ ↑1                       │   ●
   │         ●                                │ ●
   │       ╱    ↕ ↑0.5                        │
   │   ●╱                                     │
   └──────────────────→ x                     └──────────────────→ x

  红线：贴近所有点 → 4 个 ↕ 都很短 → MSE 小      蓝线：远离大多数点 → 4 个 ↕ 都很长 → MSE 大
```

**最小二乘 least squares**——「最小化**平方和**」的简称。把所有误差平方再求和（或求平均）当 Loss，然后求最小，就是最小二乘。

> 「二乘」就是平方（古汉语「乘」= 自乘 = 平方）。「最小二乘 = 最小化平方和」是术语，不是新规则。

**这一段的核心断言**：训练 LR 不需要"灵感"，只需要把 Loss 当目标，让数学帮你挑出那条红线。

### 与 LR 的连接

LR 训练 = **找 $(w, b)$ 让 MSE 最小**：

$$\min_{w, b} L(w, b) = \frac{1}{m} \sum_{i=1}^{m} (w x_i + b - y_i)^2$$

把 $L(w, b)$ 看成一个关于 $(w, b)$ 的二元二次函数（碗形曲面），最低点就是最优解。

**3D 碗形 + 等高线视角**：

```
  3D 视角（侧面看）              俯视等高线（从上往下看）
  Loss ↑                          b ↑
     │     ╲       ╱                │     ╭──────────╮
     │      ╲     ╱                 │   ╭─╯           ╰─╮
     │       ╲   ╱                  │  │   ╭─────╮     │
     │        ╲ ╱                   │  │  │  ╭─╮  │    │
     │         ★ ← 碗底             │  │  │ │ ★ │ │    │   ★ = (w*, b*) 最优解
     │       (w*,b*)                │  │  │  ╰─╯  │    │   同心圈 = Loss 等高线
     │                              │  │   ╰─────╯     │   越靠中心 Loss 越小
     │                              │   ╰─╮           ╰─╮
     └──────────────────→ (w, b)    │     ╰──────────╯
                                    └────────────────────→ w

  几何直觉：Loss 是 (w,b) 平面上方的「光滑碗」，碗只有一个底
  → 不管从哪起步往低处走，都能滚到同一个 ★（凸函数，无局部最优陷阱）
```

> Slide 25 的 Loss-k 曲线是这张碗的**一个剖面**（固定 b=-100，沿 w 方向切一刀的二次曲线）。真实情况是 (w, b) 两个维度一起调。

**两条求最低点的路**：

1. **解析解**——令 $\partial L / \partial w = 0$ 且 $\partial L / \partial b = 0$，直接解方程。这就是 [`04a-analytical/`](../04a-analytical/) 的「最小二乘法」。
2. **数值解**——从随机点出发，沿梯度反方向一步步往下走。这就是 [`04b-gd/`](../04b-gd/) 的「梯度下降」。

两条路殊途同归，都依赖 Loss 「**可导**」这个性质——所以 MSE 是 LR 的天选搭档。

**求解套路一图流**（Slide 22 的标准流程）：

```
   数据 (x_i, y_i)        假设 ŷ = wx + b
        │                       │
        └──────┬────────────────┘
               ▼
       构造 Loss(w, b) = 平方和 / m
               │
               ▼
        最小化 Loss
       ╱            ╲
   解析解          数值解
   (正规方程)     (梯度下降)
       ╲            ╱
        ▼          ▼
       (w*, b*) ← 最优参数
               │
               ▼
       最优模型 ŷ = w* x + b*
```

整个 LR 训练就这一张图。后续 04a / 04b 是把「最小化 Loss」这一步拆开实现。

### 主线案例 · 5 样本算一遍 k*（Slide 23-24）

> **本段的目的**：用真数据走一遍「构造 Loss → 最小化 → 求最优 k」，把上面的抽象碗形钉死成具体数字。

**数据**（身高 → 体重，5 个样本，跨 04a 章节同源）：

| i | 身高 $x_i$ | 真实体重 $y_i$ |
|---|---|---|
| 1 | 160 | 56.3 |
| 2 | 166 | 60.6 |
| 3 | 172 | 65.1 |
| 4 | 174 | 68.5 |
| 5 | 180 | 75.0 |

**Step 1 · 写出 Loss**（假设 $y = kx + b$，用平方和）：

$$L(k, b) = \sum_{i=1}^{5} (k x_i + b - y_i)^2$$

这是个关于 $(k, b)$ 的**二元二次函数**——就是上面那张「碗」。直接对两个变量同时求最小有点抽象，PPT 用了一招简化：

**Step 2 · 教学简化：固定 b = -100**，让 Loss 只剩 k 一个变量。这样就是上面那张碗的**一刀剖面**——一条二次抛物线。代入 $b = -100$、移项后变成 $(k x_i - (y_i - 100))^2$ 求和。记 $c_i = y_i - 100$，得 $c = [156, 160.6, 165.1, 168.5, 175]$：

$$L(k) = \sum_{i=1}^{5} (x_i\,k - c_i)^2$$

> 为什么固定 b？纯教学简化——把 2D 的碗压成 1D 的抛物线，一眼能看清「怎么求最优」。真正的 LR 不固定 b，两个一起优化（→ [`04a-analytical/`](../04a-analytical/)）。

**Step 3 · 展开合并**——每项 $(ax - c)^2 = a^2 k^2 - 2ac\,k + c^2$：

```
k² 系数 = Σ x_i²  = 160² + 166² + 172² + 174² + 180²
                  = 25600 + 27556 + 29584 + 30276 + 32400
                  = 145416

k 一次项系数（取负）= 2 Σ x_i c_i
       = 2·160·156 + 2·166·160.6 + 2·172·165.1 + 2·174·168.5 + 2·180·175
       = 49920 + 53319.2 + 56794.4 + 58638 + 63000
       = 281671.6     →  −281671.6 k

常数项 = Σ c_i²    = 156² + 160.6² + 165.1² + 168.5² + 175²
                  = 136496.32
```

合起来：

$$L(k) = 145416\,k^2 \;-\; 281671.6\,k \;+\; 136496.32$$

**Step 4 · 求最小值**——一元二次 $ak^2 + bk + c$ 顶点公式 $k^* = -b/(2a)$（这里 a=145416, b=−281671.6）：

$$\boxed{\;k^* \;=\; \frac{281671.6}{2 \times 145416} \;\approx\; \mathbf{0.9685}\;}$$

**Step 5 · 用最优模型预测**——身高 176 cm 的小明体重多少？代入 $\hat y = k^* x + b$：

$$\hat y = 0.9685 \times 176 + (-100) \;=\; \mathbf{70.456 \text{ kg}}$$

> 这个数字（**70.456 kg**）会在 [`04a-analytical/`](../04a-analytical/) 用正规方程解一次、[`04b-gd/`](../04b-gd/) 用梯度下降解一次——**三种路径殊途同归，得同一个数**。这也是为什么本章 5 样本数据要跨章数字一致（指南 §7 数据自洽）。

### Loss-k 剖面 · 偏离最优会怎样（Slide 25）

把 Step 3 那条抛物线画出来——**这就是 Slide 25 的 Loss-k 曲线**：

```
 L(k)                             ★ 偏离最优 → loss 暴涨（拟合变差）
    ↑ ╲                       ╱
    │  ╲                     ╱       L(k) = 145416 k² − 281671.6 k + 136496.32
    │   ╲                   ╱
    │    ╲                 ╱
    │     ╲               ╱
    │      ╲             ╱
    │       ╲           ╱
    │        ╲         ╱
    │         ╲       ╱
    │          ╲_____╱
    │             ●  ← L_min ≈ Σc_i² − (281671.6)²/(4·145416) ≈ 27.18
    │           k* = 0.9685（碗底，最优解）
    └────────────────────────────────→ k
              0.9685

   离 k* 越远，L 上升越快（二次抛物线 → 离顶点 Δk 远，L 上升 ≈ 145416·Δk²）
   k = 0.97 时 L ≈ 27.21（多 0.03）；k = 1.0 时 L ≈ 64.7（多 ≈ 38）
```

**关键观察**：

- **碗底唯一**——一元二次抛物线只有一个极小值，没有「局部最优陷阱」（凸函数的好处）
- **离最优越远，惩罚越大**（二次增长）——这就是为什么训练**必须**找到最优解，而不是「差不多就行」
- **Loss-k 是 Loss-(w,b) 碗形的剖面**——上面 3D 碗在 b=-100 平面的切片

> 真实 LR 的 b 不固定。同时优化 (w, b) 是正规方程 / 梯度下降的工作（→ [`04a-analytical/`](../04a-analytical/)、[`04b-gd/`](../04b-gd/)），思想完全一样：找碗底。

### 常见混淆 / 坑

- **MSE vs RMSE vs SSE**：SSE 是「平方和」（不除 m），MSE 是「均方」（除 m），RMSE 是「均方根」（再开方，单位和 y 一致）。**优化目标完全等价**——除 m、开方都是单调变换，最优 $(w, b)$ 不变。报告时用 RMSE（量纲对得上），求导时用 SSE（少写个 1/m）。
- **「最小二乘」≠ LR**——「最小二乘」是优化目标（用平方和当 Loss），LR 是模型形式（线性加权）。两者**经常一起出现**（LR + 最小二乘 = 经典配方），但不是一回事。SVM、神经网络也能用最小二乘，KNN 不能（KNN 没参数要优化）。
- **Loss = 0 ≠ 好模型**——训练集 Loss = 0 通常是过拟合 overfitting。真正衡量模型的是测试集 Loss。

### 术语版

| 故事里的元素 | 术语名 | 主场 |
|---|---|---|
| 单点偏多少 | **误差** error · $\hat y_i - y_i$ | 本章 |
| 整条线的总分 | **损失函数** loss function（= 代价 cost / 成本 / 目标 objective）| 本章 |
| 平方罚款 + 求和 + 平均 | **均方误差** MSE · Mean Squared Error | 本章 |
| 绝对值罚款 + 求和 + 平均 | **平均绝对误差** MAE · Mean Absolute Error | 本章 |
| 「最小化平方和」 | **最小二乘** least squares | 本章 / [`04a-analytical/`](../04a-analytical/) |
| 罚得最少的那组 (w, b) | **最优参数** optimal parameters · $(w^*, b^*)$ | 本章 |
| Loss = 0 但其实很糟 | **过拟合** overfitting | [`../03a-overfitting/`](../03a-overfitting/) |
| 异常数据点 | **异常值** outlier | 本章 / 数据预处理 |
| 平方+绝对值的折中 Loss | **Huber loss** | sklearn `HuberRegressor` |
| 误差除 m、开方 | **RMSE** Root Mean Squared Error | 报告指标 |
| 误差求和不除 m | **SSE** Sum of Squared Errors | 求导推导用 |

**这一节的关键启示**：**模型训练 = 找最低罚款**——把"好不好"翻译成 Loss 这个数，再让数学帮你滚到碗底。MSE 的所有数学优势（光滑、可导、有闭式解、唯一最小），都是为这件事服务的。
