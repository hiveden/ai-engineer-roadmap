# 损失函数

> 级别：【掌握】 · 维度：数学 · 底稿见 [`README.md`](./README.md)（对应 slide 19-22）

## 底稿（精简）

- 误差 = 预测值 $\hat y$ − 真实值 $y$（单点）
- 损失函数 Loss = 综合衡量整批样本预测和真实值的差异（一个标量）
- 别名：代价函数 cost / 目标函数 objective
- 回归常用：MSE 均方误差、MAE 平均绝对误差
- 「最小二乘法」= 用 MSE（平方和误差）做损失

---

## 直觉（一句话）

**损失函数 = 给「这条线拟合得多差」打一个分数；分数越小，模型越好。**

模型训练就一句话：在所有可能的 $(w, b)$ 里，挑那组让 Loss 最小的——这就是 [`04a-analytical/`](../04a-analytical/) 正规方程和 [`04b-gd/`](../04b-gd/) 梯度下降在干的事。

## 公式 + 例子

**MSE**（Mean Squared Error，均方误差）：

$$\text{MSE} = \frac{1}{m} \sum_{i=1}^{m} (\hat y_i - y_i)^2 = \frac{1}{m} \sum_{i=1}^{m} (w x_i + b - y_i)^2$$

**MAE**（Mean Absolute Error，平均绝对误差）：

$$\text{MAE} = \frac{1}{m} \sum_{i=1}^{m} |\hat y_i - y_i|$$

**例子**：3 个点，真实 $y = [10, 20, 30]$，预测 $\hat y = [12, 18, 33]$，误差 $[+2, -2, +3]$。

| 指标 | 算法 | 值 |
|---|---|---|
| MSE | $(4 + 4 + 9)/3$ | **5.67** |
| MAE | $(2 + 2 + 3)/3$ | **2.33** |

## 为什么用平方而不是绝对值

| 维度 | 平方（MSE）| 绝对值（MAE）|
|---|---|---|
| 可导性 | **处处可导** ✓ | 在 0 处不可导（尖角）✗ |
| 大误差敏感度 | **平方放大** —— 误差 10 算 100 | 线性 —— 误差 10 还是 10 |
| 异常值 outlier | **怕异常值**，会被拽歪 | 抗异常值（鲁棒）|
| 求解 | 有闭式解（正规方程）| 没有闭式解，得迭代 |

**结论**：MSE 数学性质好（光滑可导 → [`04b-gd/`](../04b-gd/) 梯度下降能用 / 可解析求最小 → [`04a-analytical/`](../04a-analytical/) 正规方程能用），但代价是怕 outlier。LR 标准教科书默认用 MSE 就是这个原因。数据有 outlier 就改用 MAE 或 Huber loss。

## 与 LR 的连接

LR 训练 = **找 $(w, b)$ 让 MSE 最小**：

$$\min_{w, b} L(w, b) = \frac{1}{m} \sum_{i=1}^{m} (w x_i + b - y_i)^2$$

把 $L(w, b)$ 看成一个关于 $(w, b)$ 的二元二次函数（碗形曲面），最低点就是最优解。

**两条求最低点的路**：

1. **解析解**——令 $\partial L / \partial w = 0$ 且 $\partial L / \partial b = 0$，直接解方程。这就是 [`04a-analytical/`](../04a-analytical/) 的「最小二乘法」。
2. **数值解**——从随机点出发，沿梯度反方向一步步往下走。这就是 [`04b-gd/`](../04b-gd/) 的「梯度下降」。

两条路殊途同归，都依赖 Loss 「**可导**」这个性质——所以 MSE 是 LR 的天选搭档。

## 常见混淆 / 坑

- **MSE vs RMSE vs SSE**：SSE 是「平方和」（不除 m），MSE 是「均方」（除 m），RMSE 是「均方根」（再开方，单位和 y 一致）。**优化目标完全等价**——除 m、开方都是单调变换，最优 $(w, b)$ 不变。报告时用 RMSE（量纲对得上），求导时用 SSE（少写个 1/m）。
- **「最小二乘」≠ LR**——「最小二乘」是优化目标（用平方和当 Loss），LR 是模型形式（线性加权）。两者**经常一起出现**（LR + 最小二乘 = 经典配方），但不是一回事。SVM、神经网络也能用最小二乘，KNN 不能（KNN 没参数要优化）。
- **Loss = 0 ≠ 好模型**——训练集 Loss = 0 通常是过拟合 overfitting。真正衡量模型的是测试集 Loss。
