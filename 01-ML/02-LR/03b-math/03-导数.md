## ━━━━━━━━ 底稿 ━━━━━━━━

### PPT

#### Slide 32 · 导数 – 基本概念

函数 y = f(x) 的自变量 x 在某点 x₀ 上产生增量 Δx 时，函数输出值的增量 Δy 与 Δx 的比值在 Δx → 0 时的极限即为 x₀ 处的导数：

$$f'(x_0) = \lim_{\Delta x \to 0} \frac{\Delta y}{\Delta x} = \lim_{\Delta x \to 0} \frac{f(x_0+\Delta x) - f(x_0)}{\Delta x}$$

#### Slide 33 · 导数的几何意义

函数在某一点的导数 = 该函数所代表的曲线在该点的**切线斜率**。

```
y                       ●  切线（斜率 = f'(x₀)）
  ↑                   ╱
  |        曲线y=f(x) ╱
  |                ╲ ╱
  |                ●(x₀, f(x₀))
  |               ╱
  |              ╱
  +────────────┴──────────→ x
              x₀
```

#### Slide 34 · 常见函数的导数

- **幂函数**：$(x^\alpha)' = \alpha x^{\alpha-1}$
- **指数函数**：$(a^x)' = a^x \ln a$，$(e^x)' = e^x$
- **对数函数**：$(\log_a x)' = \dfrac{1}{x \ln a}$，$(\ln x)' = \dfrac{1}{x}$
- **三角函数**：$(\sin x)' = \cos x$，$(\cos x)' = -\sin x$

#### Slide 35 · 导数的四则运算 + 复合函数

- $(u \pm v)' = u' \pm v'$
- $(uv)' = u'v + uv'$
- $(u/v)' = (u'v - uv')/v^2$
- **复合函数链式法则**：$\{g[h(x)]\}' = g'(h) \cdot h'(x)$ — **先对外函数求导，再对内函数求导**

例：计算 $y = (x^2 + 2x)^2$ 的导函数

$$y' = 2(x^2+2x)(2x+2) = 4(x^3 + 3x^2 + 2x) = 4x^3 + 12x^2 + 8x$$

#### Slide 38 · 导数求极值

**导数为 0 的位置是函数的极值点。**

例：求函数 $y = x^2 - 4x + 5$ 的极小值

$$y' = 2x - 4 = 0 \;\Rightarrow\; x = 2 \;\Rightarrow\; y_{\min} = 2^2 - 4\cdot 2 + 5 = 1$$

```
y
 ↑
 |   ╲                           ╱
 |    ╲                         ╱
 |     ╲                       ╱
 |      ╲_____               _╱
 |            ╲___________╱
 |                ●(2, 1)  ← 极小值点（导数=0）
 +───────────────┴──────────────→ x
                 x=2
```

### 笔记

> 【知道】导数

当函数 y = f(x) 的自变量 x 在一点 $x_0$ 上产生一个增量 Δx 时，函数输出值的增量 Δy 与自变量增量 Δx 的比值在 Δx 趋于 0 时的极限 a 如果存在，a 即为在 $x_0$ 处的导数，记作 $f^\prime(x_0)$ 或 df($x_0$)/dx。

导数是函数的局部性质。一个函数在某一点的导数描述了这个函数在这一点附近的变化率。函数在某一点的导数就是该函数所代表的曲线在这一点上的切线斜率。

**LR 用得到的两条**：
- 幂函数：$(x^\alpha)^\prime = \alpha x^{\alpha-1}$（求 MSE 关于 w 的导数靠它）
- 链式法则：$\{g[h(x)]\}^\prime = g^\prime(h) \cdot h^\prime(x)$（嵌套函数求导）

**导数求极值**：导数为 0 的位置是函数的极值点（LR 闭式解的核心思路）。

## ━━━━━━━━ 讲解 ━━━━━━━━

### 直觉

**导数 = 函数在某点的「瞬时变化率」 = 切线斜率**——函数走得多陡。

LR 用导数干一件事：找 Loss 函数的最低点。最低点斜率为 0，所以**令导数 = 0 解方程**就能找到最优 $(w, b)$。

### 公式 + 例子

**LR 用得到的两条**：

**1. 幂函数求导**：

$$\boxed{(x^\alpha)^\prime = \alpha \cdot x^{\alpha - 1}}$$

例子：$(x^2)^\prime = 2x$，$(x^3)^\prime = 3x^2$，$(x)^\prime = 1$，$(\text{常数})^\prime = 0$。

**2. 链式法则**（复合函数求导，先外后内）：

$$\boxed{\{g[h(x)]\}^\prime = g^\prime(h) \cdot h^\prime(x)}$$

例子：$f(x) = (3x + 2)^2$，外是 $g(u) = u^2$，内是 $h(x) = 3x + 2$。

$$f^\prime(x) = \underbrace{2(3x + 2)}_{g^\prime(h) = 2h} \cdot \underbrace{3}_{h^\prime(x)} = 6(3x + 2)$$

**极值例子**：$f(x) = x^2 - 4x + 3$，$f^\prime(x) = 2x - 4$。令 $f^\prime(x) = 0$ → $x = 2$ → 极小值 $f(2) = -1$。和初中「-b/2a」公式一致（$-(-4)/(2 \cdot 1) = 2$）。

### 与 LR 的连接

LR 的 MSE 损失（固定 b，只看 w）：

$$L(w) = \sum_{i=1}^{m} (w x_i - y_i)^2$$

这是个**关于 w 的二次函数**（碗形）。求最优 w 三步走：

1. **求导**——用幂函数 + 链式法则：

   $$L^\prime(w) = \sum_{i=1}^{m} 2(w x_i - y_i) \cdot x_i$$

2. **令导数为 0**：$\sum 2 x_i (w x_i - y_i) = 0$
3. **解方程**：$w = \frac{\sum x_i y_i}{\sum x_i^2}$

这就是一元 LR 的闭式解。多元情况推广到偏导（[`04-偏导.md`](./04-偏导.md)）→ 矩阵化（[`06-矩阵.md`](./06-矩阵.md)）→ 正规方程（[`04a-analytical/`](../04a-analytical/)）。

**梯度下降**（[`04b-gd/`](../04b-gd/)）也靠导数：每次沿 $-L^\prime(w)$ 方向走一小步，迭代逼近最低点。

### 常见混淆 / 坑

- **导数 = 0 不一定是最小值**——可能是最大值（倒扣的碗）或鞍点。需要看二阶导数 $f^{\prime\prime}$ 判别：$f^{\prime\prime} > 0$ 是极小，$< 0$ 是极大。LR 的 MSE 是凸函数（碗形），所以唯一极值点就是全局最小，不用担心。
- **链式法则方向别搞反**——「先外后内」：先把外函数当整体求导（保持内函数原样），再乘内函数的导数。神经网络的反向传播 backprop 就是大规模链式法则。
- **常见函数导数表**：背不下也没事，遇到查就行。LR 闭式推导**只需要幂函数 + 链式法则**两条，所以这一章不展开 $\sin / \cos / e^x / \ln$ 那些。逻辑回归（后续章节）会用到 $\ln$ 和 sigmoid 的导数，到时候再补。
