## ━━━━━━━━ 底稿 ━━━━━━━━

### PPT

#### Slide 32 · 导数 – 基本概念

函数 y = f(x) 的自变量 x 在某点 x₀ 上产生增量 Δx 时，函数输出值的增量 Δy 与 Δx 的比值在 Δx → 0 时的极限即为 x₀ 处的导数：

$$f'(x_0) = \lim_{\Delta x \to 0} \frac{\Delta y}{\Delta x} = \lim_{\Delta x \to 0} \frac{f(x_0+\Delta x) - f(x_0)}{\Delta x}$$

#### Slide 33 · 导数的几何意义

函数在某一点的导数 = 该函数所代表的曲线在该点的**切线斜率**。

```
y                       ●  切线（斜率 = f'(x₀)）
  ↑                   ╱
  |        曲线y=f(x) ╱
  |                ╲ ╱
  |                ●(x₀, f(x₀))
  |               ╱
  |              ╱
  +────────────┴──────────→ x
              x₀
```

#### Slide 34 · 常见函数的导数

- **幂函数**：$(x^\alpha)' = \alpha x^{\alpha-1}$
- **指数函数**：$(a^x)' = a^x \ln a$，$(e^x)' = e^x$
- **对数函数**：$(\log_a x)' = \dfrac{1}{x \ln a}$，$(\ln x)' = \dfrac{1}{x}$
- **三角函数**：$(\sin x)' = \cos x$，$(\cos x)' = -\sin x$

#### Slide 35 · 导数的四则运算 + 复合函数

- $(u \pm v)' = u' \pm v'$
- $(uv)' = u'v + uv'$
- $(u/v)' = (u'v - uv')/v^2$
- **复合函数链式法则**：$\{g[h(x)]\}' = g'(h) \cdot h'(x)$ — **先对外函数求导，再对内函数求导**

例：计算 $y = (x^2 + 2x)^2$ 的导函数

$$y' = 2(x^2+2x)(2x+2) = 4(x^3 + 3x^2 + 2x) = 4x^3 + 12x^2 + 8x$$

#### Slide 38 · 导数求极值

**导数为 0 的位置是函数的极值点。**

例：求函数 $y = x^2 - 4x + 5$ 的极小值

$$y' = 2x - 4 = 0 \;\Rightarrow\; x = 2 \;\Rightarrow\; y_{\min} = 2^2 - 4\cdot 2 + 5 = 1$$

### 笔记

> 【知道】导数

当函数 y = f(x) 的自变量 x 在一点 $x_0$ 上产生一个增量 Δx 时，函数输出值的增量 Δy 与自变量增量 Δx 的比值在 Δx 趋于 0 时的极限 a 如果存在，a 即为在 $x_0$ 处的导数，记作 $f^\prime(x_0)$ 或 df($x_0$)/dx。

导数是函数的局部性质。一个函数在某一点的导数描述了这个函数在这一点附近的变化率。函数在某一点的导数就是该函数所代表的曲线在这一点上的切线斜率。

**LR 用得到的两条**：
- 幂函数：$(x^\alpha)^\prime = \alpha x^{\alpha-1}$（求 MSE 关于 w 的导数靠它）
- 链式法则：$\{g[h(x)]\}^\prime = g^\prime(h) \cdot h^\prime(x)$（嵌套函数求导）

**导数求极值**：导数为 0 的位置是函数的极值点（LR 闭式解的核心思路）。

### 生活锚 · 开车码表上的瞬时速度

你开车跑高速，手机地图算的是「平均速度」——120 公里 / 1 小时 = 120 km/h。这是 1 小时维度的「平均」，没法回答「我**这一秒**到底踩了多深的油门」。

**码表**回答这个：把时间窗口缩到极短，看「**这一瞬间**走了多少米 / 这一瞬间过了多少秒」的比值——这就是**瞬时速度**。窗口越短越准，缩到 Δt → 0 的极限，就是「此刻这一脚」的真实速度。

把「位置」换成任意函数 $y = f(x)$，把「时间」换成任意自变量 $x$：

```
平均变化率 = Δy / Δx        （宏观，对应平均速度）
                   ↓ 让 Δx → 0
瞬时变化率 = lim Δy / Δx     （此刻这一点的「陡度」 = 导数）
            Δx→0
```

LR 干的就是「瞬时变化率」这件事——损失函数 Loss 在每个参数 $k$ 上有多陡？陡的方向往哪倒？走到不陡（斜率=0）的地方就是最优解。本节把这把工具吃透。

### 业务问题

[`01-损失函数.md`](./01-损失函数.md) 里我们用顶点公式 $k^* = -b/(2a)$ 算出 $k^* = 0.9685$。但顶点公式只对**一元二次**有效——LR 真上场时 Loss 是 $w_1, w_2, \ldots, w_n, b$ 的多元函数，没有「顶点公式」可套。

需要一把更通用的工具：**给任意一个函数，告诉我「最低点在哪」**。这把工具叫导数。本节先在一元上把它磨利，下一节 [`04-偏导.md`](./04-偏导.md) 推广到多元。

### 数学符号扫盲

下面公式会反复用，首次出现先扫盲：

| 符号 | 读法 | 直觉 | 例子 |
|---|---|---|---|
| $f^\prime(x)$ | 「f prime x」 | 函数 $f$ 在 $x$ 处的导数（求导后的新函数）| $f(x)=x^2$ → $f^\prime(x)=2x$ |
| $\dfrac{\mathrm{d}y}{\mathrm{d}x}$ | 「d y d x」 | 莱布尼茨记号，y 对 x 求导（同一个东西的另一种写法）| $\dfrac{\mathrm{d}(x^2)}{\mathrm{d}x} = 2x$ |
| $\lim\limits_{\Delta x \to 0}$ | 「limit Δx 趋于 0」 | 让 $\Delta x$ 无限缩小（不等于 0）后看比值收敛到哪 | $\lim\limits_{\Delta x \to 0}\dfrac{(\Delta x)^2}{\Delta x} = 0$ |
| $\{g[h(x)]\}^\prime$ | 「g of h 求导」 | 复合函数：先把 $x$ 喂给 $h$，再把结果喂给 $g$，整体求导 | 见下方链式法则 |

> **两种记号同义**：$f^\prime(x) = \dfrac{\mathrm{d}f}{\mathrm{d}x} = \dfrac{\mathrm{d}y}{\mathrm{d}x}$。`f'` 简洁，`dy/dx` 在多元和链式法则里更清楚（能看出是「对谁」求导）。

### 【数学】几何意义 · 导数 = 切线斜率

把开车码表的直觉搬到坐标系——**导数 derivative 就是曲线在那点的切线 tangent 的斜率 slope**。

```
y                                            两点连线（割线 secant）
  ↑                                         ╱   斜率 = Δy / Δx（平均变化率）
  │                                  ●(x₀+Δx, f(x₀+Δx))
  │                                ╱│
  │                              ╱  │
  │                            ╱    │ Δy = f(x₀+Δx) − f(x₀)
  │                          ╱      │
  │                        ╱        │
  │                  ●─────────────  │
  │                ╱  (x₀, f(x₀))    │
  │              ╱  ←Δx→             │
  │            ╱                     │
  │ 曲线 y=f(x)                       │
  └──────────┴──────────┴────────────→ x
            x₀         x₀+Δx

   让 Δx → 0：割线绕着 (x₀, f(x₀)) 旋转，倒向唯一一条「切线 tangent」
   切线斜率 = f'(x₀) = lim Δy/Δx
                       Δx→0
```

**物理直觉**：割线斜率 = 「这一段平均涨多快」；切线斜率 = 「**就在这一点**涨多快」。

#### 单调性 · 导数符号告诉你曲线方向

```
   f'(x) > 0 ↗            f'(x) < 0 ↘            f'(x) = 0  ─
   切线向右上              切线向右下              切线水平
   函数递增                函数递减                极值候选点
                                                 （山顶 / 谷底 / 鞍点）

       ╱─                    ─╲                       ╱──╲
      ╱                        ╲                     ╱    ╲
     ╱                          ╲                   ╱  ●   ╲   ← f'=0 的点
    ╱      正例                  ╲    反例          ╱  谷底  ╲
   ╱                              ╲                ╱          ╲
```

**正反双向激活**：导数告诉你「往左走还是往右走能让函数变小」——这是梯度下降（[`04b-gd/`](../04b-gd/)）的根，本节先记住「**斜率=0 是极值候选**」即可。

### 【数学】常见函数导数表 · 工程视角

PPT slide 34 列了 4 类（幂 / 指数 / 对数 / 三角）。**LR 工程语境下只用得到第一类 + 链式法则**，其他做到「会查表」就够：

| 类别 | 公式 | LR 用得到？ |
|---|---|---|
| **幂函数** $(x^\alpha)^\prime$ | $\alpha\, x^{\alpha-1}$ | ✓ **MSE 求导核心**（$x^2$ 的导数 = $2x$）|
| **常数** $(C)^\prime$ | $0$ | ✓ 偏导时把别的变量当常数 |
| 指数 $(e^x)^\prime$ | $e^x$ | ✗ LR 不用（LogReg 才用 sigmoid）|
| 指数 $(a^x)^\prime$ | $a^x \ln a$ | ✗ 查表即可 |
| 对数 $(\ln x)^\prime$ | $1/x$ | ✗ LR 不用（LogReg 的对数似然才用）|
| 对数 $(\log_a x)^\prime$ | $1/(x \ln a)$ | ✗ 查表即可 |
| 三角 $(\sin x)^\prime$ | $\cos x$ | ✗ 查表即可 |
| 三角 $(\cos x)^\prime$ | $-\sin x$ | ✗ 查表即可 |

**幂函数实算**（这两条 LR 必用）：

| $f(x)$ | $\alpha$ | $f^\prime(x) = \alpha x^{\alpha-1}$ |
|---|---|---|
| $x^2$ | 2 | $2x$ |
| $x^3$ | 3 | $3x^2$ |
| $x = x^1$ | 1 | $1$ |
| 常数 $= x^0$ 缩放 | — | $0$ |

> **结论**：LR 章节范围内，只需要「幂函数 + 常数 + 链式法则」三条工具。其他公式遇到查表 / 查 sympy 即可，不背。

### 【数学】四则 + 链式法则

**四则**（加 / 减 / 乘 / 除）规则不展开，需要时查 PPT slide 35。LR 推导主要用**和的导数 = 导数的和**：

$$(u + v)^\prime = u^\prime + v^\prime$$

——MSE 是 $\sum_i (\cdots)^2$，外层求和直接对每项分别求导。

**链式法则 chain rule**（复合函数求导，先外后内）：

$$\boxed{\;\{g[h(x)]\}^\prime \;=\; g^\prime(h) \cdot h^\prime(x)\;}$$

口诀「**剥洋葱**」——一层一层往里剥，每剥一层乘一次：

```
   外函数 g                内函数 h(x)
   ┌─────┐               ┌──────┐
   │  g  │  ← 先对外求导   │  h   │  ← 再对内求导
   │     │   保持 h 原样   │      │
   └──┬──┘               └──┬───┘
      └────── 相乘 ─────────┘
```

**实算**（PPT Slide 35 例题）：求 $y = (x^2 + 2x)^2$ 的导数。

```
设  外函数 g(u) = u²            其中 u = h(x) = x² + 2x
    内函数 h(x) = x² + 2x

Step 1 · 对外求导（保持 u 是整体）
        g'(u) = 2u = 2(x² + 2x)

Step 2 · 对内求导
        h'(x) = 2x + 2

Step 3 · 相乘
        y' = g'(h) · h'(x) = 2(x² + 2x) · (2x + 2)
           = 4(x² + 2x)(x + 1)
           = 4x³ + 12x² + 8x        ✓ 与 PPT 答案一致
```

> **反例**——「先内后外」搞反方向：会得到 $2x \cdot (2x+2)$ = 错。链式法则是「**先把内层当未拆封整体对外求导，再单独拆内层**」。神经网络反向传播 backpropagation 就是大规模链式法则在多层间反复套——这里把方向打牢，后面学 MLP 不会卡。

### 【数学】导数 = 0 求极值

**核心断言**：函数取极值（最大或最小）的位置，切线必然水平——也就是 $f^\prime(x) = 0$。

**几何直观**：想象登山——上坡（$f^\prime > 0$）、下坡（$f^\prime < 0$）。山顶那一刻**既不上也不下**，斜率必然为 0。山谷同理。

```
      f'>0 ↗                  ↖ f'<0
            ╲      f'=0       ╱
             ╲     ───●───   ╱       ← 极大值（山顶）：导数从正变负穿过 0
              ╲    山顶      ╱
               ╲             ╱
                ╲           ╱
                 ╲         ╱
                  ╲       ╱
                   ╲     ╱
       f'<0 ↘     ╲   ╱     ↗ f'>0
                    ╲ ╱
                     ●         ← 极小值（谷底）：导数从负变正穿过 0
                  f'=0
```

**实算**（PPT Slide 38 例题）：求 $y = x^2 - 4x + 5$ 的极小值。

```
Step 1 · 求导（幂函数 + 和规则）
        y' = (x²)' − (4x)' + (5)' = 2x − 4 + 0 = 2x − 4

Step 2 · 令 y' = 0
        2x − 4 = 0  ⇒  x = 2

Step 3 · 代回原函数
        y_min = 2² − 4·2 + 5 = 4 − 8 + 5 = 1

⇒ 极小值点 (2, 1)
```

> **坑**：$f^\prime = 0$ 是极值的**必要条件**，不是充分条件——可能是极大、极小、或鞍点（如 $f(x)=x^3$ 在 $x=0$ 处 $f^\prime=0$ 但既不是极大也不是极小）。判别要看**二阶导** $f^{\prime\prime}$：$f^{\prime\prime}>0$ 是极小（碗朝上），$f^{\prime\prime}<0$ 是极大（碗朝下）。LR 的 MSE 是凸函数 convex function（碗朝上），$f^\prime=0$ 必然是全局最小，不用纠结判别。

### 主线案例 · 用 L'(k)=0 重算 k* = 0.9685

[`01-损失函数.md`](./01-损失函数.md) 用顶点公式 $-b/(2a)$ 算出 $k^* = 0.9685$。**那个公式背后就是「导数=0」**——本节把这条路实算一遍，把两章串起来。

**回顾数据**（5 样本身高 → 体重，固定 b = -100）：

| i | 身高 $x_i$ | 真实体重 $y_i$ |
|---|---|---|
| 1 | 160 | 56.3 |
| 2 | 166 | 60.6 |
| 3 | 172 | 65.1 |
| 4 | 174 | 68.5 |
| 5 | 180 | 75.0 |

01 章已经合并出：

$$L(k) = 145416\,k^2 \;-\; 281671.6\,k \;+\; 136496.32$$

**Step 1 · 求导**（幂函数 + 和规则）：

```
L(k) = 145416 k²      − 281671.6 k       + 136496.32
        ↓ (k²)'=2k       ↓ (k)'=1            ↓ (常数)'=0
L'(k) = 145416·2k    −  281671.6·1     +  0
      = 290832 k     −  281671.6
```

$$\boxed{\;L^\prime(k) \;=\; 290832\,k \;-\; 281671.6\;}$$

**Step 2 · 令 L'(k) = 0**：

$$290832\,k - 281671.6 = 0$$

**Step 3 · 解方程**：

$$k^* \;=\; \frac{281671.6}{290832} \;=\; \frac{281671.6}{2 \times 145416} \;\approx\; \mathbf{0.9685}$$

**Step 4 · 与顶点公式对照**——殊途同归：

| 路径 | 公式 | 算式 | $k^*$ |
|---|---|---|---|
| 01 章 · 顶点公式 | $-b/(2a)$ | $281671.6 / (2 \cdot 145416)$ | **0.9685** |
| 本章 · 导数=0 | $L^\prime(k) = 0$ 解出 | $281671.6 / 290832$ | **0.9685** |

> **关键观察**：顶点公式 $-b/(2a)$ 不是天降神迹——它**就是** $L^\prime(k) = 2ak + b = 0$ 解出来的。一元二次只是导数法的一个特例。本章把工具升级到通用版，下一章 04-偏导推广到多元，05-向量 + 06-矩阵 推广到矩阵形式 → 04a 章直接吐出正规方程 $\mathbf{w}^* = (X^\top X)^{-1} X^\top \mathbf{y}$。这条链就是 LR 闭式解的脊梁。

**主线案例可视化**——L(k) 抛物线在 k* 处切线水平：

```
 L(k)
    ↑ ╲                                                       ╱
    │  ╲     ─── 抛物线 L(k) = 145416k² − 281671.6k + 136496.32
    │   ╲                                                   ╱
    │    ╲                                                 ╱
    │     ╲                                               ╱
    │      ╲    L'(k) < 0                L'(k) > 0       ╱
    │       ╲   左边斜下（往右走能减小 L）  右边斜上       ╱
    │        ╲                                           ╱
    │         ╲                                         ╱
    │          ╲                                       ╱
    │           ╲_____                            ____╱
    │                ╲_______                ___╱
    │                       ╲___________●___╱
    │                              k*=0.9685   ←  L'(k*) = 0（切线水平）
    │                              ●━━━━━━━━━ ◀── 切线斜率 = 0 ⇔ 谷底
    └─────────────────────────────┴────────────────────→ k
                                  0.9685

   核心：导数符号告诉你「往哪走能减小 L」
        L'<0 → 往右走         L'>0 → 往左走        L'=0 → 到了，不走了 ★
```

**预告链路**：

- **正规方程**（[`04a-analytical/`](../04a-analytical/)）：把「导数=0」推广到 $(w, b)$ 两个变量同时求 → 偏导（[`04-偏导.md`](./04-偏导.md)）→ 矩阵化（[`06-矩阵.md`](./06-矩阵.md)）一步到位
- **梯度下降**（[`04b-gd/`](../04b-gd/)）：不解方程，从随机起点出发，每次沿 $-L^\prime(k)$ 方向走一小步，迭代逼近 k\*。**走的方向**就靠导数符号决定

### 常见混淆 / 坑

- **导数 = 0 不一定是最小值**——可能是最大值、鞍点。判别要看二阶导 $f^{\prime\prime}$。LR MSE 是凸函数，唯一极值就是全局最小，可以放心用。
- **链式法则方向别搞反**——「先外后内」：先把外函数当整体对内层求导（保持内层原样），再乘内层导数。神经网络反向传播 backpropagation 就是大规模链式法则。
- **`f'` 和 `dy/dx` 是同一个东西**——前者简洁，后者在多元 / 链式法则里更清楚（标注「对谁求导」）。莱布尼茨记号 $\dfrac{\mathrm{d}y}{\mathrm{d}x}$ 不是分数，但**形式上**可以当分数操作（链式法则就是「分子分母约掉中间变量」的形式记忆）。
- **常见函数导数表**——背不下没事，遇到查就行。LR 范围内只需要幂函数 + 链式法则两条；LogReg 章节会用到 $\ln$ 和 sigmoid 的导数，到时再补，主场不在这。

| 故事里的元素 | 术语名 | 主场 |
|---|---|---|
| 函数在某点的瞬时变化率（码表） | **导数** derivative · $f^\prime(x)$ / $\dfrac{\mathrm{d}y}{\mathrm{d}x}$ | 本章 |
| 曲线在某点的「贴线」 | **切线** tangent line | 本章 |
| 切线的陡度 | **斜率** slope | 本章 |

| 让 $\Delta x$ 无限缩小后看比值 | **极限** limit · $\lim_{\Delta x \to 0}$ | 本章 |
| 复合函数求导先外后内 | **链式法则** chain rule | 本章 / 神经网络反向传播 |
| $(x^\alpha)^\prime = \alpha x^{\alpha-1}$ | **幂函数求导** power rule | 本章 |
| $f^\prime = 0$ 解出极值候选 | **导数=0 求极值** · 一阶必要条件 | 本章 / [`04a-analytical/`](../04a-analytical/) |
| 碗朝上、唯一极小 | **凸函数** convex function | [`04b-gd/`](../04b-gd/) |
