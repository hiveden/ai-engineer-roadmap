# 偏导

> 维度：数学
> 知识点级别：【知道】偏导 = 多元函数对单变量求导（其它当常数）；梯度 = 所有偏导组成的向量
> 章节底稿全文见 [`README.md`](./README.md)（PPT slide 39-40）

## ━━━━━━━━ 底稿 ━━━━━━━━

### PPT

#### Slide 39 · 偏导（引入）

z = z(x, y)，求 $z = (x-2)^2 + (y-3)^2$ 的极小值。

- 对 x 偏导：$\dfrac{\partial z}{\partial x} = 2(x-2) \cdot 1 = 0 \;\Rightarrow\; x=2$ 时 x 方向取极小
- 对 y 偏导：$\dfrac{\partial z}{\partial y} = 2(y-3) \cdot 1 = 0 \;\Rightarrow\; y=3$ 时 y 方向取极小

#### Slide 40 · 多元函数偏导（练习）

**1）** $z = x^2 + 2xy - 3y^2$

$$\frac{\partial z}{\partial x} = 2x + 2y, \quad \frac{\partial z}{\partial y} = 2x - 6y$$

**2）** $z = x^y$

$$\frac{\partial z}{\partial x} = y\, x^{y-1}, \quad \frac{\partial z}{\partial y} = x^y \ln x$$

**3）** $z = e^{2x^2 - 3y}$（复合函数 + 多元偏导）

$$\frac{\partial z}{\partial x} = e^{2x^2-3y} \cdot (2x^2-3y)'_x = 4x\, e^{2x^2-3y}$$

$$\frac{\partial z}{\partial y} = e^{2x^2-3y} \cdot (2x^2-3y)'_y = -3\, e^{2x^2-3y}$$

### 笔记

> 【知道】偏导

**多变量函数对其中一个变量求导**，把其它变量当常数。

设 $f(x, y) = x^2 + 3xy + y^2$：

- 对 x 偏导：$\frac{\partial f}{\partial x} = 2x + 3y$（y 当常数 → $3xy \to 3y$，$y^2 \to 0$）
- 对 y 偏导：$\frac{\partial f}{\partial y} = 3x + 2y$

**线性回归用偏导做什么**：MSE 损失 $L(w, b) = \sum (y - wx - b)^2$ 是 w 和 b 的二元函数，分别对 w、b 求偏导，令偏导 = 0，就得到正规方程的解析解（见 04a）。

**梯度**：所有偏导组成的向量，记作 $\nabla L = \left[\frac{\partial L}{\partial w_1}, \frac{\partial L}{\partial w_2}, \ldots\right]^T$。梯度方向 = 函数上升最快的方向，**反方向**就是梯度下降的方向（见 04b）。

## ━━━━━━━━ 讲解 ━━━━━━━━

### 生活锚 · 调音台两个旋钮

想象录音棚里一台调音台，主播只关心**两个旋钮**：**音量**（旋钮 1）和**混响**（旋钮 2）。听众满意度（一个分数）由这两个旋钮一起决定——你想找让满意度最高的旋钮组合。

试一下这个套路：

1. 先**只动音量**、混响按住不动 → 看满意度怎么变 → 这就是「满意度对**音量**的偏导」
2. 再**只动混响**、音量按住不动 → 看满意度怎么变 → 这就是「满意度对**混响**的偏导」
3. 两个旋钮各自找到最佳点（**两个偏导都 = 0**）→ 整体最优

这就是**偏导 partial derivative** 干的事——多变量函数里**只调一个旋钮**测影响，其它当常数按住。本节把这把工具吃透，然后接到 LR：LR 的损失就是 **(w, b) 两个旋钮**的函数，要同时拧到最佳。

### 业务问题

[`01-损失函数.md`](./01-损失函数.md) 里那个 5 样本身高→体重的「碗形」损失：

```
L(w, b) = Σ (w·xᵢ + b - yᵢ)²
```

在那一节我们用了一个**教学简化**——固定 b = -100，只动 w——把 2D 的碗压成 1D 抛物线，求得 k* = 0.9685。但真实场景**不能固定 b**：身高 175 cm 的人，截距挪 10 cm 体重就差 10 kg，b 的位置和 w 的斜率必须**一起调到最佳**。

[`03-导数.md`](./03-导数.md) 的导数 = 0 是单变量招数——只能调一个旋钮。现在两个旋钮一起调，这套招数怎么升级？答案：**两个旋钮各自的「偏导」都 = 0，才是整体最低点**。下面把这件事掰开。

### 跨章节迁移 · 从导数到偏导（升级版）

你已经懂了 [`03-导数.md`](./03-导数.md) 的单变量套路。本节做一次**升级**——变量从 1 个变成多个，求极值的招数也跟着升级：

| 维度 | 03 单变量 f(x) | 04 多变量 f(x, y) |
|---|---|---|
| 变量数 | 1 个 | 2 个或更多 |
| "斜率" 工具 | 导数 $f^\prime(x) = \frac{df}{dx}$ | **偏导** $\frac{\partial f}{\partial x}$、$\frac{\partial f}{\partial y}$（每个变量一个）|
| 符号 | $d$（直体小 d）| **$\partial$**（圆体 partial）|
| 求极值 | $f^\prime(x) = 0$ 解出 $x^*$ | **两个偏导同时 = 0**：$\frac{\partial f}{\partial x} = 0$ **且** $\frac{\partial f}{\partial y} = 0$ |
| 几何 | 切线斜率 = 0（曲线水平） | 碗底（曲面在两个方向都水平）|
| 全部斜率打包 | — | **梯度** $\nabla f = \left[\frac{\partial f}{\partial x}, \frac{\partial f}{\partial y}\right]^T$（一个向量）|

**升级原则一句话**：把单变量的"导数=0 求极值"复制到**每一个变量**——所有变量的偏导**同时 = 0**才是极值点。

> 一个变量的偏导 = 0 只保证沿这个方向是水平的——其它方向可能还在下降。**所有方向都水平 ⇔ 极值**。这也是 Slide 50 第 3 题答案 A 的底层含义。

### 【数学】数学符号扫盲

新符号首次出现，先认脸：

| 符号 | 读法 | 直觉 | 例子 |
|---|---|---|---|
| $\partial$ | 「partial」 | 偏导专用符号（对应一元导数的 $d$）| $\partial f / \partial x$ |
| $\partial f / \partial x$ | 「partial f partial x」 | f 对 x 偏导：只动 x、其它当常数 | $f(x,y)=x^2+y$，$\partial f/\partial x = 2x$ |
| $\nabla f$ | 「nabla f」或「grad f」 | **梯度** gradient——所有偏导组成的列向量 | $\nabla f = [\partial f/\partial x,\, \partial f/\partial y]^T$ |
| $\mathbf{0}$ | 「零向量」 | 每个分量都是 0 的向量 | $\mathbf{0} = [0, 0]^T$ |

**一句话规则**：偏导 = 多变量函数对**单个变量**求导，把其它变量当常数。机制和一元导数一模一样——只是多了"按住其它变量"这一步。

### 【数学】Slide 39 实算 · z = (x-2)² + (y-3)² 的极小值

这是 PPT 的引入例题——一只**最干净的二元碗**，碗底就在 $(2, 3)$。

**对 x 偏导**（y 当常数 → $(y-3)^2$ 整体当常数 → 求导得 0）：

$$\frac{\partial z}{\partial x} = 2(x-2) \cdot 1 + 0 = 2(x-2)$$

**对 y 偏导**（x 当常数 → $(x-2)^2$ 整体当常数 → 求导得 0）：

$$\frac{\partial z}{\partial y} = 0 + 2(y-3) \cdot 1 = 2(y-3)$$

**两个偏导同时 = 0**——这是关键一步：

$$\begin{cases} 2(x-2) = 0 \\ 2(y-3) = 0 \end{cases} \;\Longrightarrow\; (x^*, y^*) = (2, 3)$$

代回原式：$z_{\min} = 0$。

**碗形 + 两个方向偏导的几何意义**：

```
   z (函数值)
    ↑      ╲                                          ╱
    │       ╲     z = (x-2)² + (y-3)²              ╱
    │        ╲                                    ╱
    │         ╲          ╱─────╲                ╱
    │          ╲       ╱        ╲             ╱
    │           ╲    ╱            ╲         ╱
    │            ╲ ╱                ╲     ╱
    │             ★ ──────────────── 碗底 (2, 3, 0)
    │           (2,3,0)
    │             │  ╲                            ╱
    └─────────────┼───╲────────────────────────╱──────────→
                 (2,3) │   ↑ y 方向 ∂z/∂y                x, y 平面
                       │   （沿 y 轴切一刀的剖面斜率）
                       └─→ x 方向 ∂z/∂x
                          （沿 x 轴切一刀的剖面斜率）

   两个偏导的几何含义：
   ∂z/∂x = 在当前点沿 x 方向切一刀，得到一条曲线，那条曲线的切线斜率
   ∂z/∂y = 沿 y 方向切一刀的曲线切线斜率
   两条切线都水平（斜率都 = 0） ⇔ 站在碗底
```

**正例**：$(2, 3)$ 处两个偏导都 = 0 → 是极小值。
**反例**：$(2, 5)$ 处 $\partial z/\partial x = 0$（x 方向水平），但 $\partial z/\partial y = 2(5-3) = 4 \neq 0$（y 方向还在上坡）→ **不是**极值，y 方向还能往下走。

### 【数学】Slide 40 练习展开 · z = x² + 2xy - 3y²

PPT 给了 3 道练习题，挑第 1 题完整展开（带乘积项 $2xy$，最考验「其它当常数」纪律）：

**对 x 偏导**（y 当常数）：

| 项 | 拆解 | 结果 |
|---|---|---|
| $x^2$ | 幂函数 → $2x^{2-1}$ | $2x$ |
| $2xy$ | y 是常数，整项相当于 $(2y) \cdot x$，对 x 求导 → 系数 $2y$ | $2y$ |
| $-3y^2$ | 不含 x，整体当常数 → 求导 0 | $0$ |

$$\frac{\partial z}{\partial x} = 2x + 2y$$

**对 y 偏导**（x 当常数）：

| 项 | 拆解 | 结果 |
|---|---|---|
| $x^2$ | 不含 y → 0 | $0$ |
| $2xy$ | x 是常数 → $(2x) \cdot y$ 对 y 求导 → $2x$ | $2x$ |
| $-3y^2$ | $-3 \cdot 2y$ | $-6y$ |

$$\frac{\partial z}{\partial y} = 2x - 6y$$

> 另两题答案（PPT 已给，不再展开）：
> - $z = x^y$：$\partial z/\partial x = y\,x^{y-1}$，$\partial z/\partial y = x^y \ln x$
> - $z = e^{2x^2 - 3y}$：$\partial z/\partial x = 4x\,e^{2x^2-3y}$，$\partial z/\partial y = -3\,e^{2x^2-3y}$（链式法则——参考 [`03-导数.md`](./03-导数.md)）

**新手最常踩的坑**：$2xy$ 对 x 求导**不是 $2y$ 而是 $2y$**（系数）—— 这次说对了。但很多人会写成 $2$（漏了 y）或 $2x \cdot y'$（把 y 当变量）。**口诀：把 y 当成数字 7，$2 \cdot 7 \cdot x$ 对 x 求导显然是 14（=2y），不是 2 也不是 0。**

### 【数学】梯度 · 把所有偏导打包成向量（Slide 48）

**梯度 gradient** 就是把**所有偏导塞进一个向量**——多元函数斜率的"全家福"：

$$\nabla z(x, y) = \begin{bmatrix} \partial z / \partial x \\ \partial z / \partial y \end{bmatrix}$$

回到 Slide 39 那个碗：

$$\nabla z(x, y) = \begin{bmatrix} 2(x-2) \\ 2(y-3) \end{bmatrix}$$

代入点 $(0, 0)$：$\nabla z(0,0) = [-4, -6]^T$——意思是在原点，函数值上升最快的方向是 $(-4, -6)$（指向左下方……等等，碗底在 $(2,3)$，从 $(0,0)$ 看碗底在右上，那么"上升最快"方向应该指**远离碗底**才对——$(-4, -6)$ 指向左下，正好远离 $(2,3)$，符合"上升最快"）。

**梯度方向的几何意义**——在 (w, b) 平面上：

```
    b
    ↑                等高线 = Loss 一样的圈
    │      ╭─────────────╮
    │    ╭─╯              ╰─╮
    │   │   ╭────────╮      │
    │   │  │  ╭───╮   │     │       ↗ ∇L
    │   │  │ │  ★  │  │     │      ╱ 上升最快方向（远离碗底）
    │   │  │  ╰───╯   │     │     ╱
    │   │   ╰────────╯      │    ●  当前点 (w, b)
    │   │                   │   ╱
    │   │                   │  ╱
    │    ╰─╮              ╭─╯ ╱ -∇L（反方向）
    │      ╰─────────────╯  ↙   下降最快方向 → 碗底
    └────────────────────────────────→ w
        ★ = (w*, b*) 碗底（最优解）

    几何铁律（不证）：
    1. 梯度 ∇L 方向 = 函数上升最快的方向
    2. 梯度方向 ⊥ 等高线（垂直于等高线）
    3. 反梯度 -∇L 方向 = 下降最快的方向 ← 这是 04b-gd 梯度下降的核心
```

**记忆口诀**：**梯度指上山，反梯度指下山**。LR 训练要的是**最低点**，所以走 $-\nabla L$ 方向——这一句话就是 [`04b-gd/`](../04b-gd/) 梯度下降的全部直觉。

### 【数学】主线案例 · LR 损失对 (w, b) 求偏导

回到 [`01-损失函数.md`](./01-损失函数.md) 的主战场——5 样本身高→体重的损失：

| i | $x_i$（身高）| $y_i$（体重）|
|---|---|---|
| 1 | 160 | 56.3 |
| 2 | 166 | 60.6 |
| 3 | 172 | 65.1 |
| 4 | 174 | 68.5 |
| 5 | 180 | 75.0 |

$$L(w, b) = \sum_{i=1}^{5} (w\,x_i + b - y_i)^2$$

这是**(w, b) 的二元函数**——一只 2D 碗。01 章固定 b=-100 走过 1D 抛物线，现在两个变量一起处理。

**对 w 偏导**（b 当常数；外层 $u^2$ 对 u 求导 = $2u$，内层 $u = w x_i + b - y_i$ 对 w 求导 = $x_i$，链式法则）：

$$\boxed{\;\frac{\partial L}{\partial w} = \sum_{i=1}^{5} 2(w\,x_i + b - y_i) \cdot x_i\;}$$

**对 b 偏导**（w 当常数；内层 $u$ 对 b 求导 = $1$）：

$$\boxed{\;\frac{\partial L}{\partial b} = \sum_{i=1}^{5} 2(w\,x_i + b - y_i) \cdot 1\;}$$

**梯度**——把上面两个偏导打包：

$$\nabla L(w, b) = \begin{bmatrix} \partial L / \partial w \\ \partial L / \partial b \end{bmatrix} = \begin{bmatrix} 2 \sum_i x_i (w\,x_i + b - y_i) \\ 2 \sum_i (w\,x_i + b - y_i) \end{bmatrix}$$

**LR 闭式解的核心一招**——**令两个偏导同时 = 0**：

$$\begin{cases} \sum_i x_i (w\,x_i + b - y_i) = 0 \\ \sum_i (w\,x_i + b - y_i) = 0 \end{cases} \;\Longrightarrow\; (w^*, b^*)$$

这就是 [`04a-analytical/`](../04a-analytical/) 正规方程的雏形——一组关于 $(w, b)$ 的线性方程，解出来就是最优解。具体怎么解（用矩阵形式 $\mathbf{w} = (X^T X)^{-1} X^T \mathbf{y}$）留给 04a，这里只点穿**思路**：**偏导 = 0 是 LR 闭式解的发动机**。

### 通向 04a / 04b 的两条路

同一个梯度，两条用法：

| 路径 | 怎么用 ∇L | 主场章节 |
|---|---|---|
| **解析解** | 令 $\nabla L = \mathbf{0}$，**直接解方程组**得 $(w^*, b^*)$ | [`04a-analytical/`](../04a-analytical/) 正规方程 |
| **数值解** | 沿 **$-\nabla L$** 反方向小步迭代：$w \leftarrow w - \alpha \cdot \partial L/\partial w$；$b \leftarrow b - \alpha \cdot \partial L/\partial b$ | [`04b-gd/`](../04b-gd/) 梯度下降 |

**为什么需要数值解**？多特征 LR 有 n 个 w + 1 个 b，方程组维度大、$X^T X$ 可能不可逆，直接解析解吃不消——这时候沿 $-\nabla L$ 一步步滚下山是更通用的招数（→ [`04b-gd/`](../04b-gd/) 详谈）。

> 推迟标记：多特征版梯度的矩阵形式 $\nabla L = 2 X^T (X\mathbf{w} - \mathbf{y})$ 用到 [`05-向量.md`](./05-向量.md) + [`06-矩阵.md`](./06-矩阵.md) 之后才好写——本节先到二元 (w, b)。

### 常见混淆 / 坑

- **$d$ vs $\partial$**——单变量用 $d$（$\frac{df}{dx}$），多变量偏导用 $\partial$（$\frac{\partial f}{\partial x}$）。机制一样，符号不能混（数学老师会扣分；工程上一眼区分"上下文是几个变量"也靠它）。
- **「其它变量当常数」别忘了乘法项**——$3xy$ 对 x 求导，y 是常数，$3y$ 是**系数**跟着 x，导数 = $3y$。**新手最常漏 y**（写成 $3$）或**误把 y 当变量**（写成 $3y'$）。把 y 想成数字 7。
- **梯度是向量，导数是标量**——$\nabla L$ 形状和参数向量一致（n+1 维），可以**直接做向量减法**——这就是梯度下降一行代码 `w = w - lr * grad` 的数学基础（→ [`04b-gd/`](../04b-gd/)）。
- **梯度方向 ⊥ 等高线**——梯度永远**垂直于等高线**且指向**上升**最快。下山就走反方向。可视化在 [`04b-gd/`](../04b-gd/)。
- **偏导 = 0 是必要条件不是充分条件**——所有偏导 = 0 的点叫**驻点** stationary point，可能是极小、极大、或鞍点 saddle point（一个方向像谷、另一个方向像峰）。LR 的损失是**凸碗**（[`01-损失函数.md`](./01-损失函数.md) 已建立），所以驻点就是全局最小，没鞍点烦恼。非凸函数（神经网络）就要小心。
