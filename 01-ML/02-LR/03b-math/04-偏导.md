# 偏导

> 级别：【知道】 · 维度：数学 · 底稿见 [`README.md`](./README.md)（对应 slide 39-40）

## 底稿（精简）

- 偏导 partial derivative：**多元函数对其中一个变量求导，其它变量当常数**
- 记号：$\frac{\partial f}{\partial x}$（读作「partial f partial x」）
- 梯度 gradient：**所有偏导组成的向量**，记作 $\nabla f$
- 梯度方向 = 函数上升最快的方向；反方向 = 下降最快的方向

---

## 直觉（一句话）

**偏导 = 多元函数版的「导数」**——只盯一个变量求导，其它变量都按住不动当常数。

一元函数 $f(x)$ 只有一个方向可以走（左/右），导数 $f^\prime(x)$ 一个数说完。多元函数 $f(w, b)$ 有两个方向（w 方向 / b 方向），所以要分别求两个偏导，组成的向量叫**梯度**。

## 公式 + 例子

设 $f(x, y) = x^2 + 3xy + y^2$。

**对 x 偏导**（y 当常数）：

$$\frac{\partial f}{\partial x} = 2x + 3y \cdot 1 + 0 = 2x + 3y$$

逐项分析：$x^2 \to 2x$（幂函数）；$3xy$ 把 y 当常数 → 系数是 $3y$，对 x 求导 → $3y$；$y^2$ 不含 x → 0。

**对 y 偏导**（x 当常数）：

$$\frac{\partial f}{\partial y} = 0 + 3x + 2y = 3x + 2y$$

**梯度**（把两个偏导塞进列向量）：

$$\nabla f(x, y) = \begin{bmatrix} \partial f / \partial x \\ \partial f / \partial y \end{bmatrix} = \begin{bmatrix} 2x + 3y \\ 3x + 2y \end{bmatrix}$$

在点 $(1, 1)$：$\nabla f(1, 1) = [5, 5]^T$——意思是从 $(1,1)$ 出发，沿 $(5, 5)$ 方向走 f 上升最快。

## 与 LR 的连接

LR 的 MSE 是 $(w, b)$ 的二元函数：

$$L(w, b) = \sum_{i=1}^{m} (w x_i + b - y_i)^2$$

**对 w 偏导**（b 当常数，用链式法则）：

$$\frac{\partial L}{\partial w} = \sum_{i=1}^{m} 2(w x_i + b - y_i) \cdot x_i$$

**对 b 偏导**（w 当常数）：

$$\frac{\partial L}{\partial b} = \sum_{i=1}^{m} 2(w x_i + b - y_i) \cdot 1$$

**梯度**：

$$\nabla L(w, b) = \begin{bmatrix} \partial L / \partial w \\ \partial L / \partial b \end{bmatrix}$$

接下来分两条路用这个梯度：

| 路径 | 怎么用梯度 | 章节 |
|---|---|---|
| **解析解** | 令 $\nabla L = \mathbf{0}$（向量为零 = 两个偏导都为零），解方程组 | [`04a-analytical/`](../04a-analytical/) 正规方程 |
| **数值解** | 沿 $-\nabla L$ 方向迭代：$w \leftarrow w - \alpha \cdot \partial L / \partial w$ | [`04b-gd/`](../04b-gd/) 梯度下降 |

多特征 LR（n 个 w + 1 个 b）的梯度就是 n+1 维向量——所以矩阵化（[`06-矩阵.md`](./06-矩阵.md)）才是真正干活的形式。

## 常见混淆 / 坑

- **偏导 d 和 ∂ 的写法**——一元导数用 $d$（$\frac{df}{dx}$），多元偏导用 $\partial$（$\frac{\partial f}{\partial x}$）。混用会被批改老师扣分，但意思一样。
- **「其它变量当常数」别忘了乘法项**——$3xy$ 对 x 求导时，y 是常数，所以 $3y$ 是「系数」一起跟着 x，导数是 $3y$ 不是 $3$。新手常漏掉。
- **梯度是向量，导数是标量**——梯度 $\nabla L$ 形状和参数向量一致（n+1 维），所以可以**直接相减**做参数更新（梯度下降的核心一行 `w = w - lr * grad`）。
- **梯度方向 vs 等高线**——梯度方向**垂直于等高线**且指向函数值上升最快的方向。这是梯度下降「沿等高线法线下山」的几何直觉，[`04b-gd/`](../04b-gd/) 会画图。
