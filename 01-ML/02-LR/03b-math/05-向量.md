# 向量

> 级别：【知道】 · 维度：数学 · 底稿见 [`README.md`](./README.md)（对应 slide 41-42）

## 底稿（精简)

- 向量 vector：一列顺序排列的元素，**有大小有方向**（模 + 夹角）
- 默认是**列向量**（ML 约定）
- 基本运算：加 / 减（同维逐元素）、点积（标量结果）
- 范数 norm：向量的「长度」概念。L1 = 元素绝对值之和；**L2 = 模长 = 平方和开根**
- 转置 transpose：列向量横过来变行向量，记作 $\mathbf{x}^T$

---

## 直觉（一句话）

**向量 = 一组数 + 一个方向**——既能当「特征列表」用（数据视角），也能当「箭头」画（几何视角）。

LR 里向量的核心用途：把 n 个特征 $x_1, \ldots, x_n$ 和 n 个权重 $w_1, \ldots, w_n$ 各打包成一个向量，**预测公式从 n 项加法 → 一个点积**。

## 公式 + 例子

设 $\mathbf{a} = [1, 2, 3]^T$，$\mathbf{b} = [4, 5, 6]^T$。

**加法 / 减法**（同维度逐元素）：

$$\mathbf{a} + \mathbf{b} = [1+4, 2+5, 3+6]^T = [5, 7, 9]^T$$

**点积 / 内积**（结果是**标量**）：

$$\mathbf{a} \cdot \mathbf{b} = \mathbf{a}^T \mathbf{b} = \sum_{i=1}^{n} a_i b_i = 1 \cdot 4 + 2 \cdot 5 + 3 \cdot 6 = 32$$

**L2 范数**（向量长度，欧氏距离的根源）：

$$\lVert \mathbf{a} \rVert_2 = \sqrt{\mathbf{a}^T \mathbf{a}} = \sqrt{1 + 4 + 9} = \sqrt{14} \approx 3.74$$

**L1 范数**（绝对值之和）：$\lVert \mathbf{a} \rVert_1 = 1 + 2 + 3 = 6$。

**几何意义**：$\mathbf{a} \cdot \mathbf{b} = \lVert \mathbf{a} \rVert \lVert \mathbf{b} \rVert \cos \theta$——点积反映两向量的「方向相似度」。同向 cos = 1，垂直 cos = 0，反向 cos = -1。

## 与 LR 的连接

**LR 预测公式向量化**：

$$\boxed{\hat y = \mathbf{w}^T \mathbf{x} + b}$$

- $\mathbf{x} = [x_1, \ldots, x_n]^T$：n 个特征的列向量
- $\mathbf{w} = [w_1, \ldots, w_n]^T$：n 个权重的列向量
- $\mathbf{w}^T \mathbf{x}$：点积，得到一个标量
- $b$：偏置标量

展开就是 $\hat y = w_1 x_1 + w_2 x_2 + \cdots + w_n x_n + b$，**和原始公式完全等价**，但向量写法在 numpy 里就是一行 `w @ x + b`，而且能直接推广到 m 个样本（[`06-矩阵.md`](./06-矩阵.md) 的 $X \mathbf{w}$）。

**L2 范数在 LR 里出现两次**：

1. **MSE 损失** = 残差向量 $\mathbf{r} = X\mathbf{w} - \mathbf{y}$ 的 L2 范数平方再除 m：$L = \frac{1}{m} \lVert X\mathbf{w} - \mathbf{y} \rVert_2^2$。
2. **L2 正则化（Ridge 回归）**——给 Loss 加一项 $\lambda \lVert \mathbf{w} \rVert_2^2$ 限制权重大小，防过拟合。后续正则化章节展开。

**L1 范数 → Lasso 回归**：用 $\lambda \lVert \mathbf{w} \rVert_1$ 做正则，会让一些 $w_i$ 直接变 0（特征选择）。

## 常见混淆 / 坑

- **点积 vs 逐元素乘**——点积 $\mathbf{a}^T \mathbf{b}$ 结果是标量；逐元素乘（Hadamard 积，$\mathbf{a} \odot \mathbf{b}$）结果还是向量。numpy 里 `a @ b` 是点积，`a * b` 是逐元素乘——**符号不同，结果完全不同**，新手最常踩。
- **「列向量默认」在 numpy 不成立**——numpy 1D array `np.array([1,2,3])` 没行列概念，做矩阵乘时自动适配。要严格行列时用 2D（`(n, 1)` 或 `(1, n)`）。
- **「点积」「内积」「数量积」是同一个东西**——三个名字看心情用，结果都是 $\sum a_i b_i$。
- **L2 范数和 L2 范数平方常被混用**——MSE 用的是平方（避免开根求导麻烦），公式写 $\lVert \cdot \rVert_2^2$ 别忘那个上标 2。
