- 向量基本运算：加减、点积
- **向量转置 Transpose**：$T_{ij} \leftrightarrow T_{ji}$，形状 m×n → n×m
- 机器学习领域：**默认是列向量**

例：$X = \begin{bmatrix}1\\2\\-3\end{bmatrix}$，则 $X^T = (1,\,2,\,-3)$

#### Slide 42 · 向量和矩阵 – 范数 Norm

范数（norm）是数学中的基本概念，**具有长度的意义**。

设 $x^T = (1, 2, -3)$：

- **L1 范数**（各元素绝对值之和）：$\lVert x \rVert_1 = |1| + |2| + |-3| = 6$
- **L2 范数**（向量模长，平方和开根）：$\lVert x \rVert_2 = \sqrt{1^2 + 2^2 + (-3)^2} = \sqrt{14}$
- **Lp 范数**：$\lVert x \rVert_p = \left(\sum |x_i|^p\right)^{1/p}$

注意：$x^T x = 1^2 + 2^2 + (-3)^2 = 14$（向量的转置 @ 向量，**不能反过来**），与 $\lVert x \rVert_2^2$ 一致。

### 笔记

> 【知道】向量

**向量加法 / 减法**（同维度逐元素）：

$$\mathbf{a} + \mathbf{b} = [a_1 + b_1, a_2 + b_2, \ldots, a_n + b_n]^T$$

**点积 / 内积**（结果是标量）：

$$\mathbf{a} \cdot \mathbf{b} = \mathbf{a}^T \mathbf{b} = \sum_{i=1}^{n} a_i b_i$$

**向量 L2 范数**（向量长度）：

$$\lVert \mathbf{a} \rVert_2 = \sqrt{\sum a_i^2} = \sqrt{\mathbf{a}^T \mathbf{a}}$$

**线性回归用向量做什么**：把 n 个特征 $x_1, x_2, \ldots, x_n$ 写成列向量 $\mathbf{x}$，权重 $\mathbf{w}$ 同维列向量，预测值就是 $\hat{y} = \mathbf{w}^T \mathbf{x} + b$，一行公式覆盖所有维度。

## ━━━━━━━━ 讲解 ━━━━━━━━

### 生活锚 · 一张成绩单就是一个向量

期末考完，张三的成绩单写着：**语文 70 / 数学 80 / 英语 90**。把这三个数按顺序竖着摆，就是一个**向量** vector：

```
         张三                    李四
         ┌──┐                  ┌──┐
   语文  │70│                  │60│
   数学  │80│                  │95│
   英语  │90│                  │85│
         └──┘                  └──┘
```

围绕这张成绩单能干的事，几乎就是本节所有运算的生活原型——

| 生活动作 | 向量运算 |
|---|---|
| 班级排名分加 5 分调分 → 每科都 +5 | **向量加法**（同维度逐元素） |
| 张三 vs 李四，每科分差 | **向量减法** |
| 中考折算总分：语文 ×0.3 + 数学 ×0.4 + 英语 ×0.3 = 加权总分 | **点积** dot product（一个标量） |
| 「这个同学的综合实力多强」用一个数概括 | **范数** norm（向量的「长度」） |

> 上一节 [`02-数据表述.md`](./02-数据表述.md) 已经知道**向量 = 一列数**——本节把这列数能做的运算吃透，让 LR 的「ŷ = wᵀx + b」从「n 项加法」压成「一个点积」。

### 业务问题 · 多特征 LR 的写法压力

[`02-api/`](../02-api/) 里多特征 LR 已经登场——预测房价不止用面积，还有楼层、地段、房龄……n 个特征堆上去。如果每加一个特征就在公式后面接一项加法，板书会变成：

```
  ŷ = w₁·面积 + w₂·楼层 + w₃·地段 + w₄·房龄
     + w₅·学区 + w₆·朝向 + w₇·装修 + … + b
```

n=50 时这一行写不完，n=1000（推荐系统的特征数）时根本没法在白板上推。工程视角的痛点很具体：

- **代码层面**：50 个特征要写 50 个 `w[i] * x[i]` 求和，循环慢、出错率高
- **板书 / 推导**：写不下，一行公式塞满，求导时 50 个偏导对不齐
- **多样本扩展**：m 个样本 × n 个特征想一次算完——n 项加法的写法直接拒绝合作

向量的工作就是把「n 项加法」打包成**一个名字**，板书一行、代码一行、扩展到样本矩阵也一行。下面把向量的运算工具掰开。

### 【数学】数学符号扫盲

下面公式会反复出现这 4 个符号，先扫一遍：

| 符号 | 读法 | 直觉 | 例子 |
|---|---|---|---|
| $\mathbf{x}^T$ | 「x transpose」 | 把列向量躺倒变行向量（或反之）| $\begin{bmatrix}1\\2\\3\end{bmatrix}^T = (1,2,3)$ |
| $\mathbf{x}^T \mathbf{y}$ 或 $\sum x_i y_i$ | 「x 转置乘 y」/「sigma x i y i」 | 对应位置相乘**再求和**，结果是一个数 | $(1,2,3)\cdot(4,5,6) = 4+10+18 = 32$ |
| $\lVert \mathbf{x} \rVert$ | 「x 的范数」 | 向量的「长度 / 大小」（一个非负数）| 下标 $\lVert \cdot \rVert_2 = \sqrt{14}$ |
| $\mathbf{x}^T \mathbf{x}$ | 「x 转置乘 x」 | 自己和自己点积 = 平方和 = $\lVert x \rVert_2^2$ | $(1,2,-3)$ → $1+4+9=14$ |

> 上一节 03 / 04 学过 $\sum$ 和 $\partial$；本节新增的就是 $T$（转置）和 $\lVert \cdot \rVert$（范数）两个符号。

### 【数学】向量加减 · 几何意义

**加减规则**（同维度逐元素）：

$$\mathbf{a} + \mathbf{b} = [a_1+b_1,\, a_2+b_2,\, \ldots,\, a_n+b_n]^T$$

例：$\mathbf{a} = (1,1)^T$，$\mathbf{b} = (1,2)^T$（PPT Slide 41 的 A、B 两点），

$$\mathbf{a} + \mathbf{b} = (2, 3)^T$$

**几何意义——平行四边形法则**：把两个向量当箭头从原点出发，加法 = 沿着 a 走完再沿着 b 走完，终点连原点就是 a + b：

```
   y ↑
   3 │              ★ a+b = (2, 3)
     │            ╱ ╱
   2 │       ●b=(1,2)
     │     ╱   ╱
     │   ╱   ╱   ← 把 b 平移到 a 的终点
   1 │ ●a=(1,1)
     │ │  ╱
     │ │╱
   0 ┼─┴────●────────→ x
     0      1      2

   两条边：原 a + 平移过来的 b
   对角线：a + b = (2, 3)（终点）
```

减法同理：$\mathbf{a} - \mathbf{b} = \mathbf{a} + (-\mathbf{b})$（把 b 反向再加）。

**反例**：维度不一致的向量**不能**相加——$(1,2)^T + (3,4,5)^T$ 没有定义，numpy 里直接报 shape mismatch。

### 【数学】点积 · 一行公式覆盖加权求和

点积 dot product（也叫**内积** inner product / 数量积——三个名字同一个东西）：

$$\mathbf{a} \cdot \mathbf{b} \;=\; \mathbf{a}^T \mathbf{b} \;=\; \sum_{i=1}^{n} a_i b_i$$

**例**：$\mathbf{a} = (1,2,3)^T$，$\mathbf{b} = (4,5,6)^T$，

$$\mathbf{a}^T \mathbf{b} = 1\cdot 4 + 2\cdot 5 + 3\cdot 6 = 4 + 10 + 18 = \mathbf{32}$$

**结果是一个标量**（不是向量）——这是点积的核心性质，名字「内积」也是这意思：两个向量「内进去」吐出一个数。

**几何意义——投影 + 夹角**：

$$\mathbf{a} \cdot \mathbf{b} = \lVert \mathbf{a} \rVert \, \lVert \mathbf{b} \rVert \, \cos\theta$$

θ 是两向量的夹角。三种典型情况：

```
   ┌─同向 θ=0°─┐    ┌─垂直 θ=90°─┐    ┌─反向 θ=180°┐
        ●→●→            ●↑              ●→ ●←
        ●→              │                ●→ ●←
        cos=1           ●→               cos=-1
        a·b 最大        cos=0            a·b 最负
        (方向最像)      a·b = 0          (方向最反)
                        (彼此无关)
```

**直觉**：点积 = 「两个向量方向有多像」 × 「两个向量都多大」。同向时 a·b 拉到最大，垂直时归零，反向时变最负。

> 推荐系统里「用户向量 · 商品向量」就是这件事——方向越像，cos 越大，越该推。

### 【数学】向量转置 · 列变行 / 行变列

**转置** transpose（符号 $T$ 或 ′）：把列向量躺倒成行向量，反之亦然。

$$\mathbf{x} = \begin{bmatrix}1\\2\\-3\end{bmatrix} \quad\Longleftrightarrow\quad \mathbf{x}^T = (1,\, 2,\, -3)$$

形状变化：n×1 → 1×n（或反之）。

**机器学习默认列向量**——这是行业约定，写公式 / 推导都按列向量起手。需要点积写成 $\mathbf{x}^T \mathbf{y}$ 时，把第一个 x 转置成行向量（1×n），与列向量 y（n×1）相乘 → 1×1 标量，正好。

> 后续 [`06-矩阵.md`](./06-矩阵.md) 会看到：转置不只在向量上用，矩阵也有转置 $A^T$，性质 $(AB)^T = B^T A^T$ 等——本节先把向量转置吃透，矩阵那边只是推广。

### 【数学】范数 · 给向量打一个「大小分」

**范数** norm 是把向量映射成一个**非负标量**的函数，直觉就是「向量的长度 / 综合大小」——成绩单的「综合实力分」就是范数。

#### L1 范数 · 各元素绝对值之和

$$\lVert \mathbf{x} \rVert_1 \;=\; \sum_{i=1}^{n} |x_i|$$

**实算**（Slide 42 例：$\mathbf{x}^T = (1, 2, -3)$）：

$$\lVert \mathbf{x} \rVert_1 = |1| + |2| + |-3| = 1 + 2 + 3 = \mathbf{6}$$

直觉：曼哈顿出租车距离——只能横竖走，从 (0,0) 到 (1,2) 走 1+2=3 格，绕不开。

#### L2 范数 · 平方和开根 = 模长

$$\lVert \mathbf{x} \rVert_2 \;=\; \sqrt{\sum_{i=1}^{n} x_i^2} \;=\; \sqrt{\mathbf{x}^T \mathbf{x}}$$

**实算**：

$$\lVert \mathbf{x} \rVert_2 = \sqrt{1^2 + 2^2 + (-3)^2} = \sqrt{1+4+9} = \sqrt{\mathbf{14}} \approx 3.74$$

直觉：欧几里得距离——从 (0,0,0) 直线飞到 (1,2,-3) 的真实距离。也叫**模长**（向量的「长度」）。

**重要等式**（PPT Slide 42 强调）：

$$\boxed{\;\mathbf{x}^T \mathbf{x} \;=\; \lVert \mathbf{x} \rVert_2^2\;}$$

「自己点自己 = L2 范数的平方」——在 LR 的损失函数推导里随处可见，记住这等式，正规方程就读得顺。

> 顺序敏感：$\mathbf{x}^T \mathbf{x}$（1×n 乘 n×1 = 标量）≠ $\mathbf{x}\, \mathbf{x}^T$（n×1 乘 1×n = n×n 矩阵），**不能反过来**。

#### Lp 范数 · 推广形式

$$\lVert \mathbf{x} \rVert_p = \left(\sum_{i=1}^{n} |x_i|^p\right)^{1/p}$$

- p=1 → L1 范数
- p=2 → L2 范数
- p=∞ → L∞（最大绝对值）

实战 99% 只用 L1 / L2，Lp 知道有这玩意就够。

#### 几何对比 · L1 是菱形 / L2 是圆

「向量长度 = 1」这件事在两种范数下画出来不一样——这张图直接对应后面 L1 / L2 正则化的几何直觉：

```
        L1 单位「圆」 = 菱形              L2 单位「圆」 = 真圆
           ‖x‖₁ = 1                        ‖x‖₂ = 1

              y                                y
              │                                │
            1 ●                                ╭─╮
             ╱ ╲                            ╭─╯   ╰─╮
            ╱   ╲                          │         │
        ◀──●─────●──▶ x                ◀───●         ●───▶ x
            ╲   ╱                          │         │
             ╲ ╱                            ╰─╮   ╭─╯
            -1 ●                               ╰─╯
              │                                │

      4 个尖角（轴上）                  圆滑、无尖角
      L1 正则化倾向把权重压到 0          L2 正则化等比缩水所有权重
      → 特征选择（Lasso）              → 防过拟合（Ridge）
```

**关键观察**：L1 的「单位圆」是菱形，4 个尖角恰好落在坐标轴上——优化时和 Loss 等高线相交点容易卡在尖角，对应某个 $w_i = 0$（特征被剔除）。L2 是圆滑的，相交点一般不在轴上——所有 $w_i$ 等比缩小但不归零。

> 这张图是 [`../07a-regularization/`](../07a-regularization/) L1 / L2 正则化几何直觉的伏笔——本节先建立形状直觉，正则化主场再展开「为什么 L1 能特征选择」。

### LR 主线 · 5 样本写成向量 + ŷ = wᵀx + b

把 [`01-损失函数.md`](./01-损失函数.md) 的 5 样本身高 / 体重数据**向量化**：

$$\mathbf{x} = \begin{bmatrix}160\\166\\172\\174\\180\end{bmatrix} \quad\quad \mathbf{y} = \begin{bmatrix}56.3\\60.6\\65.1\\68.5\\75.0\end{bmatrix}$$

跨章节同源数据（指南 §7 数据自洽）——这两个 5×1 列向量在 [`04a-analytical/`](../04a-analytical/) 正规方程、[`04b-gd/`](../04b-gd/) 梯度下降都会再用。

**多特征 LR 预测公式向量化**（Slide 26 + [`02-api/`](../02-api/)）：

$$\boxed{\;\hat y \;=\; \mathbf{w}^T \mathbf{x} + b\;}$$

| 元素 | 形状 | 含义 |
|---|---|---|
| $\mathbf{x} = [x_1, \ldots, x_n]^T$ | n×1 列向量 | n 个特征值 |
| $\mathbf{w} = [w_1, \ldots, w_n]^T$ | n×1 列向量 | n 个权重 |
| $\mathbf{w}^T \mathbf{x}$ | 标量 | 点积——加权求和 |
| $b$ | 标量 | 偏置 / 截距 |

展开就是 $\hat y = w_1 x_1 + w_2 x_2 + \cdots + w_n x_n + b$——**和原始公式完全等价**，但板书一行、numpy 一行 `w @ x + b`，n=1000 也不变长。

**L2 范数在 LR 里出现两次**——这是范数和 LR 真正的接口：

1. **MSE 损失** = 残差向量 $\mathbf{r} = \mathbf{y} - X\mathbf{w}$ 的 L2 范数平方再除 m：
   $$L(\mathbf{w}) = \frac{1}{m} \lVert \mathbf{y} - X\mathbf{w} \rVert_2^2$$
   即 [`01-损失函数.md`](./01-损失函数.md) 的 MSE 用向量写法的版本。
2. **Ridge 回归 / L2 正则化**——给 Loss 加一项 $\lambda \lVert \mathbf{w} \rVert_2^2$ 限制权重大小，防过拟合（→ [`../07a-regularization/`](../07a-regularization/)）。
3. **Lasso 回归 / L1 正则化**——用 $\lambda \lVert \mathbf{w} \rVert_1$，会让一些 $w_i$ **直接变 0**（特征选择，菱形尖角的几何意义）。

### 常见混淆 / 坑

- **点积 vs 逐元素乘**——点积 $\mathbf{a}^T \mathbf{b}$ 结果是**标量**；逐元素乘（**Hadamard 积** $\mathbf{a} \odot \mathbf{b}$）结果还是**向量**。numpy 里 `a @ b` 是点积，`a * b` 是逐元素乘——符号差一个，结果完全不同，新手最常踩。
- **「列向量默认」在 numpy 不严格成立**——numpy 1D array `np.array([1,2,3])` 没有行 / 列概念，做矩阵乘时自动适配 shape。要严格区分行 / 列时用 2D：`x.reshape(-1, 1)` 或 `x[:, None]`。
- **「点积」「内积」「数量积」是同一个东西**——三个名字看心情用，结果都是 $\sum a_i b_i$。看到任何一个不要慌。
- **L2 范数 vs L2 范数平方**——MSE 用的是**平方** $\lVert \cdot \rVert_2^2$（避免开根求导麻烦），公式里别忘那个上标 2；Ridge 正则项也是平方，单写 $\lVert \mathbf{w} \rVert_2$（不平方）的写法见到要警惕。
- **$\mathbf{x}^T \mathbf{x}$ ≠ $\mathbf{x}\,\mathbf{x}^T$**——前者是标量（平方和），后者是 n×n 矩阵（外积）。**顺序不能反**。
