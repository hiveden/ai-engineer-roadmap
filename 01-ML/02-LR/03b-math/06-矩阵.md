## ━━━━━━━━ 底稿 ━━━━━━━━

### PPT

#### Slide 43 · 矩阵 1 – 加减与乘法

矩阵：m 行 n 列的数据，记 $A \in \mathbb{R}^{m\times n}$。机器学习数据集 $X \in \mathbb{R}^{N\times D}$（N 行样本、D 个特征）。

- **加减**：同形对应位置相加减
- **乘法**（**非常重要！**）：A: m×k @ B: k×n → C: m×n，对应行列元素相乘求和

$$C_{ij} = \sum_{p=1}^{k} A_{ip} B_{pj}$$

#### Slide 44 · 矩阵 2 – 转置 / 方阵 / 单位阵

设 $A = \begin{bmatrix}1&2&3\\4&5&6\end{bmatrix} \in \mathbb{R}^{2\times 3}$，则 $A^T = \begin{bmatrix}1&4\\2&5\\3&6\end{bmatrix} \in \mathbb{R}^{3\times 2}$。

**A @ Aᵀ 是方阵**（2×2）：

$$A A^T = \begin{bmatrix}1\cdot1+2\cdot2+3\cdot3 & 1\cdot4+2\cdot5+3\cdot6\\ 4\cdot1+5\cdot2+6\cdot3 & 4\cdot4+5\cdot5+6\cdot6\end{bmatrix} = \begin{bmatrix}14&32\\32&77\end{bmatrix}$$

**Aᵀ @ A 也是方阵**（3×3）：

$$A^T A = \begin{bmatrix}17&22&27\\22&29&36\\27&36&45\end{bmatrix}$$

- **方阵**：行数 = 列数
- **单位阵 I（或 E）**：主对角线为 1、其他为 0，例 $I_{3\times 3} = \begin{bmatrix}1&0&0\\0&1&0\\0&0&1\end{bmatrix}$
- **对称方阵**：沿主对角线对称，$a_{ij} = a_{ji}$
- **A @ Aᵀ 通常 ≠ Aᵀ @ A**

#### Slide 45 · 矩阵 3 – 乘法性质

- **不满足交换律**：A × B ≠ B × A
  - 例：A: m×n, B: n×m → A@B ∈ ℝ^(m×m), B@A ∈ ℝ^(n×n)，形状都不一样
  - 仅当 A、B 同阶方阵且极个别条件下才有 AB = BA
- **满足结合律**：A × (B × C) = (A × B) × C

#### Slide 46 · 矩阵的逆 + 单位阵

- **A @ I = I @ A = A**（与单位阵相乘等于自己）
- **逆矩阵**：若 A @ B = I，则 B 为 A 的逆矩阵，记 $A^{-1}$
- **不一定存在逆矩阵！**（仅方阵且满秩可逆）

例：$A = \begin{bmatrix}1&2\\3&4\end{bmatrix} \in \mathbb{R}^{2\times 2}$，可求 $A^{-1}$。

#### Slide 47 · 矩阵 4 – 转置的性质

- $(A^T)^T = A$
- $(A + B)^T = A^T + B^T$
- $(kA)^T = k A^T$
- $(AB)^T = B^T A^T$

例：$\left(\begin{bmatrix}1&2&3\\4&5&6\end{bmatrix} \cdot B\right)^T = B^T \cdot \begin{bmatrix}1&4\\2&5\\3&6\end{bmatrix}$

#### Slide 52 · 自检题（矩阵）

> **题 1**：有关矩阵相乘说法正确的是？
>
> A. 两个矩阵能相乘的条件为：A: m×n, B: d×m，A@B = C
> **B. A @ Aᵀ = C，Aᵀ @ A = D，则 C 和 D 都是方阵**
> C. A @ Aᵀ = C，Aᵀ @ A = D，则 C 和 D 都是方阵，而且 C = D
> D. 任意 A，A 的逆矩阵一定存在
>
> **答案：B**（A 错——条件应是中间维度一致 n=d；C 错——形状不同；D 错——非方阵或奇异方阵无逆）

> **题 2**：已知 $A = \begin{bmatrix}1&2&-1\\4&5&-1\end{bmatrix}$，$B = \begin{bmatrix}1&-1\\1&-1\\1&-1\end{bmatrix}$，求 A @ B。
>
> $$A B = \begin{bmatrix}1+2-1 & -1-2+1\\4+5-1 & -4-5+1\end{bmatrix} = \begin{bmatrix}2&-2\\8&-8\end{bmatrix}$$

> **题 3**：已知 $A = \begin{bmatrix}1&-1\\2&-1\end{bmatrix}$，求 $A^{-1}$。
>
> 二阶逆公式：$\begin{bmatrix}a&b\\c&d\end{bmatrix}^{-1} = \dfrac{1}{ad-bc}\begin{bmatrix}d&-b\\-c&a\end{bmatrix}$
>
> 行列式 $\det A = 1\cdot(-1) - (-1)\cdot 2 = 1$，所以
>
> $$A^{-1} = \begin{bmatrix}-1 & 1\\-2 & 1\end{bmatrix}$$
>
> 验证：$A A^{-1} = \begin{bmatrix}1&-1\\2&-1\end{bmatrix}\begin{bmatrix}-1&1\\-2&1\end{bmatrix} = \begin{bmatrix}1&0\\0&1\end{bmatrix} = I$ ✓

### 笔记

> 【知道】矩阵

**矩阵乘法**（A: m×k, B: k×n → C: m×n）：

$$C_{ij} = \sum_{p=1}^{k} A_{ip} B_{pj}$$

**转置**：$(A^T)_{ij} = A_{ji}$，行变列、列变行。

**逆矩阵**：仅方阵且满秩可逆，满足 $A A^{-1} = I$（单位阵）。

**线性回归用矩阵做什么**：把 m 个样本叠成矩阵 X (m×n)，标签叠成向量 y (m×1)，则模型变成 $\hat{\mathbf{y}} = X \mathbf{w}$。MSE 损失展开后求导得正规方程：

$$\mathbf{w} = (X^T X)^{-1} X^T \mathbf{y}$$

这个公式是 04a 的核心——**所有训练计算用一个矩阵公式搞定**，是 LR 解析解的精华。

### 生活锚 · 多人多门课的成绩单

教务处期末出全班成绩单——**5 个学生 × 3 门课**（语数英），写在一张表上：

```
            语   数   英
  张三 │   85   92   78 │
  李四 │   70   88   95 │
  王五 │   90   76   82 │
  赵六 │   65   80   70 │
  孙七 │   78   85   88 │
              ↑
       这就是矩阵 matrix——5 行 3 列的二维数据
```

学校要算每个学生的"加权总分"——语文 20%、数学 50%、英语 30%。一种笨办法是 5 个学生各算一遍 5×3 次乘加；聪明办法是把权重也写成一列 `[0.2, 0.5, 0.3]`，**整张成绩单 × 这一列权重 = 一列总分**——一次算完。

**这一次"打包计算"就是矩阵乘法**：把多人多事的批量运算压缩成一个表达式。

矩阵相比 05-向量学过的"一列数"，多了一个维度——**一行就是一个向量，多行并排就是矩阵**。LR 干的就是这件事：5 个样本身高数据并排成一张"X 表"，所有训练计算（正规方程）就靠这张表的乘、转置、求逆几下子完成。本节把矩阵的运算和性质吃透，下一章 04a-analytical 就能一行公式搞定 LR 解析解。

### 业务问题

LR 训练要解决的问题：5 个人的身高 → 体重，找最优 $(w, b)$ 让总罚款最少（01-损失函数已建立）。

**用循环写法**：5 个样本，每个样本算 $\hat y_i = w x_i + b$、算误差、累加平方——5 次循环、5 次加法、5 次平方。如果数据从 5 涨到 50000、特征从 1 涨到 100，循环就裂开了——不仅慢，写出来的代码也丑（三层 for 循环嵌套）。

**用矩阵写法**：把 5 个样本身高叠成一张 5 行 2 列的 X 表（列 1 = 身高、列 2 = 全 1 吸收偏置）、把 5 个真实体重叠成一列 y、把要找的 $(w, b)$ 也叠成一列——整套训练就一行：$\mathbf{w} = (X^T X)^{-1} X^T \mathbf{y}$。

```
   循环写法                           矩阵写法
   for i in range(m):                w = (X.T @ X).inv @ X.T @ y
       y_hat = w*x[i] + b            ↑
       err = y_hat - y[i]            一行解决 5 / 50000 / 5e8 个样本
       loss += err**2                所有训练计算压缩成 4 个矩阵操作
   ...                               （转置 → 乘 → 求逆 → 再乘）
```

**为什么要学矩阵**：因为 04a-analytical 的正规方程整张全是矩阵语言——不学矩阵就读不懂这一行公式，更别说写代码。本节把矩阵的"零件"（加减、乘、转置、单位阵、逆）一个个掰开，再组装成正规方程的形状链路。

### 【数学】数学符号扫盲

| 符号 | 读法 | 直觉 | 例子 |
|---|---|---|---|
| $A^T$ | 「A transpose / A 的转置」 | 行变列、列变行；m×n → n×m | $\begin{bmatrix}1&2&3\\4&5&6\end{bmatrix}^T = \begin{bmatrix}1&4\\2&5\\3&6\end{bmatrix}$ |
| $A^{-1}$ | 「A inverse / A 的逆」 | 矩阵世界的「除法」；满足 $A A^{-1} = I$ | $\begin{bmatrix}1&-1\\2&-1\end{bmatrix}^{-1} = \begin{bmatrix}-1&1\\-2&1\end{bmatrix}$ |
| $I$（或 $E$）| 「identity / 单位阵」 | 矩阵世界的「数字 1」；$AI = A$ | $I_2 = \begin{bmatrix}1&0\\0&1\end{bmatrix}$ |
| $\det A$ | 「determinant / 行列式」 | 一个标量，$\det A = 0$ ⇒ 不可逆 | 二阶 $\det\begin{bmatrix}a&b\\c&d\end{bmatrix} = ad - bc$ |
| $A \in \mathbb{R}^{m \times n}$ | 「A 是 m 行 n 列的实矩阵」 | 形状标签；行数 × 列数 | $X \in \mathbb{R}^{5 \times 2}$ = 5 个样本、2 列 |

### 【数学】矩阵的加减与乘法（Slide 43）

**加减**——两个**同形**矩阵对应位置加减（形状不一致直接报错）：

$$\begin{bmatrix}1&2\\3&4\end{bmatrix} + \begin{bmatrix}5&6\\7&8\end{bmatrix} = \begin{bmatrix}6&8\\10&12\end{bmatrix}$$

**乘法**——这是矩阵的主场，规则比加减复杂：

$$C_{ij} = \sum_{p=1}^{k} A_{ip} B_{pj}$$

「C 的第 i 行第 j 列 = A 的第 i 行 · B 的第 j 列」——两个向量做一次点积（05-向量的点积扩展）。

**形状对接图**——矩阵乘法最关键的直觉就这一张：

```
            A          @          B          =          C
        ┌─────┐               ┌─────────┐            ┌─────────┐
        │     │               │         │            │         │
   m=2  │  ●  │ × k=3    k=3  │  ●●●    │  × n=4 →   │         │  m=2 行 × n=4 列
        │     │               │         │            │         │
        └─────┘               └─────────┘            └─────────┘
         m × k                  k × n                   m × n

       ↑─── k 必须相等 ───↑
      （A 的列数 = B 的行数）

   规则记忆：「外面留下来，里面对掉」
       (m×k) @ (k×n) → (m×n)
            ↑─↑
         里面 k 消掉，外面 m、n 留下来组成结果形状
```

**反例**：$A: 2{\times}3 @ B: 4{\times}5$ → 中间 3 ≠ 4，**乘不了**，numpy 直接抛 `ValueError: shapes not aligned`。

**实算**——Slide 52 题 2（Slide 43 乘法直接演练）：

$$A = \begin{bmatrix}1&2&-1\\4&5&-1\end{bmatrix}_{2\times 3},\quad B = \begin{bmatrix}1&-1\\1&-1\\1&-1\end{bmatrix}_{3\times 2}$$

形状对接：(2×**3**) @ (**3**×2) → (2×2) ✓。逐项算：

$$AB = \begin{bmatrix}1{\cdot}1+2{\cdot}1+(-1){\cdot}1 & 1{\cdot}(-1)+2{\cdot}(-1)+(-1){\cdot}(-1)\\ 4{\cdot}1+5{\cdot}1+(-1){\cdot}1 & 4{\cdot}(-1)+5{\cdot}(-1)+(-1){\cdot}(-1)\end{bmatrix} = \begin{bmatrix}2&-2\\8&-8\end{bmatrix}$$

### 【数学】转置 / 方阵 / 单位阵 / 对称方阵（Slide 44）

**转置 transpose**——行列互换。$A: m{\times}n$ 转置后是 $A^T: n{\times}m$。

```
   原矩阵 A (2×3)              转置 Aᵀ (3×2)
   ┌1  2  3┐                   ┌1  4┐
   │4  5  6│       ──→         │2  5│
   └       ┘                   │3  6│
                               └    ┘
   想象沿主对角线翻折：第 i 行 → 第 i 列
```

**方阵 square matrix**——行数 = 列数（$n{\times}n$）。

**单位阵 identity matrix $I$**——主对角线全 1、其余全 0 的方阵。**矩阵世界的"数字 1"**：和谁乘都不变。

```
       ┌1 0 0┐                    数字 1 × 任何 x = x
   I = │0 1 0│   ←  类比  →       矩阵 I × 任何 A = A
       └0 0 1┘

   AI = IA = A   （前提：形状对得上）
```

**对称方阵 symmetric**——沿主对角线对称，$a_{ij} = a_{ji}$（即 $A^T = A$）。LR 正规方程里 $X^T X$ 永远是对称方阵——这是后面的关键性质。

**实算**——Slide 44 给定 $A = \begin{bmatrix}1&2&3\\4&5&6\end{bmatrix}$，算 $A A^T$ 和 $A^T A$：

$$A A^T = \begin{bmatrix}1&2&3\\4&5&6\end{bmatrix}\begin{bmatrix}1&4\\2&5\\3&6\end{bmatrix} = \begin{bmatrix}14 & 32\\32 & 77\end{bmatrix}_{2\times 2}\quad \text{（2×2 对称方阵）}$$

$$A^T A = \begin{bmatrix}1&4\\2&5\\3&6\end{bmatrix}\begin{bmatrix}1&2&3\\4&5&6\end{bmatrix} = \begin{bmatrix}17&22&27\\22&29&36\\27&36&45\end{bmatrix}_{3\times 3}\quad \text{（3×3 对称方阵）}$$

**关键观察**：

- $AA^T$ 和 $A^T A$ **形状不同**（2×2 vs 3×3），值更不可能相等（Slide 52 题 1 选项 C 是错的）
- 但**两者都是方阵、且都对称**（Slide 52 题 1 选项 B 正确）——这是因为 $(A A^T)^T = (A^T)^T A^T = A A^T$（转置反序律得到的）

### 【数学】乘法不满足交换律 / 满足结合律（Slide 45）

**不满足交换律 $AB \neq BA$**——一般情况下两边形状都不一样：

```
   A: m×n       B: n×m
   ┌─┐         ┌──┐
   │ │  ←  ✕  →│  │
   └─┘         └──┘

   AB ∈ ℝ^(m×m)        BA ∈ ℝ^(n×n)
   ┌──┐               ┌────┐
   │  │   形状都不同   │    │
   └──┘               └────┘
```

例：$A: 2{\times}3$, $B: 3{\times}2$ → $AB$ 是 2×2，$BA$ 是 3×3。**形状都不一样**，更别说值。

仅当 A、B 同阶方阵且某些极特殊条件下才有 $AB = BA$（比如 $A = I$、$B = $ 任何方阵）。**写代码时顺序不能换。**

**满足结合律 $A(BC) = (AB)C$**——只要形状对得上，括号怎么打都行。这条在正规方程推导里用得上：$X^T X w = X^T y$ 两边操作时随便选先算哪个。

### 【数学】单位阵 + 逆矩阵（Slide 46）

**$AI = IA = A$**——上面说过，单位阵是矩阵世界的 1。

**逆矩阵 inverse matrix $A^{-1}$**——若存在矩阵 B 满足 $AB = BA = I$，则记 $B = A^{-1}$，A 可逆 invertible。

```
   数字世界                    矩阵世界
   x · (1/x) = 1                A · A⁻¹ = I
        ↑                            ↑
     x ≠ 0 时存在               方阵且满秩 full-rank 时存在

   反例：x = 0 没有 1/0     反例：非方阵 / det A = 0 没有 A⁻¹
```

**重要前提**——逆矩阵**不一定存在**（Slide 52 题 1 选项 D 错就错在这）：

| 不可逆情形 | 为什么 |
|---|---|
| 非方阵 | 形状对不上：$AB = I$ 要求 A、B 都是方阵 |
| $\det A = 0$（奇异方阵）| 行列存在线性相关，没法构造出 I |

**二阶逆公式**：

$$\begin{bmatrix}a&b\\c&d\end{bmatrix}^{-1} = \frac{1}{ad - bc}\begin{bmatrix}d&-b\\-c&a\end{bmatrix}$$

记忆：**主对角线交换、副对角线取反、整体除以行列式 $\det = ad-bc$**。$\det = 0$ ⇒ 分母为 0 ⇒ 不可逆。

**实算**——Slide 52 题 3：求 $A = \begin{bmatrix}1&-1\\2&-1\end{bmatrix}$ 的逆。

```
   Step 1 算行列式
   det A = a·d − b·c = 1·(−1) − (−1)·2 = −1 + 2 = 1     ✓ ≠ 0，可逆

   Step 2 套公式（主对角线交换 + 副对角线取反）
   主对角线 [1, −1] → 交换为 [−1, 1]
   副对角线 [−1, 2] → 取反为 [1, −2]
                        ┌−1   1┐
                  → A⁻¹ │      │ × (1/1)
                        └−2   1┘

   Step 3 验证 A A⁻¹ = I
   ┌1  −1┐ ┌−1   1┐   ┌1·(−1)+(−1)·(−2)   1·1+(−1)·1┐   ┌1  0┐
   │     │ │      │ = │                              │ = │    │ = I  ✓
   └2  −1┘ └−2   1┘   └2·(−1)+(−1)·(−2)   2·1+(−1)·1┘   └0  1┘
```

最终：$A^{-1} = \begin{bmatrix}-1 & 1\\-2 & 1\end{bmatrix}$。

### 【数学】转置的性质（Slide 47）

四条性质，前三条直观、第四条是关键：

| 性质 | 说明 |
|---|---|
| $(A^T)^T = A$ | 转置两次还原（行→列→行） |
| $(A + B)^T = A^T + B^T$ | 转置对加法分配 |
| $(kA)^T = k A^T$ | 转置不影响标量倍数 |
| $(AB)^T = B^T A^T$ | **乘法转置：先反序再各自转置** |

**$(AB)^T = B^T A^T$ 是正规方程推导的关键**——新手常错写成 $(AB)^T = A^T B^T$。

**为什么要反序**？形状角度直觉一下：$A: m{\times}k$, $B: k{\times}n$ → $AB: m{\times}n$ → $(AB)^T: n{\times}m$。要让结果是 $n{\times}m$，只能 $B^T : n{\times}k$ 在前、$A^T: k{\times}m$ 在后——形状对接逼着你必须反序。

**例**（Slide 47）：

$$\left(\begin{bmatrix}1&2&3\\4&5&6\end{bmatrix} \cdot B\right)^T = B^T \cdot \begin{bmatrix}1&4\\2&5\\3&6\end{bmatrix}$$

→ 后续 04a-analytical 推导 $\frac{\partial}{\partial w}\lVert Xw - y \rVert^2 = 0$ 要用到 $(Xw)^T = w^T X^T$、$(X^T X)^T = X^T X$（即 $X^T X$ 对称）这两步——都靠这条性质。

### LR 主线案例 · 把 5 样本写成矩阵 X

01-损失函数已建立 5 样本身高→体重数据。现在把它写成 LR 训练用的矩阵形式——加一列全 1 把偏置 b 吸收进权重向量（这一招叫"增广"）：

```
   原始数据                   矩阵形式（增广 X + 权重 + 标签）
   ┌─────┬─────┐              X (5×2)        w (2×1)        y (5×1)
   │身高 │体重 │              ┌160   1┐                     ┌56.3 ┐
   │160  │56.3 │              │166   1│      ┌w┐            │60.6 │
   │166  │60.6 │              │172   1│ ×    │ │      =     │65.1 │
   │172  │65.1 │              │174   1│      └b┘            │68.5 │
   │174  │68.5 │              │180   1│                     │75.0 │
   │180  │75.0 │              └       ┘                     └     ┘
   └─────┴─────┘              特征列 偏置列
                              (身高) (全 1)
```

**模型一行写**：$\hat{\mathbf{y}} = X \mathbf{w}$

形状对接：(5×2) @ (2×1) → (5×1) ✓——5 个预测体重。展开看就是每行：$\hat y_i = 160 \cdot w + 1 \cdot b = wx_i + b$，和单样本写法等价。

**正规方程的形状链路**——这是矩阵章节的最大锚点：

```
   公式：w = (Xᵀ X)⁻¹ Xᵀ y

   形状追踪（X: 5×2, y: 5×1）：

   Xᵀ          : (2×5)         ← X 转置，行列对调
   Xᵀ X        : (2×5) @ (5×2) → (2×2)    ← 方阵！而且对称（关键！）
   (Xᵀ X)⁻¹    : (2×2)         ← 因为是方阵，可以求逆
   Xᵀ y        : (2×5) @ (5×1) → (2×1)    ← 一个 2 维列向量
   (Xᵀ X)⁻¹ Xᵀ y : (2×2) @ (2×1) → (2×1)  ← 最终就是 [w, b]ᵀ
                                       ✓ 形状和我们要找的参数对得上
```

**链路全程靠 4 个零件**（本节学的所有东西都在这里）：

1. **转置** $X^T$ ← Slide 44
2. **乘法** 两次 $X^T X$ 和 $X^T y$ ← Slide 43（形状对接 + $C_{ij}$ 求和）
3. **求逆** $(X^T X)^{-1}$ ← Slide 46（前提：$X^T X$ 是方阵且 $\det \neq 0$）
4. **乘法** 最后一次 ← Slide 43

正规方程之所以能存在、能跑通、能解出 LR 最优 $(w, b)$，**全靠这一节这 4 个零件能丝滑组装**。本节先不算具体数值（04a-analytical 主场算），只把形状链路钉死——读者一眼能看出"为什么 LR 解析解全靠矩阵"。

→ 具体数值 + 闭式解推导见 [`04a-analytical/`](../04a-analytical/) 主场。

### 常见混淆 / 坑

- **乘法不满足交换律**：$AB \neq BA$（一般情况）。形状对得上 $AB$ 能算，反过来 $BA$ 可能形状都不对。**写顺序不能乱换**。
- **矩阵不一定可逆**——$X^T X$ 不可逆的情形：(1) 特征数 > 样本数（$\det = 0$）；(2) 特征间存在线性相关（共线性 collinearity）。这时正规方程失效，得用伪逆 `np.linalg.pinv` 或 Ridge 回归（塞 $\lambda I$ 强行可逆）。详见 [`04a-analytical/`](../04a-analytical/)。
- **转置反序律**：$(AB)^T = B^T A^T$（不是 $A^T B^T$）。新手常写错，正规方程推导专坑。
- **`@` vs `*` 在 numpy**——`A @ B` 是矩阵乘，`A * B` 是逐元素乘（要求形状相同或可 broadcast）。和 05-向量章节的坑一样，**符号不同结果天差地别**。
