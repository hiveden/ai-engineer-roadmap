#### Slide 19 · 引入评判标准 – loss function 损失函数

- **误差**：每个数据点预测值 ŷ − 真实值 y（用来计算损失函数）
- **损失函数**：综合衡量"模型预测结果"和"真实值"之间的整体差异
- 也叫代价函数、成本函数、目标函数

```

  ↑                    /│                /│

  |              ───/                  /
  |       ●真实值/                    /
  |        │   /───                  /
  |   误差1│  /
  |        ●预测值
  +────────┴───────┴───────┴─────────→ 身高 X
                  172      174
```

#### Slide 20 · 损失函数数学表达

损失函数综合衡量"预测结果"与"真实值"的整体差异，即评估模型性能。

- **平均绝对误差 MAE**（Mean Absolute Error）：$\text{loss}(w,b) = \dfrac{1}{m}\sum_{i=1}^{m}|\hat y_i - y_i|$
  - 第 i 个误差的绝对值
- **均方误差 MSE**（Mean-Square Error）：$\text{loss}(w,b) = \dfrac{1}{m}\sum_{i=1}^{m}(\hat y_i - y_i)^2$
  - **对误差的平方惩罚！！** 不需要绝对值
  - **不建议用在有异常值的数据**

> **备注**：对误差的平方惩罚——怕模型优化时为了降低 loss 去拟合异常值。

#### Slide 21 · 损失函数（结论）

- 结论：损失函数越小，"模型预测结果"越接近"真实值"，模型性能越好
- 红色直线在每个数据点上有更小的误差、更小的 loss 值，因此它是更好的线性模型，能更好地描述数据分布规律
- **最小二乘**：平方和误差

```
                红线（loss 小，更优）            蓝线（loss 大，差）
  ●───●─●───●         vs                 ●  ●  ●
        红线贴近所有点                     ────────  蓝线偏离
```

#### Slide 22 · 求解最优模型的套路

- 最优模型 ⇔ 最小的损失函数
- 通过损失函数最小化，求解 w、b → 获得最优模型参数
- 最优模型：**y = wx + b**

```
最小化 loss  ──►  解出 (w*, b*)  ──►  最优模型 ŷ = w*x + b*
```

#### Slide 23 · 损失函数计算（简化）

样本数据（5 个：身高 → 体重）：

| 序号 | 身高 x | 真实体重 y |
|---|---|---|
| 1 | 160 | 56.3 |
| 2 | 166 | 60.6 |
| 3 | 172 | 65.1 |
| 4 | 174 | 58.5（PPT 原值，备注与 slide 56 用 **68.5**） |
| 5 | 180 | 56.3（PPT 原值，备注与 slide 56 用 **75**） |

> 注：PPT slide 23/24 正文 5 个 y 值（56.3, 60.6, 65.1, 58.5, 56.3）属录入错误。slide 24 备注、slide 55 平方和、**slide 56 推导用的 56.3, 60.6, 65.1, 68.5, 75 才是与最终系数 281671.6 / 145416 / 852 / 55683.8 / 325.5 自洽的"官方答案"**。

假设 y = kx + b，损失函数：

$$L(k,b) = (160k+b-56.3)^2 + (166k+b-60.6)^2 + (172k+b-65.1)^2 + (174k+b-58.5)^2 + (180k+b-56.3)^2$$

损失函数是关于 k、b 的二元二次方程，展开复杂。**为简化计算，先固定截距 b = -100**：

$$L(k, b{=}{-100}) = (160k-156.3)^2 + (166k-160.6)^2 + (172k-165.1)^2 + (174k-158.5)^2 + (180k-156.3)^2$$

#### Slide 24 · 求损失函数最小值（求最优解）

逐项展开：每个 $(ax - c)^2 = a^2 k^2 - 2ac\,k + c^2$，再求和合并 k² 系数 / k 系数 / 常数项。

```
k² 系数：160² + 166² + 172² + 174² + 180²
       = 25600 + 27556 + 29584 + 30276 + 32400
       = 145416

k 一次项系数（取自 PPT 备注，y 用 156, 160.6, 165.1, 168.5, 175）：
  2·160·156 + 2·166·160.6 + 2·172·165.1 + 2·174·168.5 + 2·180·175
  = 49920 + 53319.2 + 56794.4 + 58638 + 63000
  = 281671.6   →  作为 −281671.6 k

常数项：156² + 160.6² + 165.1² + 168.5² + 175²
       = 136496.32

⇒  L(k, b=-100) = 145416 k² − 281671.6 k + 136496.32
```

二次函数顶点：k* = −b/(2a)，导数为 0 时取得最小值：

$$k^* = \frac{281671.6}{2 \times 145416} = \mathbf{0.9685}$$

截距 b = −100 时，预测身高 176 的小明体重：

$$\hat y = 0.9685 \times 176 - 100 = \mathbf{70.456}$$

> 总结：求最好拟合线 y = kx + b ⇒ 引入损失函数 Loss(k, b) ⇒ 优化方法求最小值得 k 最优解

#### Slide 31 · 基础数学 – 数据表述

- **标量** scalar：一个独立存在的数，只有大小没有方向
- **向量** vector：一列顺序排列的元素，**默认列向量**，有大小和方向（模和夹角）

- **矩阵** matrix：二维数组（m 行 n 列）
  - 例：张三、李四的数理化成绩 [[70,80,90],[75,85,95]]
  - m = 行数（样本数），n = 列数（特征数）
- **张量** Tensor：基于向量和矩阵的推广，更高阶数组
  - 例：2 个 3×4 矩阵 / 3 个 2×4 / 4 个 2×3 等

#### Slide 32 · 导数 – 基本概念

函数 y = f(x) 的自变量 x 在某点 x₀ 上产生增量 Δx 时，函数输出值的增量 Δy 与 Δx 的比值在 Δx → 0 时的极限即为 x₀ 处的导数：

$$f'(x_0) = \lim_{\Delta x \to 0} \frac{\Delta y}{\Delta x} = \lim_{\Delta x \to 0} \frac{f(x_0+\Delta x) - f(x_0)}{\Delta x}$$

#### Slide 33 · 导数的几何意义

函数在某一点的导数 = 该函数所代表的曲线在该点的**切线斜率**。

```
y                       ●  切线（斜率 = f'(x₀)）
  ↑                   ╱
  |        曲线y=f(x) ╱
  |                ╲ ╱
  |                ●(x₀, f(x₀))
  |               ╱
  |              ╱
  +────────────┴──────────→ x
              x₀
```

#### Slide 34 · 常见函数的导数

- **幂函数**：$(x^\alpha)' = \alpha x^{\alpha-1}$
- **指数函数**：$(a^x)' = a^x \ln a$，$(e^x)' = e^x$
- **对数函数**：$(\log_a x)' = \dfrac{1}{x \ln a}$，$(\ln x)' = \dfrac{1}{x}$
- **三角函数**：$(\sin x)' = \cos x$，$(\cos x)' = -\sin x$

#### Slide 35 · 导数的四则运算 + 复合函数

- $(u \pm v)' = u' \pm v'$
- $(uv)' = u'v + uv'$
- $(u/v)' = (u'v - uv')/v^2$
- **复合函数链式法则**：$\{g[h(x)]\}' = g'(h) \cdot h'(x)$ — **先对外函数求导，再对内函数求导**

例：计算 $y = (x^2 + 2x)^2$ 的导函数

$$y' = 2(x^2+2x)(2x+2) = 4(x^3 + 3x^2 + 2x) = 4x^3 + 12x^2 + 8x$$

#### Slide 38 · 导数求极值

**导数为 0 的位置是函数的极值点。**

例：求函数 $y = x^2 - 4x + 5$ 的极小值

$$y' = 2x - 4 = 0 \;\Rightarrow\; x = 2 \;\Rightarrow\; y_{\min} = 2^2 - 4\cdot 2 + 5 = 1$$

```
y
 ↑
 |   ╲                           ╱
 |    ╲                         ╱
 |     ╲                       ╱
 |      ╲_____               _╱
 |            ╲___________╱
 |                ●(2, 1)  ← 极小值点（导数=0）
 +───────────────┴──────────────→ x
                 x=2
```

#### Slide 39 · 偏导（引入）

z = z(x, y)，求 $z = (x-2)^2 + (y-3)^2$ 的极小值。

- 对 x 偏导：$\dfrac{\partial z}{\partial x} = 2(x-2) \cdot 1 = 0 \;\Rightarrow\; x=2$ 时 x 方向取极小
- 对 y 偏导：$\dfrac{\partial z}{\partial y} = 2(y-3) \cdot 1 = 0 \;\Rightarrow\; y=3$ 时 y 方向取极小

#### Slide 40 · 多元函数偏导（练习）

**1）** $z = x^2 + 2xy - 3y^2$

$$\frac{\partial z}{\partial x} = 2x + 2y, \quad \frac{\partial z}{\partial y} = 2x - 6y$$

**2）** $z = x^y$

$$\frac{\partial z}{\partial x} = y\, x^{y-1}, \quad \frac{\partial z}{\partial y} = x^y \ln x$$

**3）** $z = e^{2x^2 - 3y}$（复合函数 + 多元偏导）

$$\frac{\partial z}{\partial x} = e^{2x^2-3y} \cdot (2x^2-3y)'_x = 4x\, e^{2x^2-3y}$$

$$\frac{\partial z}{\partial y} = e^{2x^2-3y} \cdot (2x^2-3y)'_y = -3\, e^{2x^2-3y}$$

y

例：$X = \begin{bmatrix}1\\2\\-3\end{bmatrix}$，则 $X^T = (1,\,2,\,-3)$

#### Slide 42 · 向量和矩阵 – 范数 Norm

范数（norm）是数学中的基本概念，**具有长度的意义**。

设 $x^T = (1, 2, -3)$：

- **L1 范数**（各元素绝对值之和）：$\lVert x \rVert_1 = |1| + |2| + |-3| = 6$
- **L2 范数**（向量模长，平方和开根）：$\lVert x \rVert_2 = \sqrt{1^2 + 2^2 + (-3)^2} = \sqrt{14}$
- **Lp 范数**：$\lVert x \rVert_p = \left(\sum |x_i|^p\right)^{1/p}$

注意：$x^T x = 1^2 + 2^2 + (-3)^2 = 14$（向量的转置 @ 向量，**不能反过来**），与 $\lVert x \rVert_2^2$ 一致。

#### Slide 43 · 矩阵 1 – 加减与乘法

矩阵：m 行 n 列的数据，记 $A \in \mathbb{R}^{m\times n}$。机器学习数据集 $X \in \mathbb{R}^{N\times D}$（N 行样本、D 个特征）。

- **加减**：同形对应位置相加减
- **乘法**（**非常重要！**）：A: m×k @ B: k×n → C: m×n，对应行列元素相乘求和

$$C_{ij} = \sum_{p=1}^{k} A_{ip} B_{pj}$$

#### Slide 44 · 矩阵 2 – 转置 / 方阵 / 单位阵

设 $A = \begin{bmatrix}1&2&3\\4&5&6\end{bmatrix} \in \mathbb{R}^{2\times 3}$，则 $A^T = \begin{bmatrix}1&4\\2&5\\3&6\end{bmatrix} \in \mathbb{R}^{3\times 2}$。

**A @ Aᵀ 是方阵**（2×2）：

$$A A^T = \begin{bmatrix}1\cdot1+2\cdot2+3\cdot3 & 1\cdot4+2\cdot5+3\cdot6\\ 4\cdot1+5\cdot2+6\cdot3 & 4\cdot4+5\cdot5+6\cdot6\end{bmatrix} = \begin{bmatrix}14&32\\32&77\end{bmatrix}$$

**Aᵀ @ A 也是方阵**（3×3）：

$$A^T A = \begin{bmatrix}17&22&27\\22&29&36\\27&36&45\end{bmatrix}$$

- **方阵**：行数 = 列数
- **单位阵 I（或 E）**：主对角线为 1、其他为 0，例 $I_{3\times 3} = \begin{bmatrix}1&0&0\\0&1&0\\0&0&1\end{bmatrix}$
- **对称方阵**：沿主对角线对称，$a_{ij} = a_{ji}$
- **A @ Aᵀ 通常 ≠ Aᵀ @ A**

#### Slide 45 · 矩阵 3 – 乘法性质

- **不满足交换律**：A × B ≠ B × A
  - 例：A: m×n, B: n×m → A@B ∈ ℝ^(m×m), B@A ∈ ℝ^(n×n)，形状都不一样
  - 仅当 A、B 同阶方阵且极个别条件下才有 AB = BA
- **满足结合律**：A × (B × C) = (A × B) × C

#### Slide 46 · 矩阵的逆 + 单位阵

- **A @ I = I @ A = A**（与单位阵相乘等于自己）
- **逆矩阵**：若 A @ B = I，则 B 为 A 的逆矩阵，记 $A^{-1}$
- **不一定存在逆矩阵！**（仅方阵且满秩可逆）

例：$A = \begin{bmatrix}1&2\\3&4\end{bmatrix} \in \mathbb{R}^{2\times 2}$，可求 $A^{-1}$。

#### Slide 47 · 矩阵 4 – 转置的性质

- $(A^T)^T = A$
- $(A + B)^T = A^T + B^T$
- $(kA)^T = k A^T$
- $(AB)^T = B^T A^T$

例：$\left(\begin{bmatrix}1&2&3\\4&5&6\end{bmatrix} \cdot B\right)^T = B^T \cdot \begin{bmatrix}1&4\\2&5\\3&6\end{bmatrix}$

#### Slide 48 · 数学复习小结（一）

1. 标量、向量、矩阵、张量概念
2. **导数**：函数上某点切线斜率，瞬时变化率
3. **导数求解方法**：常数 / 指数 / 幂 / 三角都有导函数（**会查表应用**）；复合函数先外后内
4. **利用导数求极值**：导数为 0 即极值点
5. **偏导数**：u(x, y, z) 只在某分量上求导即偏导。各分量偏导组成向量 $(\partial u/\partial x, \partial u/\partial y, \partial u/\partial z)$，即**梯度**

#### Slide 49 · 数学复习小结（二）

1. **向量基本运算**：L1 / L2 范数；$x^T x$ 与 $\lVert x \rVert_2^2$ 一样
2. **矩阵运算**：
   - 矩阵表示：$X \in \mathbb{R}^{N\times D}$
   - 加减：形状相同对应元素加减
   - 乘法：A: m×n @ B: n×d → C: m×d
   - 不满足交换律 A×B ≠ B×A，满足结合律
   - A @ I = A
   - A @ B = I → B = A⁻¹
   - 转置性质：(A+B)ᵀ = Aᵀ + Bᵀ，(AB)ᵀ = BᵀAᵀ

> **题 1**：有关矩阵相乘说法正确的是？
>
> A. 两个矩阵能相乘的条件为：A: m×n, B: d×m，A@B = C
> **B. A @ Aᵀ = C，Aᵀ @ A = D，则 C 和 D 都是方阵**
> C. A @ Aᵀ = C，Aᵀ @ A = D，则 C 和 D 都是方阵，而且 C = D
> D. 任意 A，A 的逆矩阵一定存在
>
> **答案：B**（A 错——条件应是中间维度一致 n=d；C 错——形状不同；D 错——非方阵或奇异方阵无逆）

> **题 2**：已知 $A = \begin{bmatrix}1&2&-1\\4&5&-1\end{bmatrix}$，$B = \begin{bmatrix}1&-1\\1&-1\\1&-1\end{bmatrix}$，求 A @ B。
>
> $$A B = \begin{bmatrix}1+2-1 & -1-2+1\\4+5-1 & -4-5+1\end{bmatrix} = \begin{bmatrix}2&-2\\8&-8\end{bmatrix}$$

> **题 3**：已知 $A = \begin{bmatrix}1&-1\\2&-1\end{bmatrix}$，求 $A^{-1}$。
>
> 二阶逆公式：$\begin{bmatrix}a&b\\c&d\end{bmatrix}^{-1} = \dfrac{1}{ad-bc}\begin{bmatrix}d&-b\\-c&a\end{bmatrix}$
>
> 行列式 $\det A = 1\cdot(-1) - (-1)\cdot 2 = 1$，所以
>
> $$A^{-1} = \begin{bmatrix}-1 & 1\\-2 & 1\end{bmatrix}$$
>
> 验证：$A A^{-1} = \begin{bmatrix}1&-1\\2&-1\end{bmatrix}\begin{bmatrix}-1&1\\-2&1\end{bmatrix} = \begin{bmatrix}1&0\\0&1\end{bmatrix} = I$ ✓

> 03 · 损失函数 + 复习导数和矩阵

> 【掌握】损失函数

需要设置一个评判标准。

**误差概念**：用预测值 y – 真实值 y 就是误差。

**损失函数**：衡量每个样本预测值与真实值效果的函数。

"红色直线能更好的拟合所有点" 也就是误差最小，误差和最小。

损失函数数学如何表达呢？又如何求损失函数的最小值呢？

当损失函数取最小值时，得到 k 就是最优解。

想求一条直线更好的拟合所有点 y = kx + b：

- 引入损失函数（衡量预测值和真实值效果）Loss(k, b)
- 通过一个优化方法，求损失函数最小值，得到 K 最优解

回归的损失函数：

- **均方误差**（Mean-Square Error, MSE）
- **平均绝对误差**（Mean Absolute Error, MAE）

> 【知道】常见的数据表述

- 为什么要学习标量、向量、矩阵、张量？
  - 因机器学习、深度学习中经常用，不要因是数学就害怕
  - 宗旨：用到就学什么，不要盲目的展开、大篇幅学数学
- **标量** scalar：一个独立存在的数，只有大小没有方向
- **向量** vector：向量指一列顺序排列的元素。默认是列向量
- **矩阵** matrix：二维数组
- **张量** Tensor：多维数组，张量是基于向量和矩阵的推广

> 【知道】导数

当函数 y = f(x) 的自变量 x 在一点 $x_0$ 上产生一个增量 Δx 时，函数输出值的增量 Δy 与自变量增量 Δx 的比值在 Δx 趋于 0 时的极限 a 如果存在，a 即为在 $x_0$ 处的导数，记作 $f^\prime(x_0)$ 或 df($x_0$)/dx。

导数是函数的局部性质。一个函数在某一点的导数描述了这个函数在这一点附近的变化率。

函数在某一点的导数就是该函数所代表的曲线在这一点上的切线斜率。

**LR 用得到的两条**：
- 幂函数：$(x^\alpha)^\prime = \alpha x^{\alpha-1}$（求 MSE 关于 w 的导数靠它）
- 链式法则：$\{g[h(x)]\}^\prime = g^\prime(h) \cdot h^\prime(x)$（嵌套函数求导）

**导数求极值**：导数为 0 的位置是函数的极值点（LR 闭式解的核心思路）。

> 完整导数公式表 / 四则运算法则不展开——遇到具体函数查表即可。

> 【知道】偏导

**多变量函数对其中一个变量求导**，把其它变量当常数。

设 $f(x, y) = x^2 + 3xy + y^2$：

- 对 x 偏导：$\frac{\partial f}{\partial x} = 2x + 3y$（y 当常数 → $3xy \to 3y$，$y^2 \to 0$）
- 对 y 偏导：$\frac{\partial f}{\partial y} = 3x + 2y$

**线性回归用偏导做什么**：MSE 损失 $L(w, b) = \sum (y - wx - b)^2$ 是 w 和 b 的二元函数，
分别对 w、b 求偏导，令偏导 = 0，就得到正规方程的解析解（见 04a）。

**梯度**：所有偏导组成的向量，记作 $\nabla L = \left[\frac{\partial L}{\partial w_1}, \frac{\partial L}{\partial w_2}, \ldots\right]^T$。
梯度方向 = 函数上升最快的方向，**反方向**就是梯度下降的方向（见 04b）。

> 【知道】向量

**向量加法 / 减法**（同维度逐元素）：

$$\mathbf{a} + \mathbf{b} = [a_1 + b_1, a_2 + b_2, \ldots, a_n + b_n]^T$$

**点积 / 内积**（结果是标量）：

$$\mathbf{a} \cdot \mathbf{b} = \mathbf{a}^T \mathbf{b} = \sum_{i=1}^{n} a_i b_i$$

**向量 L2 范数**（向量长度）：

$$\lVert \mathbf{a} \rVert_2 = \sqrt{\sum a_i^2} = \sqrt{\mathbf{a}^T \mathbf{a}}$$

**线性回归用向量做什么**：把 n 个特征 $x_1, x_2, \ldots, x_n$ 写成列向量 $\mathbf{x}$，
权重 $\mathbf{w}$ 同维列向量，预测值就是 $\hat{y} = \mathbf{w}^T \mathbf{x} + b$，一行公式覆盖所有维度。

> 【知道】矩阵

**矩阵乘法**（A: m×k, B: k×n → C: m×n）：

$$C_{ij} = \sum_{p=1}^{k} A_{ip} B_{pj}$$

**转置**：$(A^T)_{ij} = A_{ji}$，行变列、列变行。

**逆矩阵**：仅方阵且满秩可逆，满足 $A A^{-1} = I$（单位阵）。

**线性回归用矩阵做什么**：把 m 个样本叠成矩阵 X (m×n)，标签叠成向量 y (m×1)，
则模型变成 $\hat{\mathbf{y}} = X \mathbf{w}$。MSE 损失展开后求导得正规方程：

$$\mathbf{w} = (X^T X)^{-1} X^T \mathbf{y}$$

这个公式是 04a 的核心——**所有训练计算用一个矩阵公式搞定**，是 LR 解析解的精华。
