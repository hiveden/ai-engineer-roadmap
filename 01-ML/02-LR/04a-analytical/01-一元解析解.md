# 一元解析解

> 级别：【了解】
> 维度：数学
> 章节底稿全文见 [`README.md`](./README.md)（PPT slide 55-56）

## 底稿

> 损失函数用来衡量真实值和预测值之间的差异，**为优化参数指明了方向**。一元线性回归 $y = kx + b$ 要求 (k, b) 使损失最小。

---

## 直觉

一句话：**MSE 损失对 (k, b) 是一个开口向上的抛物面，山谷最低点的位置可以直接用公式算出来，不需要"走下山"**。

这就是「解析解」(closed-form solution) 的含义：列方程 → 一步代入公式 → 得最优解。也叫**最小二乘法 OLS**（Ordinary Least Squares）。

## 推导思路

目标：找一组 (k, b) 使损失 $L(k, b)$ 最小：

$$L(k, b) = \sum_{i=1}^{n}(y_i - k x_i - b)^2$$

**关键观察**：$L$ 是关于 (k, b) 的**二元二次方程**（每个 $(y_i - kx_i - b)^2$ 展开后 k、b 都是二次幂），所以 L 是凸函数，**唯一的极值就是全局最小值**。

凸函数极值点 = 梯度为零的点。分别对 k、b 求偏导（偏导基础见 [`03b-math/`](../03b-math/)）：

$$\frac{\partial L}{\partial b} = -2 \sum (y_i - k x_i - b) = 0$$

$$\frac{\partial L}{\partial k} = -2 \sum x_i (y_i - k x_i - b) = 0$$

两条偏导式联立 → 二元一次方程组 → 解出 (k, b)。

## 最终公式

设 $\bar{x} = \frac{1}{n}\sum x_i$、$\bar{y} = \frac{1}{n}\sum y_i$（样本均值），求解后得：

$$\boxed{k = \frac{\sum_{i=1}^{n} (x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^{n} (x_i - \bar{x})^2}}$$

$$\boxed{b = \bar{y} - k \bar{x}}$$

**记忆口诀**：k 是「协方差 / x 的方差」，b 让回归线必过样本均值点 $(\bar{x}, \bar{y})$。

## 算例 · 身高预测体重

5 个样本：

| $x$（身高 cm）| $y$（体重 kg）|
|---|---|
| 160 | 56.3 |
| 166 | 60.6 |
| 172 | 65.1 |
| 174 | 68.5 |
| 180 | 75.0 |

**Step 1** 算均值：$\bar{x} = (160+166+172+174+180)/5 = 170.4$，$\bar{y} = (56.3+60.6+65.1+68.5+75.0)/5 = 65.1$

**Step 2** 算分子（协方差和）：

| $x_i - \bar{x}$ | $y_i - \bar{y}$ | 乘积 |
|---|---|---|
| -10.4 | -8.8 | 91.52 |
| -4.4 | -4.5 | 19.80 |
| 1.6 | 0.0 | 0.00 |
| 3.6 | 3.4 | 12.24 |
| 9.6 | 9.9 | 95.04 |

合计：$\sum = 218.6$

**Step 3** 算分母（x 方差和）：$(-10.4)^2 + (-4.4)^2 + 1.6^2 + 3.6^2 + 9.6^2 = 108.16 + 19.36 + 2.56 + 12.96 + 92.16 = 235.2$

**Step 4** 代入：$k = 218.6 / 235.2 \approx 0.93$，$b = 65.1 - 0.93 \times 170.4 \approx -93.4$

**预测小明** $(x = 176)$：$\hat{y} = 0.93 \times 176 - 93.4 \approx 70.3$ kg。

> PPT slide 56 给出的另一组数 (k=0.0397, b=60.76) 是用「未中心化」的原始方程组直接解的，思路一致只是数值规模不同——**两者都是偏导=0 联立求解，不是不同方法**。

## 与梯度下降的对比

| 维度 | 解析解（OLS）| 梯度下降（[`04b-gd/`](../04b-gd/)）|
|---|---|---|
| 求解方式 | 一步代公式 | 迭代逼近 |
| 学习率 | 不需要 | 必须调 |
| 适用范围 | LR 等少数有闭式解的模型 | **几乎所有可导损失** |
| 大数据 | $X$ 装不下内存就崩 | mini-batch 友好 |
| 多特征 | $O(n^3)$ 矩阵求逆，n>10⁴ 慢 | $O(n)$ 每步 |
| 教学价值 | 看懂「最优解长啥样」 | 看懂「模型怎么学」 |

**结论**：一元 / 低维 LR 用解析解最爽；深度学习时代解析解几乎用不上，但**思路（梯度=0 找极值）是所有优化方法的根**。多元情况下解析解的矩阵化版本就是 [`02-多元正规方程`](./02-多元正规方程.md)。
