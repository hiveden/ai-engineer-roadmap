#### Slide 55 · 一元线性回归 – 解析解

- 损失函数用来衡量真实值和预测值之间的差异，**为优化参数指明了方向**
- 模型：y = kx + b（求 k、b 使损失最小）

5 样本（身高 → 体重）平方和损失函数：

$$J(k, b) = (160k+b-56.3)^2 + (166k+b-60.6)^2 + (172k+b-65.1)^2 + (174k+b-58.5)^2 + (180k+b-56.3)^2$$

通用 Σ 形式：

$$J(k, b) = \sum_{i=1}^{m}\bigl(h(x^{(i)}) - y^{(i)}\bigr)^2 = \sum_{i=1}^{m}\bigl(kx^{(i)} + b - y^{(i)}\bigr)^2$$

> **备注**：线性模型 h(w) 是关于 w 的函数，权重参数是 1 次幂。
>
> ⚠ slide 55 这里 5 个 y 值是 56.3 / 60.6 / 65.1 / **58.5** / **56.3**（同 03b slide 23 的"原 PPT 值"），与下一页 slide 56 推导用的 56.3 / 60.6 / 65.1 / **68.5** / **75** 不一致——属 PPT 内部录入错误。**slide 56 的版本（68.5 / 75）才是与最终系数 281671.6 / 145416 / 852 / 55683.8 / 325.5 自洽的"官方答案"**。

#### Slide 56 · 一元线性回归 – 解析解（梯度法 求 k、b）

求一元损失函数 $J(k,b) = \sum_{i=1}^{m}(kx^{(i)}+b-y^{(i)})^2$ 的极小值。

**Step 1 · 对 k、b 分别求偏导设置成 0**：

$$\frac{\partial J(k,b)}{\partial k} = \sum_{i=1}^{m} 2\bigl(kx^{(i)}+b-y^{(i)}\bigr)^{2-1} \cdot \bigl(kx^{(i)}+b-y^{(i)}\bigr)'_k = \sum_{i=1}^{m}\bigl(2kx^{(i)2} + 2bx^{(i)} - 2x^{(i)}y^{(i)}\bigr) = 0 \quad\text{——1 式}$$

$$\frac{\partial J(k,b)}{\partial b} = \sum_{i=1}^{m} 2\bigl(kx^{(i)}+b-y^{(i)}\bigr)^{2-1} \cdot \bigl(kx^{(i)}+b-y^{(i)}\bigr)'_b = \sum_{i=1}^{m}\bigl(2kx^{(i)} + 2b - 2y^{(i)}\bigr) = 0 \quad\text{——2 式}$$

**Step 2 · 化简**（两侧除 2，整理）：

$$k\sum_{i=1}^{m}x^{(i)2} + b\sum_{i=1}^{m}x^{(i)} - \sum_{i=1}^{m}x^{(i)}y^{(i)} = 0 \quad\text{——3 式}$$

$$k\sum_{i=1}^{m}x^{(i)} + bm - \sum_{i=1}^{m}y^{(i)} = 0 \quad\text{——4 式}$$

**Step 3 · 代入 5 样本数据**（y = 56.3, 60.6, 65.1, 68.5, 75）：

$$k(160^2{+}166^2{+}172^2{+}174^2{+}180^2) + b(160{+}166{+}172{+}174{+}180) - (160{\cdot}56.3{+}166{\cdot}60.6{+}172{\cdot}65.1{+}174{\cdot}68.5{+}180{\cdot}75) = 0 \quad\text{——5 式}$$

$$k(160{+}166{+}172{+}174{+}180) + 5b - (56.3{+}60.6{+}65.1{+}68.5{+}75) = 0 \quad\text{——6 式}$$

数字方程组：

```
145416 k +  852 b -  55683.8 = 0
   852 k +    5 b -    325.5 = 0
```

**Step 4 · 求解 k、b**（PPT 答案）：

- PPT 给出：**k = 0.0397，b = 60.7615** → 预测 y = 0.0397 × 176 + 60.7615 ≈ **67**

> **备注**：回顾之前 loss 曲线。
>
> ⚠ **PPT 答案错误**——代回验证：145416·0.0397 + 852·60.7615 ≈ 57541.7 ≠ 55683.8。
>
> 正确解（消元法）：从 6 式得 b = 65.1 − 170.4k，代入 5 式得 235.2 k = 218.6
>
> $$k^* = \frac{218.6}{235.2} \approx \mathbf{0.9295}, \quad b^* = 65.1 - 170.4 \cdot 0.9295 \approx \mathbf{-93.29}$$
>
> 预测 ŷ = 0.9295 × 176 − 93.29 ≈ **70.3**（与笔记中"中心化均值公式" $k = \dfrac{\sum(x_i-\bar x)(y_i-\bar y)}{\sum(x_i-\bar x)^2}$ 完全一致）。

#### Slide 57 · 多元线性回归 – 正规方程法（损失函数）

**多元线性回归方程式**：

$$y = w_1 x_1 + w_2 x_2 + w_3 x_3 + \cdots + w_d x_d + b = \mathbf{w}^T \mathbf{x} + b$$

**数据集**：$D = \{(x_1, y_1), (x_2, y_2), \ldots, (x_n, y_n)\}$，每个 $x_i \in \mathbb{R}^d$ 有 d 个特征，共 n 条样本。模型权重向量 $\mathbf{w} = (w_1, w_2, \ldots, w_d)$，d 代表特征数。

**第 1 个样本的预测值**：

$$\hat y_1 = w_1 x_{11} + w_2 x_{12} + \cdots + w_d x_{1d} + b = \sum_{j=1}^{d}\bigl(w_j x_{1j} + b\bigr)$$

**第 1 个样本的平方损失**：

$$\varepsilon_1^2 = (\hat y_1 - y_1)^2 = \Bigl(\sum_{j=1}^{d}(w_j x_{1j} + b) - y_1\Bigr)^2$$

**整体 Loss（平方和损失）** = 把所有 n 个样本平方损失相加：

$$\text{Loss}(\mathbf{w}) = \sum_{i=1}^{n}(\hat y_i - y_i)^2 = \sum_{i=1}^{n}\Bigl(\sum_{j=1}^{d}(w_j x_{ij} + b) - y_i\Bigr)^2$$

**目标**：让多元损失函数取最小值，此时的权重 W 向量就是最优解。

**求最优解的两条路**：
1. 解矩阵方程（即**正规方程**）
2. 通过**梯度下降**求解（[`04b-gd/`](../04b-gd/)）

> 对比一元回归 y = kx + b（单特征），多元就是把"一个 w 一个 x"扩展为"d 个 w · d 个 x"。

#### Slide 58 · 如何求最小值？需要矩阵表达

把上面的 Σ 形式改写成矩阵形式（紧凑、可求导）：

$$L(\mathbf{w}) = (\mathbf{y} - X\mathbf{w})^T (\mathbf{y} - X\mathbf{w})$$

> **备注**：原则上特征 x 和 w 向量表达式都是**多行一列**（列向量）。

#### Slide 59-60 · 矩阵推导（PPT 用图展示，文字仅"相等！！"）

**展开损失函数**：

$$L(\mathbf{w}) = \mathbf{y}^T \mathbf{y} - 2 \mathbf{w}^T X^T \mathbf{y} + \mathbf{w}^T X^T X \mathbf{w}$$

**对 w 求偏导令零**（矩阵求导规则：$\partial(\mathbf{w}^T X^T X \mathbf{w})/\partial \mathbf{w} = 2 X^T X \mathbf{w}$）：

$$\frac{\partial L}{\partial \mathbf{w}} = -2 X^T \mathbf{y} + 2 X^T X \mathbf{w} = 0 \;\Rightarrow\; X^T X \mathbf{w} = X^T \mathbf{y}$$

两侧左乘 $(X^T X)^{-1}$（注意 $(X^T X)^{-1}(X^T X) = I$ 单位矩阵——slide 60 红字 "**相等！！**" 强调的正是这一步）。

#### Slide 61 · 多元线性回归 – 正规方程法

- 损失函数普通方式转成矩阵方式（正规方程）
- 损失函数最小值：**对 w 求偏导**
- 变成单位矩阵：$(X^T X)^{-1} (X^T X) = I$
- **如果逆矩阵不存在？无法求解！！**

#### Slide 62 · 多元线性回归 – 正规方程 w 公式

PPT 给出最终公式：

$$\mathbf{w} = (X^T X)^{-1} X^T \mathbf{y}$$

> 公式解释说明（PPT 原图）：每一项的维度与含义。

#### Slide 63 · 本节小结

**多元线性回归的正规方程**：

最小二乘损失函数 $J(\mathbf{w}) = \lVert X\mathbf{w} - \mathbf{y} \rVert_2^2$ 取最小值，其解为：

$$\boxed{\mathbf{w} = (X^T X)^{-1} X^T \mathbf{y}}$$

> 04 · 一元 / 多元线性回归的解析解

> 【了解】一元线性回归的解析解

**目标**：找一组 (k, b) 使损失 $L(k, b) = \sum_{i=1}^{n}(y_i - k x_i - b)^2$ 最小。

**思路**：偏导 = 0 的位置就是极小值。分别对 k、b 求偏导：

$$\frac{\partial L}{\partial b} = -2 \sum (y_i - k x_i - b) = 0$$

$$\frac{\partial L}{\partial k} = -2 \sum x_i (y_i - k x_i - b) = 0$$

**联立求解**（设 $\bar{x}, \bar{y}$ 为均值）：

$$k = \frac{\sum (x_i - \bar{x})(y_i - \bar{y})}{\sum (x_i - \bar{x})^2}, \qquad b = \bar{y} - k \bar{x}$$

> 也叫**最小二乘法 OLS**（Ordinary Least Squares）的闭式解。**直接代公式即得最优，不需要迭代**。

身高体重例（5 个样本）：
| $x$ (身高) | $y$ (体重) |
|---|---|
| 160 | 56.3 |
| 166 | 60.6 |
| 172 | 65.1 |
| 174 | 68.5 |
| 180 | 75.0 |

代入公式：$\bar{x}=170.4$，$\bar{y}=65.1$，算出 $k \approx 0.93$，$b \approx -93.5$。
预测小明 (x=176)：$\hat{y} = 0.93 \times 176 - 93.5 \approx 70.2$ kg。

> 【了解】多元线性回归的解析解 - 正规方程法

**矩阵化表示**：m 个样本、n 个特征。

- 特征矩阵 $X \in \mathbb{R}^{m \times (n+1)}$（多一列全 1 吸收偏置 b 进 $w_0$）
- 权重向量 $\mathbf{w} \in \mathbb{R}^{n+1}$
- 标签向量 $\mathbf{y} \in \mathbb{R}^{m}$

预测：$\hat{\mathbf{y}} = X \mathbf{w}$
损失：$L(\mathbf{w}) = (\mathbf{y} - X\mathbf{w})^T (\mathbf{y} - X\mathbf{w})$

**对 $\mathbf{w}$ 求导令零**：

$$\frac{\partial L}{\partial \mathbf{w}} = -2 X^T (\mathbf{y} - X\mathbf{w}) = 0$$

整理得 **正规方程**（Normal Equation）：

$$\boxed{\mathbf{w} = (X^T X)^{-1} X^T \mathbf{y}}$$

**优点**：
- 一步出解，**无需调学习率 / 不需要迭代**
- 数学优雅，是 LR 闭式解的"教科书答案"

**缺点**：
- 需要 $X^T X$ **可逆**——特征数过多 / 多重共线性时失败
- $(X^T X)^{-1}$ 计算复杂度 $O(n^3)$，特征数 ~10⁴ 以上就慢得离谱
- 大数据集 $X$ 内存放不下时不可用

→ 工程实践中**特征 < 10⁴ 用正规方程，否则用梯度下降**（见 04b）。
sklearn 的 `LinearRegression` 内部用 SVD 分解避免直接求逆，是正规方程的稳定版。
