### V4.4 收敛三态对学习率三态的呼应

设计目标（章节 README）："学习率太小→龟速 / 太大→振荡 / 极大→飞出 / 恰好→收敛"。

实现 → 收敛徽章三态：
- ✓ 已收敛（绿）= 完美预设最终态
- ○ 未收敛（灰）= 龟速 / 振荡 / 远征 / 鞍点的最终态
- ⚠ 已发散（红）= 飞出预设的最终态

配合 B2 loss-vs-step 折线图四种形状（review S2 的核心教学价值）：
- 完美 = 单调降快收敛
- 龟速 = 单调降但慢
- 振荡 = 锯齿
- 飞出 = 冲天柱（log 轴 cap 后是平顶）

**结论**：徽章 + B2 + C 黄箭头三件套，把 04b 章「学习率三态」教学目标固化为可视化符号，远胜 04a 的纯 loss 形状演示。

## 微观瑕疵（不阻塞验收）

1. **dev-log §4 表格 emoji 编号偏移**：dev-log 把"鞍点"列为 #6，但代码里它是字典第 7 项（index 6）。表格里 #0 = 手动，与代码 dropdown options 顺序一致 ✅，仅文档层面的"6 个 vs 7 个"措辞略有歧义，不影响功能。
2. **`active_lr` 显示精度**：`:198/:200` 直接 `f"...lr={active_lr}"`，未指定小数位。手动模式 lr=0.05 时显示正常，预设 lr=0.005 / 0.27 / 0.4 等也是直传——无 bug，但样式上若用 `{active_lr:.3f}` 会更整齐。**建议项，非必修**。
3. **B1 标题信息冗余**：`:401` 标题写 `"固定 b={cur_b:.2f}"`，发散后 cur_b 可能是 1e6，标题会变成「固定 b=1000000.00」，与红徽章「∞ (发散)」并存。视觉上有点滑稽但教学正确（确实就是发散到那个值）。**不建议改**，反而强化了"发散"的实感。
4. **`fig` 全局变量名复用**：`:300` 用 `fig` 作为 matplotlib figure 名，`:471` 用 `fig3d`——不冲突（不同 cell），命名一致性 OK。
