# 梯度下降思想

> 级别：【掌握】
> 维度：算法
> 章节底稿全文见 [`README.md`](./README.md)（PPT slide 67-69, 74）

## 底稿

**梯度下降法**（gradient descent，GD）：沿损失函数梯度的反方向迭代，逐步逼近极小值点。

**核心公式**：

$$\mathbf{w} \leftarrow \mathbf{w} - \alpha \cdot \nabla L(\mathbf{w})$$

四要素：起点 → 梯度方向 → 步长 α → 收敛判据。

---

## 直觉

一句话：**蒙眼下山——每步原地探坡度最陡的方向，朝下迈一步，重复直到脚下平**。

正规方程（[`04a-analytical/`](../04a-analytical/)）是「一步到位」——直接列方程解出山底坐标。GD 是「步步为营」——大模型上亿参数没人列得出闭式方程，只能一步步挪。深度学习时代默认就是 GD。

## 下山法五步

| 步骤 | 动作 | 对应到模型 |
|---|---|---|
| 1 | 站在山的任意位置 S | 随机初始化权重 $\mathbf{w}_0$ |
| 2 | 环顾四周，找坡度最陡的下坡方向 | 计算当前点的负梯度 $-\nabla L$ |
| 3 | 沿该方向走固定步长 a | 更新 $\mathbf{w} \leftarrow \mathbf{w} - \alpha \nabla L$ |
| 4 | 到新位置 S'，重复 2-3 | 下一轮迭代 |
| 5 | 四周都比 S' 高 → 到底 | loss 不再下降 → 收敛 |

## 梯度到底是什么

| 函数类型 | 梯度的本质 | 几何含义 |
|---|---|---|
| 单变量 $f(\theta)$ | **导数** $f'(\theta)$ | 切线斜率，有正负 |
| 多变量 $L(w_1, w_2, \dots)$ | **偏导数向量** $\left[\frac{\partial L}{\partial w_1}, \frac{\partial L}{\partial w_2}, \dots\right]$ | 指向函数**上升最快**的方向 |

偏导数概念见 [`03b-math/04-偏导.md`](../03b-math/04-偏导.md)。

**关键点**：梯度永远指**上升**最快的方向。我们要找最小值，所以必须**取反**——这就是公式里那个负号的来历。

```
∇L  →  上山最快方向  （loss 越来越大）
-∇L →  下山最快方向  （loss 越来越小）✓
```

## 学习率 α 的作用

α 是**每一步走多远**。它和方向无关，只控制步长。

| α 大小 | 后果 |
|---|---|
| 太小（如 1e-6）| 收敛慢，迭代次数爆炸，浪费算力 |
| 合适（0.001 ~ 0.1）| 平稳收敛，loss 单调下降 |
| 太大（如 1.0）| 一脚跨过山底，来回**震荡** |
| 极大 | **梯度爆炸**，loss 飙到 NaN |

经验值：搭配标准化特征时取 0.01；深度学习里 Adam 默认 1e-3。

## 收敛判据（什么时候停）

任一条件触发就停：

1. **loss 变化小于阈值**：$\lvert L_{t} - L_{t-1} \rvert < \epsilon$（常见 ε = 1e-6）
2. **梯度模长接近 0**：$\lVert \nabla L \rVert < \epsilon$
3. **达到最大迭代数** `max_iter`（兜底，防止永远跑下去）

工程实践：三条都设上，先到先停。

## 单变量数值演示

目标：最小化 $f(\theta) = \theta^2$，对应导数 $f'(\theta) = 2\theta$。
设初值 $\theta_0 = 1$，学习率 $\alpha = 0.4$，最优解显然是 $\theta^* = 0$。

更新公式：$\theta_{t+1} = \theta_t - \alpha \cdot 2\theta_t = (1 - 2\alpha) \theta_t = 0.2 \theta_t$

| 步 t | θ | 计算 | f(θ) |
|---|---|---|---|
| 0 | 1.0000 | 初值 | 1.0000 |
| 1 | 0.2000 | 1 − 0.4·2 = 0.2 | 0.0400 |
| 2 | 0.0400 | 0.2 − 0.4·0.4 = 0.04 | 0.0016 |
| 3 | 0.0080 | 0.04 − 0.4·0.08 = 0.008 | 6.4e-5 |
| 4 | 0.0016 | 0.008 − 0.4·0.016 = 0.0016 | 2.6e-6 |
| 5 | 0.00032 | … | 1.0e-7 |

5 步就把 θ 从 1 拉到了 3.2e-4，几乎贴着 0。**指数级收敛**——这是凸函数 + 合适学习率的甜蜜点。

> 试一下：把 α 改成 1.1，θ 会发散到 ±∞；改成 0.001，要跑几千步才能到 0。可视化见 [`demos/gd-landscape.py`](./demos/gd-landscape.py)。

## 凸 vs 非凸（一句话预告）

LR 的损失函数（MSE）是**凸函数**——一个碗，GD 必收敛到全局最优。

神经网络的 loss 是**非凸**——很多坑，GD 可能卡在局部最优或**鞍点**（saddle point：某些方向上是极小值、其他方向是极大值的"马鞍形"点，梯度=0 但不是最优）。这是后续 [`03-LogReg`](../../03-LogReg/) 之后讲优化器（Momentum / Adam）的动机。
