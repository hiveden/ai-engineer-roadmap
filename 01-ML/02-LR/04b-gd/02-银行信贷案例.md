## ━━━━━━━━ 底稿 ━━━━━━━━

### PPT

#### Slide 71-73 · 梯度下降之 多元线性回归

> PPT 这 3 页全是图（标题统一为"梯度下降法 之 多元线性回归"，slide 文字仅标题）。核心内容：把 GD 套到多元 LR 上，对 $\mathbf{w}$ 求偏导得到梯度向量，应用更新公式。

**多元 LR 损失函数**：

$$L(\mathbf{w}) = \frac{1}{n}\sum_{i=1}^{n}(y_i - \mathbf{w}^T \mathbf{x}_i)^2$$

**对 w 求偏导得梯度向量**：

$$\nabla_\mathbf{w} L = -\frac{2}{n} X^T (\mathbf{y} - X\mathbf{w})$$

**应用更新公式**：

$$\mathbf{w} \leftarrow \mathbf{w} - \alpha \nabla_\mathbf{w} L = \mathbf{w} + \frac{2\alpha}{n} X^T (\mathbf{y} - X\mathbf{w})$$

> 一行矩阵运算搞定 d 个权重的同步更新，是 GD 落到 LR 上的工程公式。

### 笔记

> 【理解】银行信贷案例

**场景**：银行根据客户**收入**预测**贷款额度**。

**模型**：$\hat{y} = w x + b$，损失 $L(w, b) = \frac{1}{n} \sum (y_i - w x_i - b)^2$。

**梯度下降迭代**：

$$w \leftarrow w - \alpha \cdot \frac{\partial L}{\partial w}, \qquad b \leftarrow b - \alpha \cdot \frac{\partial L}{\partial b}$$

其中：

$$\frac{\partial L}{\partial w} = -\frac{2}{n} \sum x_i (y_i - w x_i - b), \qquad \frac{\partial L}{\partial b} = -\frac{2}{n} \sum (y_i - w x_i - b)$$

**关键超参 学习率 $\alpha$**：太小收敛慢；太大震荡发散；经验值 0.001-0.1，配合标准化后的特征。

**收敛判据**：$\lvert L_{t} - L_{t-1} \rvert < \epsilon$ 或达到最大迭代数。

## ━━━━━━━━ 讲解 ━━━━━━━━

### 场景与数据

银行手里有历史样本——「这个客户当年收入多少 → 我们批了多少额度」：

| 客户 | $x$ 收入（万）| $y$ 额度（万）|
|---|---|---|
| 1 | 4.0 | 20 |
| 2 | 8.0 | 50 |
| 3 | 5.0 | 30 |
| 4 | 7.5 | 45 |
| ... | ... | ... |

业务问题：**新客户报来一个收入 x，应该批多少 y？**

工程视角：这是个监督学习的回归任务。模型只有两个旋钮——斜率 $w$（收入每涨 1 万，额度涨多少）和截距 $b$（收入为 0 时的基础额度，可能为负，表示不批）。

### 推导：MSE 对 w 和 b 的偏导

设单样本误差 $e_i = y_i - w x_i - b$。则 $L = \frac{1}{n} \sum e_i^2$。

链式法则（[`03b-math/04-偏导.md`](../03b-math/04-偏导.md)）：

$$\frac{\partial L}{\partial w} = \frac{1}{n} \sum 2 e_i \cdot \frac{\partial e_i}{\partial w} = \frac{1}{n} \sum 2 e_i \cdot (-x_i) = -\frac{2}{n} \sum x_i (y_i - w x_i - b)$$

$$\frac{\partial L}{\partial b} = \frac{1}{n} \sum 2 e_i \cdot \frac{\partial e_i}{\partial b} = \frac{1}{n} \sum 2 e_i \cdot (-1) = -\frac{2}{n} \sum (y_i - w x_i - b)$$

**直觉**：

- 梯度里的 $(y_i - \hat{y}_i)$ 就是「预测差多少」
- 对 w 的梯度还乘了 $x_i$——收入大的样本对斜率影响大（合理）
- 对 b 的梯度只是平均误差——整体预测偏低，b 就该往上调

### 更新公式

$$w \leftarrow w - \alpha \cdot \frac{\partial L}{\partial w}, \qquad b \leftarrow b - \alpha \cdot \frac{\partial L}{\partial b}$$

代入偏导：

$$w \leftarrow w + \frac{2\alpha}{n} \sum x_i (y_i - w x_i - b)$$

$$b \leftarrow b + \frac{2\alpha}{n} \sum (y_i - w x_i - b)$$

> 注意：常数因子 2 经常被吸收进 α，所以很多教材直接写 $\frac{1}{n}$ 而不是 $\frac{2}{n}$——不影响结果，只是 α 的具体数值差 2 倍。

### 一次迭代的完整伪代码

```python
# 输入：x[n], y[n]，超参 α, max_iter, ε
# 初始化
w, b = 0.0, 0.0
prev_loss = float('inf')

for t in range(max_iter):
    # 1. 前向：预测
    y_hat = w * x + b                       # 长度 n 的向量

    # 2. 算 loss（用于监控收敛）
    loss = ((y - y_hat) ** 2).mean()

    # 3. 算梯度
    grad_w = -2 * (x * (y - y_hat)).mean()  # ∂L/∂w
    grad_b = -2 * (y - y_hat).mean()        # ∂L/∂b

    # 4. 更新
    w -= α * grad_w
    b -= α * grad_b

    # 5. 收敛判据
    if abs(prev_loss - loss) < ε:
        break
    prev_loss = loss
```

5 步就是 GD 的全部。换成多元 LR 只是把 `x` 变成矩阵 `X`、`w` 变成向量、梯度变成 $\nabla_\mathbf{w} L = -\frac{2}{n} X^T (\mathbf{y} - X \mathbf{w} - b)$——形式一样，维度变了。

### 跑通要注意的工程细节

| 坑 | 后果 | 解法 |
|---|---|---|
| 多特征量纲悬殊（收入 + 年龄 + 流水）| GD 震荡甚至发散 | 标准化（StandardScaler）|
| 初始 w, b 全设 0 | 一元 LR 可以；多层网络会**对称失效** | LR 没事；DL 必须随机初始化 |
| α 拍脑袋选 | 大概率不收敛 | 先小后大，画 loss 曲线观察 |
| 没有 max_iter 兜底 | 不收敛时死循环 | 永远设 max_iter |
| 整数除法 | Python 2 / numpy int 数组遇到 / 会截断 | 强制 float |

### 和正规方程的对比预告

这个一元案例其实有闭式解（[`04a-analytical/`](../04a-analytical/)）：

$$w = \frac{\sum (x_i - \bar{x})(y_i - \bar{y})}{\sum (x_i - \bar{x})^2}, \qquad b = \bar{y} - w \bar{x}$$

一行代码出结果，比 GD 快得多。**但**——当特征数 > 1 万、样本数 > 1 亿，或者损失函数压根没有闭式解（LogReg / NN）时，GD 是唯一选择。详见 [`04`](./04-正规方程vs梯度下降.md)。
