# 正规方程 vs 梯度下降

> 级别：【理解】
> 维度：算法选型
> 章节底稿全文见 [`README.md`](./README.md)（PPT slide 77）

## 底稿

LR 求最优参数有两条路：

- **正规方程**（normal equation）：闭式解一步出答案 $\mathbf{w} = (X^T X)^{-1} X^T \mathbf{y}$（详见 [`04a-analytical/`](../04a-analytical/)）
- **梯度下降**（GD）：迭代逼近 $\mathbf{w} \leftarrow \mathbf{w} - \alpha \nabla L$（前三节）

二者**结果一样**（都收敛到 MSE 的全局最优），区别在**怎么算出来**。本节讲什么时候选哪个。

---

## 直觉

一句话：**正规方程 = 一招制敌的招式（前提：题目允许）；梯度下降 = 万金油刀法（任何模型都能砍）**。

深度学习时代基本只用 GD——但理解正规方程能帮你看穿 LR 的本质：**MSE 是凸函数，全局唯一解**，所以 GD 收敛后必然等于正规方程的解。

## 8 维对比表

| 维度 | 正规方程 | 梯度下降 |
|---|---|---|
| **求解方式** | 一步闭式解 $\mathbf{w} = (X^T X)^{-1} X^T \mathbf{y}$ | 迭代逼近，逐步逼近最优 |
| **学习率 α** | 不需要 | **必须调**（最敏感超参之一）|
| **特征标准化** | 可有可无（不影响结果，只影响数值稳定）| **强烈建议**（量纲悬殊 → 椭圆碗 → 震荡）|
| **特征数 d** | $d \lesssim 10^4$（受 $O(d^3)$ 求逆限制）| 任意（每步 $O(d)$）|
| **样本数 n** | 受内存限制（X 必须能塞进 RAM）| 任意（SGD/MBGD 流式读取）|
| **适用算法** | **几乎只有 LR**（极少模型有闭式解）| LR / LogReg / SVM / NN / 几乎所有 |
| **典型场景** | 教学 / 小数据 / 特征明确独立 | 大数据 / 深度学习 / 在线学习 / **现代 ML 默认** |
| **工程优势** | 代码 1 行、无超参、结果确定 | 通用、可扩展、显存友好、能并行 |

## 何时选哪个：决策树

```
特征数 d < 1 万   ─┬─ 是 ─→ 样本数 n 能塞进内存 ─┬─ 是 ─→ X^T X 可逆？─┬─ 是 ─→ 正规方程
                                                                    └─ 否 ─→ 加 L2（岭回归）正规方程
                                              └─ 否 ─→ 梯度下降
                  └─ 否 ─→ 梯度下降（必选）

模型 ≠ LR ─→ 梯度下降（基本没的选）
```

## 工程师的决策口诀

1. **教学和小数据（n < 1 万、d < 1000）**：正规方程，1 行 numpy 代码 `np.linalg.solve(X.T @ X, X.T @ y)`，无超参可调，结果稳定。
2. **传统 ML 中等数据（n ~ 100 万）**：sklearn `LinearRegression` 内部默认就用正规方程或 SVD；超大用 `SGDRegressor`。
3. **深度学习 / 上亿参数**：GD 是**唯一选择**。NN 没有闭式解、参数矩阵求逆 $O(d^3)$ 不可承受。
4. **在线学习 / 数据流**：SGD 必选——数据来一条更一次，闭式解每来一条都要重算 $(X^T X)^{-1}$，灾难。

## 为什么深度学习不能回头用正规方程

三个硬约束：

| 约束 | 深度学习现状 | 正规方程是否可行 |
|---|---|---|
| 模型可微但**非线性**、loss 非凸 | ReLU / sigmoid 嵌套层 | 不可行（没闭式解）|
| 参数量 d | GPT 级别 1B-1T | $O(d^3)$ 求逆 = 宇宙寿命级别 |
| 样本量 n | ImageNet 1.4M、CommonCrawl PB 级 | $X^T X$ 都构造不出来 |

→ 即使你想用，也用不了。

## 正规方程的隐藏坑

PPT slide 79 第 2 题考过：「$X^T X$ **一定**可逆吗？」**答案是不一定**。

- 特征间**完全共线**（一个特征 = 另两个的线性组合）→ $X^T X$ 不满秩、不可逆
- 特征数 > 样本数（n < d）→ 几乎必然不可逆
- 解决：加 L2 正则化（$\lambda I$ 项）→ **岭回归（Ridge）**，详见 [`07b-regularization/`](../07b-regularization/)

GD 没有「不可逆」这个问题——永远能跑，最坏不收敛。这也是 GD 更鲁棒的原因之一。

## 标准化为什么对 GD 更关键

正规方程的解析式不挑量纲（数学上等价）。但 GD 不一样：

- 量纲悬殊 → loss 等高线变成**狭长椭圆**
- GD 在椭圆里**沿短轴震荡**，沿长轴爬得慢 → 收敛极慢甚至发散

标准化把碗形拉圆，GD 一路下滑到底。详细几何意义见 [`demos/gd-landscape.py`](./demos/gd-landscape.py)。

## Tech Lead 视角

- 正规方程是「数学家视角」——优雅、闭式、无超参
- GD 是「工程师视角」——通用、可扩展、抗噪
- 学正规方程的目的不是为了用它，而是**理解 LR 是凸优化、有唯一全局最优**。理解后再看 GD：
  - 为什么 GD 在 LR 上一定收敛？→ 凸函数 + 合适 α
  - 为什么 GD 在 NN 上可能卡住？→ 非凸 + 局部最优 + 鞍点

→ 这是后续 [`03-LogReg`](../../03-LogReg/) 之后讲优化器（Momentum / Adam / RMSProp）的前置认知。
