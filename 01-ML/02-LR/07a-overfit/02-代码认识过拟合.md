```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error
from sklearn.model_selection import train_test_split

def dm01_模型欠拟合():
    # 1 准备数据 x y（增加上噪声）
    np.random.seed(666)
    x = np.random.uniform(-3, 3, size=100)
    y = 0.5 * x**2 + x + 2 + np.random.normal(0, 1, size=100)
    # 2 实例化线性回归模型
    estimator = LinearRegression()
    # 3 训练模型
    X = x.reshape(-1, 1)
    estimator.fit(X, y)
    # 4 模型预测
    y_predict = estimator.predict(X)
    # 5 计算均方误差
    myret = mean_squared_error(y, y_predict)
    print('myret-->', myret)
    # 6 画图
    plt.scatter(x, y)
    plt.plot(x, y_predict, color='r')
    plt.show()
```

> 数据是抛物线非线性的，用线性模型去拟合。模型过于简单，**出现欠拟合**。
>
> `myret --> 3.0750025765636577`

### 实验设计

**已知真相**：构造一份合成数据，真实关系是抛物线 + 高斯噪声。

$$y = 0.5 x^2 + x + 2 + \varepsilon, \quad \varepsilon \sim \mathcal{N}(0, 1), \quad N = 100$$

**未知**（假装我们不知道真相）：用线性回归 + 多项式特征去拟合，分别试 degree = 1 / 2 / 10，看每种情况下训练 MSE（→ [`05-metrics/`](../05-metrics/)）和测试 MSE 的变化。

```python
import numpy as np
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error
from sklearn.model_selection import train_test_split

np.random.seed(666)
x = np.random.uniform(-3, 3, size=100)
X = x.reshape(-1, 1)
y = 0.5 * x**2 + x + 2 + np.random.normal(0, 1, size=100)
```

### 三段对照实验（全数据训练 MSE）

#### 阶段 1 · degree = 1（仅 X，一次项）→ 欠拟合

```python
estimator = LinearRegression().fit(X, y)
mse1 = mean_squared_error(y, estimator.predict(X))
# mse1 = 3.0750
```

数据是抛物线，硬用直线拟合 —— 模型过于简单，训练误差就大。

#### 阶段 2 · degree = 2（X, X²）→ good fit

```python
X2 = np.hstack([X, X**2])
estimator = LinearRegression().fit(X2, y)
mse2 = mean_squared_error(y, estimator.predict(X2))
# mse2 = 1.0987 ≈ 噪声方差 σ² = 1
```

模型恰好匹配数据生成过程，**MSE 已经接近噪声下限**（不可约误差），再降也降不了多少。

#### 阶段 3 · degree = 10（X, X², ..., X¹⁰）→ 过拟合

```python
X10 = np.hstack([X**i for i in range(1, 11)])
estimator = LinearRegression().fit(X10, y)
mse10 = mean_squared_error(y, estimator.predict(X10))
# mse10 = 1.0508  （训练 MSE 还在降！但拟合曲线已经在数据点之间剧烈抖动）
```

### 关键观察

| degree | 训练 MSE | 备注 |
|---|---|---|
| 1 | 3.0750 | 欠拟合 |
| 2 | 1.0987 | good fit |
| 10 | 1.0508 | **训练 MSE 仍下降** —— 但你眼睛看图就知道曲线疯了 |

**陷阱**：训练 MSE 单调下降 ≠ 模型变好。它是过拟合的「沉默共犯」，单看它判断不了过拟合。

### 用 train_test_split 揭穿过拟合

把数据划成训练集 / 测试集，重新跑：

```python
def eval_split(features, y, seed=5):
    X_tr, X_te, y_tr, y_te = train_test_split(features, y, random_state=seed)
    m = LinearRegression().fit(X_tr, y_tr)
    return mean_squared_error(y_te, m.predict(X_te))

print(eval_split(X,   y))  # degree=1  → 3.1531
print(eval_split(X2,  y))  # degree=2  → 1.1119
print(eval_split(X10, y))  # degree=10 → 1.4146  ← 反弹！
```

| degree | 训练 MSE | 测试 MSE | 区域 |
|---|---|---|---|
| 1 | 3.0750 | **3.1531** | 欠拟合（两边都大） |
| 2 | 1.0987 | **1.1119** | good fit（两边都小且接近） |
| 10 | 1.0508 | **1.4146** | 过拟合（训练 ↓ 测试 ↑） |

**这就是 U 形曲线在三个点上的离散显形**：训练 MSE 单调降，测试 MSE 先降后升 —— 中间那个最低点（degree=2）就是 [`01-欠拟合与过拟合.md`](./01-欠拟合与过拟合.md) 说的拐点。
