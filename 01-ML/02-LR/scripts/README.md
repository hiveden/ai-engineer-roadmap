## 拆分理由摘要（按 _0-workflow §2 五条对照）

1. **e01 = 01-intro 全章合一期**（4 个 md + 1 个 demo 合并）
   理由：4 个 md 都是【理解】/【知道】级，单独成期太短；`k-to-w-migration` demo 横跨"一元 → 多元 + 3D"，恰好覆盖 03-分类 这一节；合一期约 15 min 刚好。

2. **e02 = 02-api 独立 1 期**
   理由：实操 API 章，`api-walkthrough.py`（5 步 + 6 场景）独立成期；为后续 e09 加州房价做 sklearn API 直觉铺垫。

3. **e03 拆分点：损失函数从 03b-math 单拆**
   理由：损失是【掌握】级（其他 5 个数学知识点是【知道】级），且 `mse-residual-squares.py` demo 强绑定损失。损失独立成期突出"模型评估 + 求最优"主线，后续数学才有动机。

4. **e04 拆双期：e04a 偏导线 + e04b 矩阵线**
   - **e04a-偏导与极值** = 导数 + 偏导，核心神器是「**导数 = 0 找极值**」（LR 找最优 (k, b) 的钥匙）；配独立 demo `partial-derivative.py`（端口 2762）做 3D 曲面 + 切片可视化。
   - **e04b-向量与矩阵** = 数据表述 + 向量 + 矩阵，核心神器是「**矩阵化 = 一行公式覆盖所有训练样本**」（LR 多元的语言）；配独立 demo `matrix-mul-viz.py`（端口 2763）做维度配对动画。
   - 拆分理由（按 `_0-workflow.md` §2）：原 e04 把 5 件套压成 10 min 太赶，且偏导 / 矩阵两条线**功能不同**——前者是「找极值的方法论」、后者是「批量计算的语言」，合期会模糊主线。拆双期让 E05 正规方程能直接「调用 E04ab 已铺好的桥」，不再「现讲数学现用」。

5. **e05 = 04a 全章合一期**
   理由：一元 + 多元正规方程是同一思路（偏导=0）的两种规模，合一期讲"凸优化的教科书答案"。**调用 E04a 偏导 + E04b 矩阵化一气呵成推导** $\mathbf{w} = (X^T X)^{-1} X^T \mathbf{y}$，本期不引入新数学。无 demo（公式推导用动画）。

6. **e06 拆分点：梯度下降思想 + 银行信贷案例从 04b 拆出独立期** ⭐核心
   理由：`gd-landscape.py`（5 视图 GD）是全章最重 demo，"坡度最陡下山法"思想需要充分时间消化，估 ~18 min 合理。银行信贷案例作为思想的具象例子拼入本期，不单独成期。

7. **e07 = 04b 后半合一期（BGD/SGD/MBGD + vs 正规方程）**
   理由：BGD/SGD/MBGD 是【了解】级一句话清单 + vs 正规方程是【理解】级对比表，合并约 12 min，纯概念无 demo。

8. **e08 = 05 全章合一期**
   理由：MAE/MSE/RMSE 三指标 + 比较，本质一组对比；`metric-vs-outlier.py` 一个 demo 即覆盖三指标对异常点的差异。【知道】级合并合理。

9. **e09 = 06 全章合一期**
   理由：加州房价 5 个 md 是同一 pipeline 的连贯步骤（API 复习 → 数据 → 标准化 → 训练 → 评估），不可拆。无独立 marimo demo（用 jupyter 录屏代替，参考 KNN e04/e05 模式）。

10. **e10 = 07a 全章合一期**
    理由：欠拟合/良好/过拟合三档 + `poly-degree-overfit.py` U 形 demo 一气呵成，3 个 md 自然合并。

11. **e11 = 07b 全章合一期**
    理由：L1/L2 是同一组对比（菱形 vs 圆形），`l1-l2-shrinkage.py` 一个 demo 同时覆盖；3 个 md 合一期 ~15 min。

## 衔接钩主线（每期 cta 伏笔 → 下期 hook 应答）

> **数学桥主线（e03 → e05）**：损失（e03）→ 偏导极值（e04a）→ 矩阵化（e04b）→ 一气呵成正规方程（e05）。E04ab 把桥铺好，E05 只走过去。

```
e01 cta: "API 怎么 4 行调用？"
       ↓
e02 hook: "上期我们讲 LR 是什么，这期 4 行 sklearn 跑通"
e02 cta: "API 把直线找到了——但它内部是怎么找的？什么是好直线？"
       ↓
e03 hook: "上期 fit 一行就出结果，这期我们打开黑盒：先定义 '好' = 损失最小"
e03 cta: "MSE 抛物线底就是最优 k——但凭什么导数为零就是最低？需要点数学"
       ↓
e04a hook: "上期抛物线底 = 最优 k——这期把'导数=0 找极值'这把钥匙磨亮：导数 → 偏导 → 梯度"
e04a cta: "偏导=0 钥匙握上了——但多元 LR 要联立 n+1 个方程，一行行写太累，下期上矩阵语言"
       ↓
e04b hook: "上期我们拿到了偏导钥匙——这期升级到矩阵化：一行 $X\mathbf{w}$ 覆盖所有样本"
e04b cta: "E04a 偏导 + E04b 矩阵两件家伙齐了——下期合体一气呵成正规方程"
       ↓
e05 hook: "E04a 偏导钥匙 + E04b 矩阵语言两件家伙合体，这期一气呵成推 w = (X^T X)^{-1} X^T y"
e05 cta: "正规方程很美，但特征数 1 万 +、X^T X 不可逆怎么办？需要更通用的方法"
       ↓
e06 hook: "上期一步出解很爽，但现实数据吃不消——这期换思路：迭代下山" ⭐
e06 cta: "下山每步用全部样本算梯度——大数据怎么办？还有 SGD 和 mini-batch"
       ↓
e07 hook: "上期下山思想懂了，这期看三种下山方式 BGD/SGD/MBGD + 对比正规方程"
e07 cta: "训练完了——但模型好不好得有评估指标，光看 loss 不直观"
       ↓
e08 hook: "上期我们训练出模型，这期换个角度：怎么评估它？MAE/MSE/RMSE 三件套"
e08 cta: "三种指标有了，但单元数据集太抽象——下期上真实案例：加州房价"
       ↓
e09 hook: "上期讲完评估指标，这期把全套 pipeline 跑一遍：加州房价"
e09 cta: "案例跑完看似完美——但拟合得'太好'是好事吗？训练误差 0 测试误差爆炸的现象"
       ↓
e10 hook: "上期 pipeline 顺利收尾，这期我们看一个反直觉现象：训练越准，测试越差" U 形
e10 cta: "过拟合解决方案除了减特征 / 加数据，还有个更工程的招——正则化"
       ↓
e11 hook: "上期讲过拟合的病，这期讲药：L1 砍特征 / L2 整体收缩"
e11 cta: "LR 章 12 期完结。下一章我们把 LR 加 sigmoid 推广到分类问题——逻辑回归"
       ↓ （02-LR 大章节钩 → 03-LogReg 章节）
```

## 已知灰色地带（待用户审）

1. **e04a / e04b 双期是否平衡**：原 e04 合期 ~10 min 太挤，拆双期后各 ~12 min 更舒展，但风险是**两期都依赖独立 demo（2762 / 2763）**——demo 同时在做（agent 并行），若任一 demo 延期，对应期会变成"纯动画"形态（仍可成片，但视觉密度下降）。预案：demo 没就绪时先用公式定格动画占位，后期补录屏。
2. **e04a 与 e04b 的"独立性 vs 连贯性"**：两期教学目标不同（钥匙 vs 语言），但同处 03b-math 章节，紧邻播出。需要 e04b 的 hook 强呼应 e04a 的 cta（已在 plan.md 中对齐），避免观众觉得"为什么数学又分两期"。如果用户回看时发现观感断裂，可考虑做"e04a + e04b 合并版"特供 30 min 长视频（不影响 B 站 12+12 切片版）。
3. **e07 是否合期**：BGD/SGD/MBGD + vs 正规方程，无 demo、纯概念表格，估时 12 min。如果用户认为太干，可以给它配个轻量 demo（学习率振荡可视化），那样 e06 的 `gd-landscape.py` 主线就只承载"思想"。当前方案是合期 + 无 demo，最简化。
4. **e09 是否独立 marimo demo**：加州房价 pipeline 教材代码非常成熟（13 + 13 行的 sklearn 直跑），KNN e04/e05 同类型走的是 jupyter 录屏路线（README 已注明"Jupyter 录屏"）。本期沿用同模式，不写独立 marimo demo——若用户希望统一走 marimo 风格，需补一个 `06-boston/demos/california-pipeline.py`。

## 立场要点（沿用 KNN 章）

- **第一人称：我们**（"我们来看"/"我们注意"）
- **AI 老师**主讲，不端权威——结尾留"如有不对欢迎指正"
- **教学语气**可断言，但不强势；用"想象"/"假设"/"注意"做引导
- **不引官方文档**——直接讲概念
- **术语首次出现中英双标**：线性回归（Linear Regression，LR）、均方误差（Mean Squared Error，MSE）、梯度下降（Gradient Descent，GD）等
