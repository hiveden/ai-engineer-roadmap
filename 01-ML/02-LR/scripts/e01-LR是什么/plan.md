## 教学目标（3-5 条）

1. 知道 LR 解决的是**回归（连续值预测）**问题
2. 理解"先定模型再求参数"——y = wx + b 的两个参数 (w, b)
3. 知道一元（单特征）与多元（多特征）的区别，多元用 w·x + b（向量内积）
4. 知道 LR 的应用场景（房价 / 销售 / GDP / 剂量-反应等线性关系）
5. 知道何时**不要**用 LR（离散类别 / 强非线性 / 共线性）
