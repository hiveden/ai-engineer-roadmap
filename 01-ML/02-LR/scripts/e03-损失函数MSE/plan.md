## 教学目标（3-5 条）

1. 理解"误差 = 预测值 - 真实值"，损失函数是综合衡量
2. 掌握 MSE（均方误差）公式 $\frac{1}{n}\sum(y_i - \hat y_i)^2$ + 为什么是平方而非绝对值（光滑可导 + 重罚大误差）
3. 知道损失越小模型越好——LR 的目标是 **min L(k, b)**
4. 知道损失也叫代价函数 / 目标函数 / cost function（同一概念多个名字）
5. 直觉理解：固定 b 后，L(k) 是 k 的二元二次方程（抛物线），最小值在抛物线底

## 本期主线案例

**沿用 E01/E02 身高体重数据**：5 样本拟合 y = kx + b。

固定 b = -100（PPT 简化）后：
$$L(k) = 145416k^2 - 281671.6k + 136496.32$$

抛物线底 k = 281671.6 / (2 × 145416) ≈ **0.9685**（与 E02 sklearn 实测 ~0.93 接近，差异因 b 固定值不同——讲解时一句话带过即可）。
