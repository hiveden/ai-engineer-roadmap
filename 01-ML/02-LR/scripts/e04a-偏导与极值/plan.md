5. 知道梯度 ∇ = 所有偏导组成的列向量；$\nabla L = \mathbf{0}$ ⇔ 所有偏导同时为零（LR 联立解的接口）

## 本期主线案例

**主线：从 E03 的抛物线一步步升到二元偏导**。

1. **一元导数**（呼应 E03）：$L(k) = 145416 k^2 - 281671.6 k + 136496.32$（E03 留下的抛物线）。
   - $L^\prime(k) = 290832 k - 281671.6$
   - 令 $L^\prime(k) = 0$ → $k \approx 0.9685$ ✅ 与 E03 抛物线底数值一致
   - **这就是「导数 = 0 找极值」神器的第一次落地**

2. **二元偏导**：松开 b，$L(k, b) = \sum (k x_i + b - y_i)^2$（5 样本身高体重，沿用 E01-E03 数据）。
   - 对 k 偏导：$\partial L / \partial k = \sum 2(k x_i + b - y_i) x_i$
   - 对 b 偏导：$\partial L / \partial b = \sum 2(k x_i + b - y_i)$
   - 梯度向量 $\nabla L = [\partial L/\partial k, \partial L/\partial b]^T$
   - **联立 ∇L = 0 → 两条方程两未知数**（具体解法埋钩 E05，本期只到「列出方程」为止）

3. **链式法则的隐式登场**：$L^\prime(k)$ 中 $(k x_i - y_i)^2$ 求导用的就是链式（外平方 → 内一次）；不展开公式，但口播一句"嵌套函数先外后内，神经网络反向传播 backprop 就靠这个"。
