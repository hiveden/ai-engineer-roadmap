## 教学目标（3-5 条）

1. 了解一元解析解：对 L(k, b) 分别求偏导 = 0 → 联立 → 公式解 (k, b)
2. 了解最小二乘法 OLS（Ordinary Least Squares）= 平方和误差闭式解
3. 了解多元正规方程：w = (X^T X)^{-1} X^T y（Normal Equation）
4. 知道正规方程的优点（一步出解 / 不调学习率）和缺点（X^T X 须可逆 / O(n^3) / 大数据慢）
5. 知道 sklearn 的 LinearRegression 内部用 SVD 分解，是正规方程的稳定版

## 核心方法论：E04ab 已铺桥，E05 只走过去

本期**不再现讲数学现用**——所有数学家伙事 E04ab 已备齐：

| E04ab 给的工具 | E05 怎么用 |
|---|---|
| E04a · 导数 = 0 找极值 | 一元损失 $L(k, b)$ 分别对 k、b 求偏导 → 联立 = 0 → 解 (k, b) |
| E04a · 偏导 / 梯度 ∇ | 多元损失 $L(\mathbf{w})$ 对向量 $\mathbf{w}$ 求梯度 → $\nabla L = \mathbf{0}$ |
| E04b · 矩阵化 $X\mathbf{w}$ | 损失矩阵形式 $L(\mathbf{w}) = (\mathbf{y} - X\mathbf{w})^T(\mathbf{y} - X\mathbf{w})$ |
| E04b · 转置 + 求逆 | 整理 $X^T X \mathbf{w} = X^T \mathbf{y}$ → 两侧乘 $(X^T X)^{-1}$ |

→ **一气呵成**：$\boxed{\mathbf{w} = (X^T X)^{-1} X^T \mathbf{y}}$。每一步只调用 E04ab 已讲过的概念，本期不引入新数学。

## 本期主线案例

**一元**：沿用身高体重 5 样本，用均值公式：

$$k = \frac{\sum(x_i - \bar x)(y_i - \bar y)}{\sum(x_i - \bar x)^2}, \quad b = \bar y - k\bar x$$

代入：x̄ = 170.4，ȳ = 65.1 → k ≈ 0.93、b ≈ -93.5、预测 176 ≈ 70.2 kg。
**与 E01 / E02 跨期对齐**（铁律）。
