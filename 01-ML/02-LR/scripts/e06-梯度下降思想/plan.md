## 教学目标（3-5 条）

1. 掌握梯度下降（Gradient Descent，GD）的核心思想：沿负梯度方向迭代逼近最低点
2. 理解梯度 = 偏导组成的向量，方向 = 函数上升最快的方向（取负即下降最快）
3. 掌握更新公式 **w ← w - α∇L**：α 是学习率（步长）
4. 知道学习率的影响：太小慢、太大震荡 / 发散；经验值 0.001-0.01
5. 知道收敛判据：相邻两步损失差 < ε 或达到最大迭代数

## 本期主线案例

**主轴 1：单变量 f(θ) = θ²，初值 θ=1，α=0.4**

PPT 经典例：

| 步 | θ | 更新 |
|---|---|---|
| 1 | 1.0 | — |
| 2 | 0.2 | 1 - 0.4·(2·1) |
| 3 | 0.04 | 0.2 - 0.4·(2·0.2) |
| 4 | 0.008 | … |
| 5 | 0.0016 | 接近 0 |

4 步后基本到底——直觉地展示"步步逼近"。

**主轴 2：银行信贷案例**（具象）

x = 客户收入，y = 贷款额度，模型 ŷ = wx + b。
讲偏导 ∂L/∂w 和 ∂L/∂b 的具体表达式，配学习率 α 的取值经验。
