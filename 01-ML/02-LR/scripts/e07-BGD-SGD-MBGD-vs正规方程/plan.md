## 教学目标（3-5 条）

1. 了解梯度下降按"每步用多少样本算梯度"分三类：
   - **BGD** 批量梯度下降（全部 n 个样本）
   - **SGD** 随机梯度下降（1 个样本随机抽）
   - **MBGD** 小批量梯度下降（一批 batch_size 个，主流默认）
2. 知道直觉对比："看完整本书 / 看一句话 / 看一段"再下结论
3. 知道现代深度学习 ~~几乎都用 MBGD~~（batch_size 32-512）
4. 理解正规方程 vs 梯度下降的全面对比（求解方式 / 学习率 / 特征数 / 样本数 / 适用算法 / 典型场景）
5. 知道 sklearn `SGDRegressor` 是 SGD 实现 / `LinearRegression` 是正规方程实现

## 本期主线案例

**对比表 1：三种 GD**

| 算法 | 每次用样本数 | 优点 | 缺点 | sklearn |
|---|---|---|---|---|
| **BGD** | n（全部）| 梯度准、收敛稳 | 慢、内存爆 | — |
| **SGD** | 1（随机）| 极快、能跳出局部最优 | 噪声大、震荡 | `SGDRegressor` |
| **MBGD** | batch（如 32/64）| 兼顾速度 + 稳定 | 多 1 个超参 | 深度学习默认 |
