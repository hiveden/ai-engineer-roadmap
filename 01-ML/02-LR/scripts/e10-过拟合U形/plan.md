## 教学目标（3-5 条）

1. 理解欠拟合（模型太简单）/ 良好（适中）/ 过拟合（太复杂）三档区分
2. 知道判定标志：训练误差 vs 测试误差曲线分化是过拟合的典型信号
3. 实操多项式次数 U 形曲线：训练误差单调下降 / 测试误差先降后升
4. 理解 bias-variance 权衡：欠拟合 = high bias，过拟合 = high variance
5. 知道 4 类解决办法：清洗数据 / 加数据 / 正则化 / 减特征——埋钩 E11 正则化

## 本期主线案例

**合成数据**（教材统一用 np.random.seed(666)）：

```python
np.random.seed(666)
x = np.random.uniform(-3, 3, size=100)
y = 0.5 * x**2 + x + 2 + np.random.normal(0, 1, size=100)  # 抛物线 + 噪声
```

**三档对比**（教材实测 MSE）：

| 模型 | 训练 MSE | 测试 MSE（random_state=5） | 状态 |
|---|---|---|---|
| **X**（一次项）| 3.075 | 3.153 | 欠拟合 |
| **X + X²**（二次项）| 1.099 | 1.112 | **良好** |
| **X^1 ~ X^10**（10 次项）| ↓ ~0.85 | 1.415 | **过拟合**（训练继续降但测试反弹） |

→ 这是 U 形曲线的核心证据。
