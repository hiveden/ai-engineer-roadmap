> Slide 7 · 逻辑回归数学基础 – 极大似然估计

极大似然估计

核心思想：根据观测到的结果 来 估计模型算法中的未知参数
即：什么样的参数，才最可能产生观测到的现象？

举个例子

假设有一枚不均匀的硬币，出现正面的概率和反面的概率是不同的。假定出现正面的概率为θ，
抛了6次得到如下现象 D = {正面，反面，反面，正面，正面，正面}。每次投掷事件都是相互独立的。
则根据产生的现象D，来估计参数θ是多少？

未知参数p（待估计），产生的现象和后果（估计）：

P(D|θ) = P {正面，反面，反面，正面，正面，正面}
 = P(正面|θ) P(反面|θ) P(反面|θ) P(正面|θ) P(正面|θ) P(正面|θ)    联合概率
= θ *(1-θ)*(1-θ)*θ*θ*θ = θ⁴(1-θ)²

问题来了：求此函数的极大值时（将"可能性"最大化），估计θ为多少！  极大似然法

f(θ) = θ⁴(1-θ)²  令导数=0求极值，可估计出θ值

∂f(θ)/∂θ = 4θ³(1-θ)² + θ⁴ * 2(1-θ)(-1)
= 4θ³(1-θ)² - 2θ⁴(1-θ)
= θ³(1-θ)[4(1-θ) - 2θ]
= θ³(1-θ)(4-6θ) = 0

从而得：θ=0，θ=1，θ=2/3；取 θ=2/3

### 笔记

> 【了解】应用场景

逻辑回归（Logistic Regression, LR）是解决**二分类**问题的利器。名字带"回归"，本质是分类——线性回归输出连续值，LR 把它压到 [0, 1] 当作概率，再按阈值切类。

典型场景：

- 广告点击率预估（CTR）
- 是否流失 / 是否欺诈 / 是否垃圾邮件
- 信用评分卡（金融风控）
- 疾病诊断阳性 / 阴性

工程视角：LR 训练快、可解释（权重 = 特征重要性）、易上线（一行点积 + sigmoid），是工业界最常见的 baseline。

> 【熟悉】数学知识

> 【知道】sigmoid 函数

把任意实数 $z \in (-\infty, +\infty)$ 压到 $(0, 1)$ 区间，作为概率使用。LR 的输出层就是它。

**定义**：

$$\sigma(z) = \frac{1}{1 + e^{-z}}$$

**关键性质**：

| 性质 | 公式 / 描述 |
|---|---|
| 值域 | $\sigma(z) \in (0, 1)$ |
| 中点 | $\sigma(0) = 0.5$ |
| 单调性 | 严格单调递增 |
| 对称性 | $\sigma(-z) = 1 - \sigma(z)$ |
| 导数 | $\sigma^\prime(z) = \sigma(z)(1 - \sigma(z))$ |

**工程接口**：

```python
import numpy as np
def sigmoid(z): return 1.0 / (1.0 + np.exp(-z))
# scipy 版（数值稳定）
from scipy.special import expit  # expit(z) == sigmoid(z)
```

> 【理解】概率

**概率**（probability）：随机事件发生可能性的度量，取值 $P \in [0, 1]$。

**联合概率**：多个独立事件同时发生，等于各自概率相乘。

$$P(A \cap B) = P(A) \cdot P(B) \quad (\text{当 } A, B \text{ 独立})$$

**条件概率**：在事件 $B$ 已发生的前提下 $A$ 发生的概率。

$$P(A \mid B) = \frac{P(A \cap B)}{P(B)}$$

LR 中的输出 $\sigma(w^\top x)$ 直接解释为 $P(y=1 \mid x; w)$——给定特征 $x$ 和参数 $w$ 时正类的概率。

> 【理解】极大似然估计

**极大似然估计**（Maximum Likelihood Estimation, MLE）：模型含待估参数 $w$，已观测到样本 $D$，从 $w$ 的所有可能取值中**选使 $P(D \mid w)$ 最大的那个**作为估计值。

**直觉**：哪个参数能让"我们观测到的现象"看起来最不像意外，就选哪个。

**抛硬币例子**（不均匀硬币，正面概率 $\theta$）：

观测 $D = \{\text{正反反正正正}\}$，6 次独立投掷的联合概率：

$$P(D \mid \theta) = \theta \cdot (1-\theta) \cdot (1-\theta) \cdot \theta \cdot \theta \cdot \theta = \theta^4 (1 - \theta)^2$$

求 $\theta^*$ 使 $P(D \mid \theta)$ 最大 → 对 $\theta$ 求导令其为 0，解得 $\theta^* = 4/6$。

**LR 的连接**：把 N 个样本的联合概率（似然函数）写出来，对参数 $w$ 求极大 → 等价于对**负对数似然**求极小，这就是 LR 的损失函数（交叉熵）的来历，下一章详述。

> 【知道】对数函数

把乘法变加法，把指数变乘法——MLE 里把连乘的似然函数取 $\ln$ 就成了求和，求导/数值计算都方便。

**定义**：若 $a^x = N$（$a > 0, a \ne 1$），则 $x = \log_a N$。$\ln x = \log_e x$ 是自然对数。

**关键性质**：

| 公式 | 说明 |
|---|---|
| $\log_a(MN) = \log_a M + \log_a N$ | 乘 → 加（MLE 取 log 的核心动机） |
| $\log_a(M/N) = \log_a M - \log_a N$ | 除 → 减 |
| $\log_a(M^n) = n \log_a M$ | 幂 → 乘 |
| $\log_a 1 = 0$，$\log_a a = 1$ | 特殊值 |
| $\log_a x$ 单调递增（$a > 1$） | 取 log 不改变极值点位置 |

**为什么 MLE 取 log**：$\ln$ 单调递增，$\arg\max_w \prod_i P_i = \arg\max_w \sum_i \ln P_i$——极大值点不变，连乘 underflow 也避免了。
