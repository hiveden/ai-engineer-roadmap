问题从：

因此，将负的对数似然函数定义为 二分类 任务的loss函数，然后用梯度下降法求解最优解，得到模型的最佳参数。

Loss(L) = -∑ᵢ₌₁ⁿ（ yᵢlog(pᵢ) + (1−yᵢ)log(1−pᵢ) )

### 笔记

> 【知道】损失函数

**问题**：MSE 用在分类上，loss 曲面非凸（多个局部最小值），梯度下降会卡住。需要换损失。

**对数似然损失**（log loss / binary cross-entropy）——单样本：

$$\mathrm{loss}(h_w(x), y) = \begin{cases} -\log(h_w(x)) & y = 1 \\ -\log(1 - h_w(x)) & y = 0 \end{cases}$$

**直觉**：真实标签是 1 时，预测概率越接近 1 损失越小；接近 0 损失趋于 $+\infty$。预测错了被无情放大。

**合并写法**（用 $y \in \{0, 1\}$ 当开关）：

$$\mathrm{loss}(h_w(x), y) = -y \log(h_w(x)) - (1 - y) \log(1 - h_w(x))$$

**整体损失**（$m$ 个样本求和/平均）：

$$J(w) = -\frac{1}{m} \sum_{i=1}^{m} \left[ y_i \log(h_w(x_i)) + (1 - y_i) \log(1 - h_w(x_i)) \right]$$

**为什么这个损失好**：与 sigmoid 配合后 $J(w)$ 是凸函数，梯度下降稳定收敛到全局最优。

**代码 API**：

```python
from sklearn.metrics import log_loss
log_loss(y_true, y_pred_proba)  # y_pred_proba 是 predict_proba 的输出
```

## ━━━━━━━━ 讲解 ━━━━━━━━

### 业务问题

01 节把假设函数 $h_w(x) = \sigma(w^\top x + b)$ 立起来了，下一步：**怎么知道 $w$ 和 $b$ 学得好不好？** 必须有一个数字把"预测概率 vs 真实标签"的差距量化出来——这个数字就是损失函数 loss function（也叫代价函数 cost function）。损失越小，参数越好。

线性回归用均方误差 MSE，分类能不能直接照搬？答案是不能。这一节讲两件事：

1. **MSE 在分类上为什么失败** —— 反例打底
2. **对数似然损失**（log loss / binary cross-entropy / 二分类交叉熵）**怎么从极大似然估计 MLE 推出来** —— 正例链路

### 数学方法

#### 数学符号扫盲

| 符号 | 读法 | 直觉 | 例子 |
|---|---|---|---|
| $\prod$ | product | 连乘符号，类比 $\sum$ 是连加 | $\prod_{i=1}^{3} a_i = a_1 a_2 a_3$ |
| $\log$ | log | 自然对数 ln（ML 默认底数 $e$） | $\log e = 1$，$\log 1 = 0$ |
| $J(w)$ | J of w | 损失函数对参数 $w$ 的取值 | $J(w) = 0.42$ |
| $\nabla_w J$ | nabla J / J 对 w 的梯度 | 损失对所有 $w_i$ 偏导数组成的向量 | 梯度下降的方向 |
| $\arg\max_w f(w)$ | arg max | 让 $f$ 取最大值的那个 $w$ | $\arg\max_w (-w^2) = 0$ |

#### 反例先行：MSE 为什么不行

把 MSE 套到 LR 上：

$$J_\text{MSE}(w) = \frac{1}{m} \sum_{i=1}^{m} \big( \sigma(w^\top x_i + b) - y_i \big)^2$$

两个致命问题：

1. **曲面非凸**：$\sigma$ 套进平方里，$J_\text{MSE}$ 关于 $w$ 不是凸函数 convex function，存在多个局部最优 local optimum。梯度下降从不同初值出发可能停在不同的洼地，结果不稳定。
2. **梯度饱和**：$\sigma$ 在两端导数趋近 0，预测严重错时（比如真实 1，预测 0.01）梯度反而很小，参数走不动，学不出来。

> ASCII 示意（一维参数 $w$ 下 $J_\text{MSE}$ 形状）：
>
> ```
>  J
>  │   ╱╲       ╱╲
>  │  ╱  ╲     ╱  ╲     ← 多个局部最小值
>  │ ╱    ╲___╱    ╲
>  └──────────────────→ w
> ```

对照之下，对数似然损失配 sigmoid 后 $J(w)$ 是凸的，**全局只有一个最低点**，梯度下降必到全局最优 global optimum。

#### 正例链路：从 MLE 推对数似然

PPT slides 17-19 的推导路径是 **MLE → 似然函数 → 取对数 → 加负号 → 交叉熵**。一步一步过。

> 极大似然估计 maximum likelihood estimation, MLE 的概念定义 / 抛硬币例子在 [`../01-foundation/04-极大似然估计.md`](../01-foundation/04-极大似然估计.md) 是主场。这里只用一句话的结论：**给定一组观测数据，挑参数让"这组数据出现的联合概率"最大**。

##### 步骤 1 · 单样本"分类正确"概率统一写法

样本 $x$ 真实标签 $y \in \{0, 1\}$，模型给"它是 1 类"的概率 $p = h_w(x)$。

- 若 $y = 1$，分类正确的概率 $= p$
- 若 $y = 0$，分类正确的概率 $= 1 - p$

两种情况合并成一条式子（用 $y$ 当开关）：

$$P(y \mid x; w) = p^{y} \cdot (1 - p)^{1 - y}$$

代入验证：$y = 1$ 时 $p^1 (1-p)^0 = p$ ✓；$y = 0$ 时 $p^0 (1-p)^1 = 1 - p$ ✓。

##### 步骤 2 · n 个样本的联合概率（似然函数 likelihood）

样本独立同分布 i.i.d.，联合概率就是连乘：

$$L(w) = \prod_{i=1}^{m} p_i^{y_i} (1 - p_i)^{1 - y_i}, \quad p_i = h_w(x_i)$$

这就是**似然函数 likelihood function**。MLE 的目标：

$$\hat{w} = \arg\max_w L(w)$$

##### 步骤 3 · 取对数 → 对数似然 log-likelihood

直接对 $L(w)$ 求导难（连乘求导用乘法法则，$m$ 个因子炸开成 $m$ 项再乘一堆其他因子）。两个工程理由：

1. **连乘转连加**：$\log(ab) = \log a + \log b$，求导按线性分项处理
2. **数值稳定**：$m$ 个 $(0,1)$ 概率连乘很快下溢 underflow 到 0；取对数后变成负数加和，浮点稳

> 对数函数性质（单调、$\log 1 = 0$、防 underflow）在 [`../01-foundation/05-对数函数.md`](../01-foundation/05-对数函数.md) 是主场。

$\log$ 单调递增，"让 $L$ 最大"等价于"让 $\log L$ 最大"。对两边取对数：

$$\log L(w) = \sum_{i=1}^{m} \big[ y_i \log p_i + (1 - y_i) \log(1 - p_i) \big]$$

##### 步骤 4 · 加负号 → 极大变极小（"倒反天罡"）

PPT 讲师管这步叫"倒反天罡"——指 max 翻成 min。梯度下降只会求最小值，所以把"最大化对数似然"翻成"最小化负对数似然 negative log-likelihood, NLL"，再除以样本数 $m$ 做平均：

$$J(w) = -\frac{1}{m} \log L(w) = -\frac{1}{m} \sum_{i=1}^{m} \big[ y_i \log h_w(x_i) + (1 - y_i) \log(1 - h_w(x_i)) \big]$$

这就是**二分类交叉熵 binary cross-entropy / 对数似然损失 log loss**——LR 的标准损失函数，sklearn `log_loss` 算的就是它。

#### 实算：底稿 PPT 例子

5 个样本，真实标签和预测概率如下（$\log$ 取自然对数，0.68 是底稿 typo，按底稿"保留原文"，实算时用 0.6 对齐 $y=0$ 列）：

| i | $y_i$ | $p_i = h_w(x_i)$ | 单样本 loss $-[y_i \log p_i + (1-y_i)\log(1-p_i)]$ |
|---|---|---|---|
| 1 | 1 | 0.40 | $-\log 0.40 \approx 0.916$ |
| 2 | 0 | 0.60 | $-\log(1-0.60) = -\log 0.40 \approx 0.916$ |
| 3 | 1 | 0.41 | $-\log 0.41 \approx 0.892$ |
| 4 | 0 | 0.55 | $-\log 0.45 \approx 0.799$ |
| 5 | 1 | 0.92 | $-\log 0.92 \approx 0.083$ |

$$J = \frac{1}{5}(0.916 + 0.916 + 0.892 + 0.799 + 0.083) \approx 0.721$$

观察：样本 5 是"易例"（预测 0.92，真实 1），loss 只有 0.083；前四个"难例"loss 都在 0.8 以上——**预测错得越离谱，惩罚越重**，这正是对数函数 $-\log p$ 在 $p \to 0$ 时趋于 $+\infty$ 的效果。

#### 梯度下降找极小（点到为止）

有了 $J(w)$，沿负梯度方向迭代更新参数：

$$w \leftarrow w - \eta \cdot \nabla_w J(w)$$

$\eta$ 是学习率 learning rate。$J(w)$ 是凸函数，梯度下降必到全局最优。

> 梯度下降的算法细节、学习率调度、动量、Adam 等优化器在后续优化器章节是主场，本节只点到链路：**有损失 → 求梯度 → 更新参数**。

### 代码落地

```python
from sklearn.metrics import log_loss

y_true = [1, 0, 1, 0, 1]
y_proba = [[0.6, 0.4], [0.4, 0.6], [0.59, 0.41], [0.45, 0.55], [0.08, 0.92]]
log_loss(y_true, y_proba)   # ≈ 0.721，和上面手算一致
```

`log_loss` 的输入是 `predict_proba` 的输出（每行 `[P(y=0), P(y=1)]`），不是 `predict` 的 0/1 标签。

**这一节的关键启示**：MLE 取对数加负号变损失，配 sigmoid 凸函数，梯度下降到全局最优。
