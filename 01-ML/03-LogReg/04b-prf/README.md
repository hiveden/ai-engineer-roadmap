也叫查全率，指的是预测为真正例样本占所有真实正例样本的比重
例如：恶性肿瘤当做正例样本，则我们想知道模型是否能把所有的恶性肿瘤患者都预测出来。

避免漏报

> Slide 41 · 分类评估方法 – F1-score

分类评估方法 – 精确率、召回率、F1

F1-score

若对模型的精度、召回率都有要求，希望知道模型在这两个评估方向的综合预测能力？
计算方法：精准率和召回率的调和平均：F1 = 2 * Precision * Recall / (Precision + Recall)

即：正例有没有找准，找全？？

F1-score - 举个例子

已知：样本集10样本，有 6 个恶性肿瘤样本，4 个良性肿瘤样本，我们假设恶性肿瘤为正例

- 伪反例 FN 为：3
- 伪正例 FP 为：0
- 真反例 TN：4
- 精准度：100%
- 召回率：50%
- F1-score 67%

模型B：预测对了 6 个恶性肿瘤样本，1个良性肿瘤样本
- 真正例 TP 为：6
- 伪反例 FN 为：0
- 伪正例 FP 为：3
- 真反例 TN：1
- 精准度：67%

- F1-score：80%

公式：F1 = 2 * Precision * Recall / (Precision + Recall)

**适用场景**：误报代价高的业务——垃圾邮件过滤（误杀正常邮件）、推送广告（打扰用户）、刑事定罪（宁可放过不可错杀）。

> 【掌握】Recall（召回率）

**召回率**（Recall，又称查全率、TPR）：所有真实正例样本中，被模型成功预测出来的比例。回答"**该抓的抓到了多少**"。

$$
Recall = \frac{TP}{TP + FN}
$$

**直觉**：分母是"真实的全部正例"，分子是"被抓到的"。FN（伪反例 / 漏报）越多，召回率越低。

**例**：6 个恶性肿瘤为正例。
- 模型 A：只识别出 3 个恶性 → $R = 3/(3+3) = 50\%$（漏掉 3 个）
- 模型 B：6 个恶性全部识别出 → $R = 6/(6+0) = 100\%$

**适用场景**：漏报代价高的业务——癌症 / 疾病筛查、风控反欺诈、安防监控（宁可错杀不可放过）。

**Precision vs Recall 取舍**：调高分类阈值 → 精确率↑召回率↓；调低阈值 → 召回率↑精确率↓。鱼和熊掌不可兼得。

> 【掌握】F1-score

**F1-score**：精确率与召回率的**调和平均**，单一标量综合衡量模型在两个方向的表现。回答"**两头都要时该选谁**"。

$$
F1 = \frac{2 \cdot P \cdot R}{P + R} = \frac{2 \cdot TP}{2 \cdot TP + FP + FN}
$$

**直觉**：调和平均对小值敏感——P 和 R 任何一个塌了，F1 都会被拉下来；只有两者都高，F1 才高。比算术平均更"严格"。

**例**（接前例）：
- 模型 A：$P=100\%$，$R=50\%$ → $F1 = (2 \times 3) / (2 \times 3 + 3 + 0) \approx 67\%$
- 模型 B：$P=67\%$，$R=100\%$ → $F1 = (2 \times 6) / (2 \times 6 + 0 + 3) \approx 80\%$

模型 B 综合更优。

**适用场景**：类别不平衡 + 精确率召回率都要兼顾时（如医疗诊断、检索系统）。要偏向某一侧用 $F_\beta$（$\beta>1$ 偏召回，$\beta<1$ 偏精确）。

> 模型A：预测对了 3 个恶性肿瘤样本，4 个良性肿瘤样本
> 请计算：TP、FN、FP、TN
> 模型B：预测对了 6 个恶性肿瘤样本，1个良性肿瘤样本
> 请计算：TP、FN、FP、TN
> 真正例 TP 为：6
> 伪反例 FN 为：0
> 伪正例 FP 为：3
> 真反例 TN：1
> 精准度： 67%
> 召回率： 100%
> F1-score：80%
> 真正例 TP 为：3
> 伪反例 FN 为：3
> 伪正例 FP 为：0
> 真反例 TN：4
> 精准度：100%
> 召回率：50%
> F1-score 67%
>
> 公式：$F_1 = \frac{2 \cdot Precision \cdot Recall}{Precision + Recall}$（图片公式）

> 【代码】F1-score 程序计算（PPT slide 42）

```python
from sklearn.metrics import f1_score

def dm03_F1():
    # 样本集中共有6个恶性肿瘤样本, 4个良性肿瘤样本
    y_true = ["恶性", "恶性", "恶性", "恶性", "恶性", "恶性", "良性", "良性", "良性", "良性"]
    # 1. 模型 A: 预测对了3个恶性肿瘤样本, 4个良性肿瘤样本
    y_pred = ["恶性", "恶性", "恶性", "良性", "良性", "良性", "良性", "良性", "良性", "良性"]
    result = f1_score(y_true, y_pred, pos_label="恶性")
    print("模型Af1-score:", result)
    # 2. 模型 B: 预测对了6个恶性肿瘤样本, 1个良性肿瘤样本
    y_pred = ["恶性", "恶性", "恶性", "恶性", "恶性", "恶性", "恶性", "恶性", "恶性", "良性"]
    result = f1_score(y_true, y_pred, pos_label="恶性")
    print("模型Bf1-score::", result)
```
