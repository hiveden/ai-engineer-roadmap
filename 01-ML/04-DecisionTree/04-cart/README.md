> 【知道】CART 树简介

**CART = Classification And Regression Tree**（1984）。**始终是二叉树**，分类回归通吃：

- **分类树**：基尼指数最小化作准则
- **回归树**：平方误差最小化（详见第 6 章）

> 【理解】基尼指数计算方法

**基尼值**（衡量集合纯度）：

$$\text{Gini}(D) = 1 - \sum_{k=1}^{K} \left(\frac{|C_k|}{|D|}\right)^2$$

**基尼指数**（特征 $A$ 把 $D$ 切成 $D_1, D_2$ 的加权基尼值）：

$$\text{Gini\_index}(D, A) = \frac{|D_1|}{|D|} \text{Gini}(D_1) + \frac{|D_2|}{|D|} \text{Gini}(D_2)$$

**选择准则对比**：

| 算法 | 准则 | 方向 |
|---|---|---|
| ID3 | 信息增益 | 越**大**越好 |
| C4.5 | 信息增益率 | 越**大**越好 |
| CART | 基尼指数 | 越**小**越好 |

**为什么 Gini 比熵快**：无 $\log$ 计算，工程上更高效；两者效果接近。

> 【理解】基尼指数计算举例

**贷款数据集**（10 样本，3 特征：是否有房 / 婚姻状况 / 年收入）。

**1. 是否有房**（二值，天然二分）：

- 有房（3 样本，全 no）：$\text{Gini} = 1 - 0^2 - 1^2 = 0$
- 无房（7 样本，4 no / 3 yes）：$\text{Gini} = 1 - (3/7)^2 - (4/7)^2 = 0.4898$
- 加权：$\frac{3}{10} \times 0 + \frac{7}{10} \times 0.4898 = 0.343$

**2. 婚姻状况**（三值，需枚举所有二分组合）：

- $\{married\} \mid \{single, divorced\}$：Gini = **0.3**
- $\{single\} \mid \{married, divorced\}$：Gini = 0.367
- $\{divorced\} \mid \{single, married\}$：Gini = 0.4
- 选最小的 0.3，分裂点 = $\{married\}$ vs 其他

**3. 年收入**（连续，排序后取相邻中点作候选）：

- 候选点：65、72.5、80、87.5、92.5、97.5、110、122.5、172.5
- 例：以 65 切，Gini = $\frac{1}{10} \times 0 + \frac{9}{10} \times [1 - (6/9)^2 - (3/9)^2] = 0.4$
- 全扫一遍找最小：97.5 处取得 **0.3**

**最终选择**：三特征最小 Gini 分别 0.343 / 0.3 / 0.3 → 0.3 平局，随机选（如婚姻状况）作根节点。递归切剩余子集，直到叶子纯净。

> 【知道】比较

| 算法 | 年份 | 准则 | 树形 | 特点 |
|---|---|---|---|---|
| ID3 | 1975 | 信息增益 | 多叉 | 仅离散；偏向多值特征 |
| C4.5 | 1993 | 信息增益率 | 多叉 | 修正多值偏好；支持连续 + 缺失；不适合大数据 |
| CART | 1984 | 基尼指数 / 平方误差 | **二叉** | 分类 + 回归；计算量小；工业首选 |

**sklearn 默认实现的就是 CART**（`DecisionTreeClassifier` / `DecisionTreeRegressor`）。
