#### Slide 69 · CART 回归决策树 · 思路

**已知**：数据集只有 1 个特征 x，共 10 个样本，特征值 1~10。目标值为 y。

```
思路示意：用一刀把样本切两半，每边取 y 的均值作预测

y                    │
9                ●●● │
                ●    │
              ●      │       ← 右边均值 c₂ = ȳ_R
              ───────│───────
                     │  ●
                     │ ●
                     │●         ← 左边均值 c₁ = ȳ_L
              ●●●    │
    ─────────────────┼─────── x
                  切分点 s
```

**思路 · 训练过程**：

1. 先将样本切两半
2. 计算叶节点的子样本均值，将均值作为预测值

**推理过程**：输入测试样本（房屋面积）

- 但凡其面积 < 80，都预测售价为 100W，无论实际面积是 70 还是 7 或者 0.7 平方米
- 但凡面积 > 80，都预测售价为 200W，无论实际面积是 90 还是 900 或 9000 平米

**问题**：如何选合适特征和切分标准？

#### Slide 70 · 与 CART 分类树的比较

```
CART 分类树                 CART 回归树
─────────────              ─────────────
输出：离散类别              输出：连续值
准则：基尼指数              准则：平方损失
叶子：多数类                叶子：样本均值
   ┌──┴──┐                    ┌──┴──┐
  类A   类B                   c₁    c₂
  (众数)(众数)                (均值)(均值)
```

CART 回归树和 CART 分类树的不同之处在于：

- CART 分类树预测输出的是一个**离散值**，CART 回归树预测输出的是一个**连续值**
- CART 分类树使用**基尼指数**作为划分、构建树的依据，CART 回归树使用**平方损失**
- 分类树使用叶子节点**多数类别**作为预测类别，回归树则采用叶子节点里**均值**作为预测输出

```
平方损失公式

  Loss(R) = Σ (yᵢ - c)²       c = mean(y in R)
            i∈R

切分总损失 = Loss(R₁) + Loss(R₂)
           └─ 左边 ─┘  └─ 右边 ─┘
                     ↓
              枚举所有切分点 s
              取总损失最小者
```

CART 回归树的**平方损失**（类似方差，但方差要取平均 np.var * N）越**小**越好。

```
例：以切分点 s 把数据分两段，每段算均值 → 算平方损失 → 求和

           y
              │  ┌──── c₂ ────
              │  │   ●  ●
       ── c₁ ─┤  │ ●
       ●  ●   │  │
              │  │
              ┴──┴────────── x
                 ↑
                 s
       Loss = Σ_L(y - c₁)² + Σ_R(y - c₂)²
```

**已知**：数据集只有 1 个特征 x，共 10 个样本，特征值 1~10。目标值为 y。

**分析**：因只有 1 个特征，所以只需选择该特征的最优划分点，并不需要计算其他特征。

7. 假设在生成 3 个区域之后停止划分，以上就是最终回归树。

每一个叶子节点的输出为：挂在该节点上的所有样本均值。
回归的目标是：子节点的均值。

**CART 回归树构建过程小结**（原则上还可以划分更细，比如右枝是否大于 8.5）：

1. 选择一个特征，将该特征的值进行排序，取相邻点计算均值作为待划分点
2. 根据所有划分点，将数据集分成两部分：R1、R2
3. R1 和 R2 两部分的平方损失相加作为该切分点平方损失
4. 取最小的平方损失的划分点，作为当前特征的划分点（**预测值就是成员的均值**）
5. 以此计算其他特征的最优划分点、以及该划分点对应的损失值
6. 在所有的特征的划分点中，选择出最小平方损失的划分点，作为当前树的分裂点

#### Slide 77 · 过拟合：树的切分太多次

```
深树：每个叶子只剩 1 个样本（10 叶子）

         x ≤ 5.5
         ┌─┴─┐
        ...   ...
       /   \  /   \

     /\  /\  /\  /\  /\
    ●● ●● ●● ●● ●● ●● ●● ●● ●● ●●
    每片叶子 = 1 样本 → 预测值 = 该样本 y
```

训练时，树每个分支只有 1 个样本，回归的预测值（样本均值），只能是它的标签值。
测试时，表现拉胯。

- 决策树模型过于复杂
- 精准穿过了每个数据点

#### Slide 78 · 案例代码

```python
import numpy as np
import pandas as pd
from sklearn.tree import DecisionTreeRegressor  # 回归决策树
from sklearn.linear_model import LinearRegression
import matplotlib.pyplot as plt

def dm01_回归分类():
    # 1 准备数据
    x = np.array(list(range(1,11))).reshape(-1, 1)
    y = np.array([5.56, 5.70, 5.91, 6.40, 6.80, 7.05, 8.90, 8.70, 9.00, 9.05])
    print('x-->\n', x)
    print('y-->\n', y)

    # 2 实例化模型 模型训练
    model1 = DecisionTreeRegressor(max_depth=1)
    model2 = DecisionTreeRegressor(max_depth=3)
    model3 = LinearRegression()
    model1.fit(x, y)
    model2.fit(x, y)
    model3.fit(x, y)

    # 3 模型预测 # 等差数组-按照间隔
    x_test = np.arange(0.0, 10.0, 0.01).reshape(-1, 1)
    y_pre1 = model1.predict(x_test)
    y_pre2 = model2.predict(x_test)
    y_pre3 = model3.predict(x_test)
    print(y_pre1.shape, y_pre2.shape, y_pre3.shape)

    # 4 结果可视化
    plt.figure(figsize=(10, 6), dpi=100)
    plt.scatter(x, y, label='data')
    plt.plot(x_test, y_pre1, label='max_depth=1')   # 深度 1 层
    plt.plot(x_test, y_pre2, label='max_depth=3')   # 深度 3 层
    plt.plot(x_test, y_pre3, label='linear')
    plt.xlabel('data')
    plt.ylabel('target')
    plt.title('DecisionTreeRegressor')
    plt.legend()
    plt.show()
```

```
图例：
  ──── linear            一条直线
  ──── max_depth=1       1 刀（s=6.5）→ 2 段阶梯
  ──── max_depth=3       多刀 → 多段细阶梯
```

### 笔记

> 06 · 回归决策树

**学习目标**：

1. 了解回归决策树的构建原理
2. 能用回归决策树 API 解决问题

> 【了解】回归决策树构建原理

**CART 回归树 vs CART 分类树**：

| 维度 | 分类树 | 回归树 |
|---|---|---|
| 输出 | 离散类别 | 连续值 |
| 划分准则 | 基尼指数 | 平方损失 |
| 叶子预测 | 多数类 | 样本均值 |

**损失函数**：

$$\text{Loss}(y, f(x)) = (f(x) - y)^2$$

**切分点搜索**（单特征演示）：

数据 $x = [1..10]$，$y = [5.56, 5.7, 5.91, 6.4, 6.8, 7.05, 8.9, 8.7, 9, 9.05]$。

**步骤**：

1. **特征值排序**，取相邻中点作候选切分点：$\{1.5, 2.5, ..., 9.5\}$
2. **每个候选点算平方损失**：把样本切成 $R_1$（左）、$R_2$（右），各自算均值 $c_1, c_2$，损失 = $\sum_{x \in R_1}(y - c_1)^2 + \sum_{x \in R_2}(y - c_2)^2$
3. **以 1.5 切**：$R_1 = \{5.56\}$（损失 0），$R_2$ 9 个样本均值 7.50，损失 15.72
4. **全扫**得到 $m(s)$ 序列：$[15.72, 12.07, 8.36, 5.78, 3.91, \mathbf{1.93}, 8.01, 11.73, 15.74]$
5. **最小在 $s=6.5$** → 第一刀切在 6.5
6. **递归**：左子树（前 6 样本）继续切，最优在 $s=3.5$
7. 满足停止条件后，每个叶子输出 = 该叶子内样本 $y$ 均值

**预测示例**：1/2/3 号样本均落在叶子 $\{1,2,3\}$，预测值 = $(5.56+5.7+5.91)/3 = 5.72$。

**完整构建流程（多特征）**：

1. 对每个特征，将值排序，取相邻中点作候选切分点
2. 按切分点把数据集切成 $R_1, R_2$
3. 算两部分平方损失之和作为该切分点的损失
4. 该特征下损失最小的切分点 = 该特征的最优切分点
5. 所有特征的最优切分点中，挑全局最小损失者作当前节点分裂点
6. 递归子集

> 【实践】回归决策树实践

**API**：

```python
from sklearn.tree import DecisionTreeRegressor
reg = DecisionTreeRegressor(max_depth=5)
reg.fit(X_train, y_train)
reg.predict(X_test)
```

**vs 线性回归对比实验**（典型可视化）：

```python
import numpy as np
from sklearn.linear_model import LinearRegression
from sklearn.tree import DecisionTreeRegressor

X = np.arange(1, 11).reshape(-1, 1)
y = np.array([5.56, 5.7, 5.91, 6.4, 6.8, 7.05, 8.9, 8.7, 9, 9.05])

lr = LinearRegression().fit(X, y)
dt = DecisionTreeRegressor(max_depth=3).fit(X, y)

X_test = np.linspace(0, 10, 1000).reshape(-1, 1)
# lr.predict 是直线；dt.predict 是阶梯曲线
```

**观察结论**：

1. 线性回归 → **直线**
2. 决策树 → **阶梯曲线**（每个叶子一段常数）
3. 树**拟合能力强**，但极易过拟合 → 必须配合 `max_depth` / 剪枝 / 集成（随机森林、GBDT）
