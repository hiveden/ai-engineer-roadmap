- **PPT 来源**: 主 PPT Slide 43
- **PPT 原文摘要**: "学习率：作用于模型权重，用于增大或缩小每个权重的贡献，从而影响样本权重的更新快慢，范围 0~+∞"
- **README 当前状态**: 代码里设了 `learning_rate=0.1` 但未解释作用
- **建议补入位置**: README 第 91 行代码块之后
- **建议补入文本**（≤80 字）:
  ```

### 缺漏 5.6: 引入泰勒展开的动机（loss 含正则难求梯度）
- **PPT 来源**: 补充材料 Slide 18

- **README 当前状态**: README 直接列出泰勒二阶展开，没说"为什么要展开"
- **建议补入位置**: README 第 52 行"**泰勒二阶展开**"标题之前
- **建议补入文本**（≤80 字）:
  ```
  **为何要展开**：损失加正则后形式复杂，难以直接对 $f_t$ 求负梯度优化。泰勒二阶近似把目标变成关于 $f_t$ 的二次函数，可直接闭式求解。
  ```
