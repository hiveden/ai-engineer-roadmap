> 【知道】集成学习是什么

**Ensemble Learning**：组合多个**弱学习器（weak learner / base learner）** 形成精度更高的强学习器。每个弱学习器独立训练、独立预测，最后把预测结果合并。

工程类比：微服务集群 + 网关聚合。单个服务能力有限，多副本投票/加权得到更稳的输出。

俗语版：三个臭皮匠，赛过诸葛亮。

> 【了解】集成学习分类

主流两类：

| 流派 | 核心 | 代表 |
|---|---|---|
| **Bagging**（Bootstrap Aggregating）| 并行 + 有放回采样 + 平权投票 | Random Forest |
| **Boosting** | 串行 + 关注前一轮错样本 + 加权投票 | AdaBoost / GBDT / XGBoost |

> 【理解】bagging 集成

**思想**：有放回抽样（bootstrap）产生 N 个不同训练集 → 各自训练弱学习器 → 平权投票（分类）/ 平均（回归）。

**关键点**：

- 弱学习器之间**独立**，训练可并行
- 抽样制造**差异性**，差异越大集成增益越高
- 投票权重相等（所以叫 "平权"）

> 【理解】boosting 集成

**思想**：串行训练，每轮**重点关注前一轮分错的样本**，最后加权投票。

**流程**：

1. 初始训练集 → 训练基学习器 1
2. 根据 1 的表现调整样本权重（错样本权重 ↑）
3. 在新分布上训练基学习器 2
4. 重复 T 轮
5. T 个学习器**加权**结合（误差小的权重大）

**核心**：每加一个弱学习器，整体能力提升。

> Bagging vs Boosting

| 维度 | Bagging | Boosting |
|---|---|---|
| **数据** | 有放回采样的子集 | 全量数据 + 样本权重 |
| **投票** | 平权 | 加权（按学习器误差） |
| **顺序** | 并行（无依赖） | 串行（后一个依赖前一个） |
| **侧重** | 降方差 | 降偏差 |
