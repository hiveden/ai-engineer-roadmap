```python
# AdaBoost 实战葡萄酒数据
import pandas as pd
from sklearn.preprocessing import LabelEncoder
from sklearn.model_selection import train_test_split
from sklearn.tree import DecisionTreeClassifier
from sklearn.ensemble import AdaBoostClassifier  # 集成学习
from sklearn.metrics import accuracy_score

def dm01_adaboost():
    # 1 读数据到内存 df_wine
    df_wine = pd.read_csv('./data/wine0501.csv')

    # 2 特征处理
    # 2-2 Adaboost 一般做二分类，去掉一类 (1,2,3)
    df_wine = df_wine[df_wine['Class label'] != 1]

    # 2-3 准备特征值和目标值  Alcohol 酒精含量  Hue 颜色
    x = df_wine[['Alcohol', 'Hue']].values
    y = df_wine['Class label']

    # 2-4 类别转化 y (2,3) => (0,1)
    y = LabelEncoder().fit_transform(y)

    # 2-5 划分数据
    X_train, X_test, y_train, y_test = train_test_split(x, y, random_state=22, test_size=0.2)
    print(X_train.shape, X_test.shape, y_train.shape, y_test.shape)
```

```
#   2-4 类别转化 (2,3)=>(0,1)  注意 (0,1) 标签与 (-1,1) 标签均可接受
#   2-5 划分数据
```

### 笔记

> 【实践】AdaBoost 葡萄酒数据

```python
from sklearn.ensemble import AdaBoostClassifier
from sklearn.tree import DecisionTreeClassifier

ada = AdaBoostClassifier(
    estimator=DecisionTreeClassifier(max_depth=1),  # 决策桩；sklearn <1.2 用 base_estimator=
    n_estimators=500,
    learning_rate=0.1,
)
ada.fit(X_train, y_train)
```

**`learning_rate`**：缩放每轮模型权重 $\alpha_t$ 的贡献，间接影响样本权重更新速度。范围 $(0, +\infty)$，越小需要越多轮 `n_estimators` 配合。

**两条实战提醒**：
- 标签编码 (0,1) 或 (-1,1) sklearn 均可接受
- 基学习器若过强（深树），集成反而可能更差，决策桩 `max_depth=1` 是常用起点

**结论**：AdaBoost 比单决策树过拟合更轻、测试集略胜，与 bagging 准确率相近。

## ━━━━━━━━ 讲解 ━━━━━━━━

### 数据概览

葡萄酒数据集（`wine0501.csv`）：原始有 3 类（Class label = 1, 2, 3）。AdaBoost 一般用于二分类，PPT 中去掉类别 1，保留类别 2 和 3，再用 `LabelEncoder` 转为 (0, 1)。

使用的两个特征：

| 特征 | 含义 |
|---|---|
| `Alcohol` | 酒精含量 |
| `Hue` | 色泽（颜色深浅） |

**为什么只用这两个**：教学演示目的——两个特征可以画二维决策边界，容易可视化对比单棵树 vs AdaBoost。

### 完整 pipeline 代码

```python
import pandas as pd
from sklearn.preprocessing import LabelEncoder
from sklearn.model_selection import train_test_split
from sklearn.tree import DecisionTreeClassifier
from sklearn.ensemble import AdaBoostClassifier
from sklearn.metrics import accuracy_score

# ── 1. 读数据 ──────────────────────────────────────────────
df_wine = pd.read_csv('./data/wine0501.csv')

# ── 2. 特征处理 ────────────────────────────────────────────
# 2-1 保留类别 2 和 3（去掉类别 1）
df_wine = df_wine[df_wine['Class label'] != 1]

# 2-2 选特征 + 标签
X = df_wine[['Alcohol', 'Hue']].values
y = df_wine['Class label']

# 2-3 标签转换 (2,3) → (0,1)
y = LabelEncoder().fit_transform(y)

# 2-4 划分训练集 / 测试集
X_train, X_test, y_train, y_test = train_test_split(
    X, y, random_state=22, test_size=0.2
)

# ── 3. 单棵决策树（基线对比）──────────────────────────────
single_tree = DecisionTreeClassifier(
    criterion='entropy',
    max_depth=1,       # 决策桩：只问一个问题
    random_state=0
)
single_tree.fit(X_train, y_train)
score_tree = single_tree.score(X_test, y_test)
print(f'单棵决策树准确率: {score_tree:.4f}')

# ── 4. AdaBoost（500 棵决策桩）────────────────────────────
# ⚠ sklearn >= 1.2 将 base_estimator 重命名为 estimator
ada = AdaBoostClassifier(
    estimator=DecisionTreeClassifier(criterion='entropy', max_depth=1, random_state=0),
    n_estimators=500,
    learning_rate=0.1,
    random_state=0
)
ada.fit(X_train, y_train)
score_ada = ada.score(X_test, y_test)
print(f'AdaBoost 准确率:  {score_ada:.4f}')
```

### 关键参数详解

#### `n_estimators`：弱分类器数量

```
n_estimators 对准确率的影响（示意）：

准确率
100%│
    │             ──────── (趋于平稳)
 90%│       ────╯
    │  ──╯
 80%│╯
    └──────────────────── n_estimators
       10   50  200  500
```

- 太少：欠拟合，强分类器组合不充分
- 适中：准确率快速提升
- 过多：提升边际效益递减，且基学习器若较强时可能过拟合

PPT 选 500 是一个常见的偏大值，保证充分收敛。

#### `learning_rate`：学习率

学习率 $\eta$ 修改每轮弱分类器的贡献：

$$H(x) = \mathrm{sign}\left(\sum_{t=1}^{m} \eta \cdot \alpha_t h_t(x)\right)$$

- `learning_rate = 1.0`：不缩放，标准 AdaBoost
- `learning_rate = 0.1`：每轮贡献缩小到 10%，需要更多 `n_estimators` 才够
- `learning_rate` 与 `n_estimators` 成反比权衡：lr 小 → 需要更多轮次

**调参直觉**：先固定 `learning_rate=0.1` 调大 `n_estimators`；或者用 GridSearchCV 同时搜索两者。

#### `estimator`（原 `base_estimator`）：基学习器

```python
# sklearn >= 1.2（新 API）
AdaBoostClassifier(estimator=DecisionTreeClassifier(max_depth=1))

# sklearn < 1.2（旧 API，仍兼容但有 DeprecationWarning）
AdaBoostClassifier(base_estimator=DecisionTreeClassifier(max_depth=1))
```

**常见选择**：

| 基学习器 | 说明 |
|---|---|
| `DecisionTreeClassifier(max_depth=1)` | 决策桩，最弱，最常用，避免过拟合 |
| `DecisionTreeClassifier(max_depth=2)` | 稍强，准确率上限更高但更易过拟合 |
| 默认（不传）| sklearn 默认 `max_depth=1` 的决策树 |

> PPT Slide 44 说"如果基学习器太强，集成学习不一定能得到更好结果"——原因：强基学习器本身偏差已低，AdaBoost 的"纠错叠加"优势体现不出来，反而可能因为方差增大而变差。

### 单棵树 vs AdaBoost 对比

```
测试集效果对比（示意，实际数值因数据集而异）：

单棵决策桩（max_depth=1）: ~80%
AdaBoost 500 棵决策桩:    ~85-90%

差异来源：
  - 单棵树：只问一个问题（Alcohol > 阈值？），线性边界
  - AdaBoost：500 个问题的加权组合，决策边界更复杂、更贴合数据
```

**过拟合对比**：单棵决策桩由于过于简单，**欠拟合**（训练集准确率也不高）；AdaBoost 通过叠加多个弱分类器逼近真实边界，训练集误差趋近 0，但测试集因正则化效果通常也更好。

### sklearn 版本兼容提示

```python
# 兼容写法：自动适配新旧 sklearn
import sklearn
from sklearn.ensemble import AdaBoostClassifier
from sklearn.tree import DecisionTreeClassifier

stump = DecisionTreeClassifier(max_depth=1, random_state=0)

# sklearn < 1.2
if sklearn.__version__ < '1.2':
    ada = AdaBoostClassifier(base_estimator=stump, n_estimators=500, learning_rate=0.1)
else:
    ada = AdaBoostClassifier(estimator=stump, n_estimators=500, learning_rate=0.1)
```

或者直接升级 sklearn 到 >=1.2 并统一使用 `estimator=`。

### 一句话钉板

**`AdaBoostClassifier(estimator=DecisionTreeClassifier(max_depth=1), n_estimators=500, learning_rate=0.1)`：决策桩做基，lr 控节奏，500 轮加权投票——比单棵树准确率更高、过拟合更轻。**
