### PPT

> Slide 54 · 梯度提升树（GBDT）

**GBDT（Gradient Boosting Decision Tree）**：利用梯度下降的近似方法，利用损失函数的负梯度作为提升树算法中的残差近似值。

**假设**：
- 前一轮迭代得到的强学习器是：$f_{i-1}(x)$
- 损失函数为平方损失：$L(y, f_{i-1}(x))$
- 本轮迭代的目标是找到一个弱学习器：输出预测值 $h_i(x)$
- 让本轮的损失最小化：$\text{Loss} = L(y, f_{i-1}(x) + h_i(x))$
- 则要拟合单个样本的负梯度为：

$$r_i = -\left[\frac{\partial L(y, f(x))}{\partial f(x)}\right]_{f = f_{i-1}(x)}$$

即：对于平方损失，**GBDT 拟合的负梯度就是残差**。

```
对于任一样本（负梯度 = 残差，平方损失情形）：
  r_i = y_i - f_{i-1}(x_i)

对于任一模型参数：
  (梯度下降方向 = 负梯度方向)
```

> Slide 55 · 梯度提升树——基础学习器（初始化）

**对于整个训练集：给出标签的均值，作为基准值。**

执行标准：当模型预测值为何值时，会使得第一个弱学习器在所有训练样本上预测的平方误差最小，即：求损失函数对 $f(x_i)$ 的导数，并令导数为 0。

$$L(y, f(x)) = \frac{1}{2}(y - f(x))^2 \Rightarrow \text{求平方误差最小}$$

$$\frac{\partial}{\partial f(x_i)} \sum_{i} L(y_i, f(x_i)) = 0$$

$$\Rightarrow f(x) = \frac{1}{n}\sum_{i=1}^{n} y_i = \bar{y}$$

从公式可得：初始化树的输出值为目标值的均值时，可使得第一个弱学习器的平方误差最小。

```
商品总价 82 元类比：
  初始猜测 = 均值（50+20+10 周围某个值）
  作为一个基准值，不能差太多！！
```

> Slide 62 · 梯度提升树的构建流程

1. 初始化弱学习器（目标值的均值作为预测值）
2. 迭代构建学习器，每一个学习器拟合上一个学习器的负梯度
3. 直到达到指定的学习器个数
4. 当输入未知样本时，将所有弱学习器的输出结果组合起来作为强学习器的输出

### 笔记

> 【理解】梯度提升树

**核心改进**：不直接拟合残差，而是拟合**损失函数对当前模型的负梯度**（最速下降近似）。

**形式化**：

- 前一轮强学习器：$f_{i-1}(x)$
- 损失函数：$L(y, f_{i-1}(x))$
- 本轮目标：找弱学习器 $h_i(x)$ 使 $L(y, f_{i-1}(x) + h_i(x))$ 最小

**负梯度**作为本轮拟合目标：

$$r_i = -\left[\frac{\partial L(y, f(x))}{\partial f(x)}\right]_{f = f_{i-1}}$$

**平方损失情形**：$L = \frac{1}{2}(y - f(x))^2$，则 $-\frac{\partial L}{\partial f(x)} = y - f(x)$，即负梯度 $r_i = y_i - f_{i-1}(x_i) =$ 残差。所以平方损失下 GBDT 退化为提升树。

**分类情形**：损失换成 logloss，拟合的就是 logloss 的负梯度（不再是残差）。

## ━━━━━━━━ 讲解 ━━━━━━━━

### 业务问题：残差只适合平方损失

上一章提升树的残差很直觉——但它暗藏了一个假设：**损失函数是平方损失**。

为什么？拿一个反例来想：

```
场景：预测房价（回归）
  平方损失  L = (y - f(x))²
  → 残差 = y - f(x)  ← "差了多少" 直接有意义

场景：泰坦尼克（分类，存活概率）
  交叉熵损失  L = -y·log(p) - (1-y)·log(1-p)
  → 残差 = y - p    ← p 是概率 0-1，y 是 0/1 标签
    差值含义不明——用这个去"拟合"新树是什么意思？
```

还有更多损失函数——MAE（绝对值损失）、Huber 损失（抗噪）……每种都对应不同场景。

**问题**：提升树要求拟合残差，但残差只在平方损失下有"最小化损失"的严格意义。其他损失函数没有对应的"残差"概念，没法照搬。

**GBDT 的解法**：把损失函数的**负梯度**作为通用"伪残差（pseudo-residual）"。不管什么损失函数，都能算出一个方向信号，告诉新树：**要往哪个方向修正**。

### 数学符号扫盲

| 符号 | 读法 | 直觉 | 例子 |
|---|---|---|---|
| $L(y, f(x))$ | L of y and f | 损失函数：真实值 $y$ 与预测 $f(x)$ 的"差距度量" | $\frac{1}{2}(y-f)^2$ |
| $\frac{\partial L}{\partial f(x)}$ | partial L over partial f | L 对 f 的偏导；预测稍微变一点，损失怎么变 | $-(y - f)$（平方损失） |
| $-\frac{\partial L}{\partial f(x)}$ | 负梯度（negative gradient） | 损失下降最快的方向 | $y - f$（平方损失 = 残差） |
| $\tilde{y}_i$ / $r_{mi}$ | r sub m i | 第 m 轮第 i 个样本的伪残差 | $y_i - f_{m-1}(x_i)$ |
| $\eta$ | eta / 学习率 | 每棵树贡献打折，防过拟合 | 0.1 |
| $f_{m-1}(x)$ | f sub m-1 | 前 m-1 棵树的累积预测值 | 前 2 轮预测之和 |

### 核心推导：负梯度在平方损失下 = 残差

平方损失（带 1/2 是为了求导后消掉系数）：

$$L(y, f(x)) = \frac{1}{2}(y - f(x))^2$$

对预测值 $f(x)$ 求偏导：

$$\frac{\partial L}{\partial f(x)} = \frac{\partial}{\partial f(x)} \frac{1}{2}(y - f(x))^2 = -(y - f(x))$$

加负号（取负梯度）：

$$-\frac{\partial L}{\partial f(x)} = y - f(x) = \text{残差}$$

**结论**：平方损失下，负梯度 = 残差。提升树是 GBDT 的特殊情形（平方损失版本）。

```
GBDT（通用）
    ↓  当 L = 平方损失时
提升树（Boosting Tree）
```

### 负梯度：不同损失函数下的"伪残差"

| 损失函数 | 场景 | $L(y, f)$ | 负梯度 $\tilde{y}_i$ |
|---|---|---|---|
| 平方损失 | 回归（连续值） | $\frac{1}{2}(y-f)^2$ | $y - f$（就是残差） |
| 绝对损失（MAE） | 回归（抗离群点） | $\lvert y-f \rvert$ | $\text{sign}(y-f)$（只有 +1/-1，方向信号） |
| 对数损失（logloss） | 二分类 | $-y\log(p)-(1-y)\log(1-p)$ | $y - p$（真实标签 − 预测概率） |

每种损失的负梯度都有直觉：**损失函数对当前预测说"差了多少、往哪修"，负梯度就是这个方向信号**。

### 梯度提升算法（5 步伪代码）

```
GBDT 算法

输入：训练集 {(x₁,y₁), ..., (xₙ,yₙ)}
      损失函数 L(y, f(x))
      弱学习器数量 T，学习率 η

──────────────────────────────────────────────

Step 0 · 初始化（找让所有样本平均损失最小的常数）

    f₀(x) = argmin_c Σᵢ L(yᵢ, c)

    平方损失下：令导数为 0 → c* = ȳ（均值）

──────────────────────────────────────────────

for m = 1, 2, ..., T:

    Step 1 · 计算伪残差（负梯度）

        r̃ᵢₘ = -[ ∂L(yᵢ, f(xᵢ)) / ∂f(xᵢ) ]  f=f_{m-1}

        平方损失下 = yᵢ - f_{m-1}(xᵢ)  （就是残差）

    Step 2 · 拟合一棵 CART 回归树

        以 {(xᵢ, r̃ᵢₘ)} 为训练集，得到 hₘ(x)
        （树的目标变量是伪残差，不是原始 y）

    Step 3 · 更新累积预测

        f_m(x) = f_{m-1}(x) + η · hₘ(x)
                                ↑
                      学习率：每棵树贡献打折，防过拟合

──────────────────────────────────────────────

Step 4 · 输出强学习器

    F(x) = f_T(x) = f₀(x) + η · Σₘ hₘ(x)
```

**学习率 $\eta$（shrinkage）**：每棵树的修正量乘以 $\eta$（通常 0.01-0.3），步子小一点，需要更多棵树，但泛化更好。类比梯度下降的步长：太大容易过冲，太小收敛慢。

### 一句话钉板

**GBDT = 提升树的通用化：用损失函数负梯度替代残差，让"每棵树拟合当前误差"的机制能适配任意可微损失函数；平方损失只是其中最简单的特例，此时负梯度退化为残差。**
