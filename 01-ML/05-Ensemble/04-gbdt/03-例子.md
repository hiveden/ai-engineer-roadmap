### 笔记

> 【理解】GBDT 例子（回归 · 平方损失下负梯度=残差，等价于提升树）

数据：10 个样本，目标值如 $\{5.56, 5.7, 5.91, \ldots, 9.05\}$。

**步骤 0 · 初始化**：用目标均值作为初始预测（最小化平方误差）

$$f_0(x) = \arg\min_c \sum (y_i - c)^2 = \bar{y}$$

**步骤 1 · 构建第 1 棵 CART**：拟合负梯度 $r_i = y_i - f_0(x_i)$

- 遍历切分点（如 1.5, 2.5, ..., 9.5）
- 每个切分点把数据分两份，每份用均值做预测，计算平方误差和
- 选误差最小的切分点（例如 6.5）

> **平方误差计算**（切分点 1.5）：左侧 [5.56]、右侧 [5.7, ..., 9.05]
> 左 SSE = 0；右 SSE = $\sum (y_i - 7.5011)^2 \approx 15.72$

**步骤 2 · 第 2 棵树**：在 $f_1 = f_0 + h_1$ 的基础上重新算负梯度，再建一棵 CART（最优切分点 3.5）。

**步骤 3 · 第 3 棵树**：同理（切分点 6.5）。

**最终强学习器**：$F(x) = f_0 + \sum_{t=1}^{T} h_t(x)$。

#### Step 1 · 第 1 棵树：拟合残差

计算各样本残差（$r = y - f_0 = y - 7$）：

```
残差表（轮 1）
──────────────────────────────────────
年级 x  │  1    2    3    4    5
真实 y  │  5    6    7    8    9
f₀(x)  │  7    7    7    7    7
残差 r  │ -2   -1    0    1    2
──────────────────────────────────────
```

现在以残差为目标，建一棵 CART 回归树，候选切分点 1.5 / 2.5 / 3.5 / 4.5：

**以 1.5 切分**：
- 左（年级=1）：均值 = -2，SSE = 0
- 右（年级=2,3,4,5）：均值 = (-1+0+1+2)/4 = 0.5，SSE = (-1-0.5)² + (0-0.5)² + (1-0.5)² + (2-0.5)² = 2.25+0.25+0.25+2.25 = 5.0
- 总 SSE = 5.0

**以 2.5 切分**：
- 左（年级=1,2）：均值 = (-2-1)/2 = -1.5，SSE = (-2+1.5)² + (-1+1.5)² = 0.25+0.25 = 0.5
- 右（年级=3,4,5）：均值 = (0+1+2)/3 = 1，SSE = (0-1)² + (1-1)² + (2-1)² = 1+0+1 = 2.0
- 总 SSE = 2.5 ← 更小

**以 3.5 切分**：
- 左（年级=1,2,3）：均值 = (-2-1+0)/3 = -1，SSE = (-2+1)² + (-1+1)² + (0+1)² = 1+0+1 = 2.0
- 右（年级=4,5）：均值 = (1+2)/2 = 1.5，SSE = (1-1.5)² + (2-1.5)² = 0.25+0.25 = 0.5
- 总 SSE = 2.5 ← 与 2.5 切分并列

**以 4.5 切分**：
- 左（年级=1,2,3,4）：均值 = (-2-1+0+1)/4 = -0.5，SSE = (-2+0.5)²+(-1+0.5)²+(0+0.5)²+(1+0.5)² = 2.25+0.25+0.25+2.25 = 5.0
- 总 SSE = 5.0

```
4 个候选切分点汇总

x      1.5    2.5    3.5    4.5
SSE     5     2.5    2.5     5

最优：2.5 或 3.5 均可，PPT 取 x=2.5
```

### 完整的 GBDT 推理流程

训练完成后，模型保存了：
1. $f_0 = 7.31$（常数基准）
2. 第 1 棵树（切分点 6.5，左叶 -1.07，右叶 +1.60）
3. 第 2 棵树（切分点 3.5，左叶 ...，右叶 ...）
4. 第 3 棵树（切分点 6.5，左叶 ...，右叶 ...）

推理时，新样本 $x=4.2$ 进来：

```
推理流程（x=4.2）

Step 1: f₀ = 7.31

Step 2: 第 1 棵树
   x=4.2 ≤ 6.5 → 走左支 → 修正 = -1.07
   f₁ = 7.31 + (-1.07) = 6.24

Step 3: 第 2 棵树
   x=4.2 > 3.5 → 走右支 → 修正 = +0.22
   f₂ = 6.24 + 0.22 = 6.46

Step 4: 第 3 棵树
   x=4.2 ≤ 6.5 → 走左支 → 修正 = +0.15
   f₃ = 6.46 + 0.15 = 6.61

最终预测 = 6.61
```

每棵树独立处理样本，结果累加。推理和训练的树数无关——有几棵就过几棵。

### 一句话钉板

**GBDT 手算流程：初始化为均值 → 每轮算残差（负梯度）→ 用 CART 拟合残差 → 累加修正量；残差轮轮收缩，最终预测 = 初始均值 + Σ(学习率 × 各棵树修正值)。**
