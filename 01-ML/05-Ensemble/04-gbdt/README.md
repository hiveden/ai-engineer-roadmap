#### Slide 50 · 提升树（Boosting Decision Tree）—— 残差树，做回归

**思想**：通过拟合残差的思想来进行性能提升

> 残差：真实值 - 预测值

**生活中的例子**：预测某人的年龄为 100 岁

```
真实值 = 100 岁

第 1 次预测：对 100 岁预测，预测成 80 岁；100 – 80 = 20（残差）
第 2 次预测：上一轮残差 20 岁作为目标值，预测成 16 岁；20 – 16 = 4（残差）
第 3 次预测：上一轮的残差 4 岁作为目标值，预测成 3.2 岁；4 – 3.2 = 0.8（残差）

若三次预测的结果串联起来：80 + 16 + 3.2 = 99.2
通过拟合残差可将多个弱学习器组成一个强学习器，这就是提升树的最朴素思想
```

**举例 2**（商品 82 元）：

```
商品总价 = 82 元

弱学习器 1：看到商品，预测 50 元；残差 32 元
弱学习器 2：看到商品，回归目标 32 元，预测 20 元；残差 12 元
弱学习器 3：看到商品，回归目标 12 元，预测 10 元；残差 2 元

模型之间的信息传递 = 前一个模型的残差（通信）

累加：50 + 20 + 10 = 80 ≈ 82 元
```

#### Slide 54 · 梯度提升树（GBDT）

**GBDT（Gradient Boosting Decision Tree）**：利用梯度下降的近似方法，利用损失函数的负梯度作为提升树算法中的残差近似值。

**假设**：
- 前一轮迭代得到的强学习器是：$f_{i-1}(x)$
- 损失函数为平方损失：$L(y, f_{i-1}(x))$
- 本轮迭代的目标是找到一个弱学习器：输出预测值 $h_i(x)$
- 让本轮的损失最小化：$\text{Loss} = L(y, f_{i-1}(x) + h_i(x))$
- 则要拟合单个样本的负梯度为：

$$r_i = -\left[\frac{\partial L(y, f(x))}{\partial f(x)}\right]_{f = f_{i-1}(x)}$$

即：对于平方损失，**GBDT 拟合的负梯度就是残差**。

```
对于任一样本（负梯度 = 残差，平方损失情形）：
  r_i = y_i - f_{i-1}(x_i)

对于任一模型参数：
  (梯度下降方向 = 负梯度方向)
```

#### Slide 55 · 梯度提升树——基础学习器（初始化）

**对于整个训练集：给出标签的均值，作为基准值。**

执行标准：当模型预测值为何值时，会使得第一个弱学习器在所有训练样本上预测的平方误差最小，即：求损失函数对 $f(x_i)$ 的导数，并令导数为 0。

$$L(y, f(x)) = \frac{1}{2}(y - f(x))^2 \Rightarrow \text{求平方误差最小}$$

$$\frac{\partial}{\partial f(x_i)} \sum_{i} L(y_i, f(x_i)) = 0$$

$$\Rightarrow f(x) = \frac{1}{n}\sum_{i=1}^{n} y_i = \bar{y}$$

从公式可得：初始化树的输出值为目标值的均值时，可使得第一个弱学习器的平方误差最小。

```
商品总价 82 元类比：
  初始猜测 = 均值（50+20+10 周围某个值）
  作为一个基准值，不能差太多！！
```

1. 初始化弱学习器（目标值的均值作为预测值）

3. 直到达到指定的学习器个数
4. 当输入未知样本时，将所有弱学习器的输出结果组合起来作为强学习器的输出

### 笔记

> 【理解】提升树（Boosting Tree）

**思想**：每棵新树拟合**上一轮的残差**（真实值 − 当前累积预测值）。

**直觉例子**：拟合 30 岁

- 第 1 棵树预测 20 岁，残差 10
- 第 2 棵树拟合残差 10，预测 6，剩余残差 4
- 第 3 棵树拟合 4，预测 3，剩余 1
- ...
- 累加所有树的预测 = 最终输出

> 【理解】梯度提升树

**核心改进**：不直接拟合残差，而是拟合**损失函数对当前模型的负梯度**（最速下降近似）。

**形式化**：

- 前一轮强学习器：$f_{i-1}(x)$
- 损失函数：$L(y, f_{i-1}(x))$
- 本轮目标：找弱学习器 $h_i(x)$ 使 $L(y, f_{i-1}(x) + h_i(x))$ 最小

**负梯度**作为本轮拟合目标：

$$r_i = -\left[\frac{\partial L(y, f(x))}{\partial f(x)}\right]_{f = f_{i-1}}$$

**平方损失情形**：$L = \frac{1}{2}(y - f(x))^2$，则 $-\frac{\partial L}{\partial f(x)} = y - f(x)$，即负梯度 $r_i = y_i - f_{i-1}(x_i) =$ 残差。所以平方损失下 GBDT 退化为提升树。

**分类情形**：损失换成 logloss，拟合的就是 logloss 的负梯度（不再是残差）。

> 【理解】GBDT 例子（回归 · 平方损失下负梯度=残差，等价于提升树）

数据：10 个样本，目标值如 $\{5.56, 5.7, 5.91, \ldots, 9.05\}$。

**步骤 0 · 初始化**：用目标均值作为初始预测（最小化平方误差）

$$f_0(x) = \arg\min_c \sum (y_i - c)^2 = \bar{y}$$

**步骤 1 · 构建第 1 棵 CART**：拟合负梯度 $r_i = y_i - f_0(x_i)$

- 遍历切分点（如 1.5, 2.5, ..., 9.5）
- 每个切分点把数据分两份，每份用均值做预测，计算平方误差和
- 选误差最小的切分点（例如 6.5）

> **平方误差计算**（切分点 1.5）：左侧 [5.56]、右侧 [5.7, ..., 9.05]
> 左 SSE = 0；右 SSE = $\sum (y_i - 7.5011)^2 \approx 15.72$

**步骤 2 · 第 2 棵树**：在 $f_1 = f_0 + h_1$ 的基础上重新算负梯度，再建一棵 CART（最优切分点 3.5）。

**步骤 3 · 第 3 棵树**：同理（切分点 6.5）。

**最终强学习器**：$F(x) = f_0 + \sum_{t=1}^{T} h_t(x)$。

**算法流程**：

1. 初始化（目标均值）
2. 迭代：每棵树拟合上一轮的负梯度
3. 直到达到指定树数 $T$
4. 预测：所有弱学习器输出之和

> 【实践】泰坦尼克号案例

```python
from sklearn.ensemble import GradientBoostingClassifier

gbdt = GradientBoostingClassifier(
    n_estimators=100,
    learning_rate=0.1,
    max_depth=3,
)
gbdt.fit(X_train, y_train)
```

**`learning_rate` 直觉**：对应梯度下降的步长 $\eta$，每棵新树的输出乘以 $\eta$ 再累加（shrinkage）。越小越稳但需要更多树。

**对比**：在随机森林案例上替换分类器即可，注意 GBDT 串行训练比 RF 慢。
