### PPT

> Slide 73 · XGBoost 目标函数

进行 $t$ 次迭代的学习模型的目标函数如下：

$$\text{Obj}^{(t)} = \sum_{i=1}^{n} L(y_i, \hat{y}_i^{(t-1)} + f_t(x_i)) + \Omega(f_t) + \text{const}$$

直接对目标函数求解比较困难，通过泰勒展开将目标函数换一种近似的表示方式。

> Slide 74 · XGBoost（复习）泰勒展开

**泰勒展开**：将一个函数在某一点处展开成无限项的多项式表达式（用一系列表达式，等价于 $f(x)$ 的值）

**一阶泰勒展开**：

$$f(x + \Delta x) \approx f(x) + f'(x) \Delta x$$

**二阶泰勒展开**：

$$f(x + \Delta x) \approx f(x) + f'(x) \Delta x + \frac{1}{2} f''(x) \Delta x^2$$

> Slide 75 · XGBoost 目标函数推导 2——泰勒展开

目标函数对 $\hat{y}_i^{(t-1)}$ 进行泰勒二阶展开，得到如下近似表示的公式：

$$L(y_i, \hat{y}_i^{(t-1)} + f_t(x_i)) \approx L(y_i, \hat{y}_i^{(t-1)}) + g_i f_t(x_i) + \frac{1}{2} h_i f_t^2(x_i)$$

观察目标函数，发现以下两项表示 $t-1$ 个弱学习器构成学习器的目标函数，都是常数，我们可以将其去掉：

$$\text{Obj}^{(t)} \approx \sum_{i=1}^{n} \left[g_i f_t(x_i) + \frac{1}{2} h_i f_t^2(x_i)\right] + \Omega(f_t)$$

其中 $g_i$ 和 $h_i$ 分别为损失函数的一阶导、二阶导：

$$g_i = \frac{\partial L(y_i, \hat{y}_i^{(t-1)})}{\partial \hat{y}_i^{(t-1)}}, \quad h_i = \frac{\partial^2 L(y_i, \hat{y}_i^{(t-1)})}{\partial (\hat{y}_i^{(t-1)})^2}$$

> Slide 76 · XGBoost 从样本角度转为叶子节点角度

从样本角度转为按照叶子节点输出角度，优化损失函数：

举个栗子：$m = 10$ 个样本，落在 D 结点 3 个样本，落在 E 结点 2 个样本，落在 F 结点 2 个样本，落在 G 结点 3 个样本；计算 $f_t(x_i)$ 的表达形式如下：

```
样本 → 叶子节点映射：
  D 结点（3 个样本）: w_D
  E 结点（2 个样本）: w_E
  F 结点（2 个样本）: w_F
  G 结点（3 个样本）: w_G
```

上式中：
- $g_i$ 表示每个样本的一阶导，$h_i$ 表示每个样本的二阶导
- $f_t(x_i)$ 表示样本的预测值
- $T$ 表示叶子结点的数目
- $\|w\|^2$ 由叶子结点值组成向量的模

> Slide 77 · XGBoost 目标函数转化

目标函数中的各项可以做以下转换：

$$g_i f_t(x_i) \Rightarrow \text{样本的预测值 } w_{q(x_i)}$$

$$h_i f_t^2(x_i) \Rightarrow \text{转换从叶子结点的角度看}$$

$$\lambda\|w\|^2 \Rightarrow \text{从叶子角度来看}$$

汇总得到：

$$\text{Obj}^{(t)} = \sum_{j=1}^{T} \left[G_j w_j + \frac{1}{2}(H_j + \lambda)w_j^2\right] + \gamma T$$

> Slide 78 · XGBoost 目标函数推导 3——转化为叶子节点输出角度

令：

$$G_j = \sum_{i \in I_j} g_i \quad \text{（所有落在叶子 j 的样本一阶导之和）}$$

$$H_j = \sum_{i \in I_j} h_i \quad \text{（所有落在叶子 j 的样本二阶导之和）}$$

最终：

$$\text{Obj}^{(t)} = \sum_{j=1}^{T} \left[G_j w_j + \frac{1}{2}(H_j + \lambda)w_j^2\right] + \gamma T$$

> Slide 79 · XGBoost 目标函数推导 4——最优解

**求损失函数最小值**

对 $w_j$ 求导并令其等于 0，可得到 $w$ 的最优值：

$$\frac{\partial}{\partial w_j}\left[G_j w_j + \frac{1}{2}(H_j + \lambda)w_j^2\right] = G_j + (H_j + \lambda)w_j = 0$$

$$w_j^* = -\frac{G_j}{H_j + \lambda}$$

最优 $w^*$ 带入公式可求目标函数的最小值：

$$\text{Obj}^* = -\frac{1}{2}\sum_{j=1}^{T} \frac{G_j^2}{H_j + \lambda} + \gamma T$$

> Slide 80 · XGBoost 打分函数

目标函数最终为：

$$\text{Obj}^* = -\frac{1}{2}\sum_{j=1}^{T} \frac{G_j^2}{H_j + \lambda} + \gamma T$$

该公式也叫做**打分函数（scoring function）**，从损失函数、树的复杂度两个角度来衡量一棵树的优劣。当我们构建树时，可以用来选择树的划分点：

**分裂增益（Gain）**：

$$\text{Gain} = \frac{1}{2}\left[\frac{G_L^2}{H_L + \lambda} + \frac{G_R^2}{H_R + \lambda} - \frac{(G_L + G_R)^2}{H_L + H_R + \lambda}\right] - \gamma$$

> Slide 81 · XGBoost 分裂判断

根据上一页 PPT 中计算的 gain 值：

对树中的每个叶子结点尝试进行分裂，计算分裂前 - 分裂后的分数：

- 如果 $\text{Gain} > 0$，则分裂之后树的损失更小，会考虑此次分裂
- 如果 $\text{Gain} < 0$，说明分裂后的分数比分裂前的分数大，此时不建议分裂

**当触发以下条件时停止分裂**：
- 达到最大深度
- 叶子结点数量低于某个阈值
- 所有的结点在分裂不能降低损失
- 等等...

**当损失函数是平方损失的时候：**

$$L = \frac{1}{2}(y_i - \hat{y}_i)^2$$

$$g_i = \frac{\partial L}{\partial \hat{y}} = -(y_i - \hat{y}_i) \quad (\text{系数} \times 2 \text{ 抵消})$$

$$h_i = \frac{\partial^2 L}{\partial \hat{y}^2} = 1$$

标签与随机猜测的差异即为一阶导。

### 笔记

> 【理解】XGBoost 目标函数

**整体目标**（前 $K$ 棵树）：

$$\mathrm{Obj} = \sum_{i=1}^{n} L(y_i, \hat{y}_i) + \sum_{k=1}^{K} \Omega(f_k)$$

第一项是样本损失，第二项是 $K$ 棵树的复杂度正则。

**树复杂度**：

$$\Omega(f) = \gamma T + \frac{1}{2}\lambda \|w\|^2$$

- $T$：叶子结点数；$\gamma$ 是叶子数惩罚系数
- $w$：叶子输出值向量；$\lambda$ 是 L2 正则系数

**第 $t$ 轮目标**（前 $t-1$ 棵已固定）：

$$\mathrm{Obj}^{(t)} = \sum_{i=1}^{n} L(y_i, \hat{y}_i^{(t-1)} + f_t(x_i)) + \Omega(f_t) + \mathrm{const}$$

**为何要展开**：损失加正则后形式复杂，难以直接对 $f_t$ 求负梯度优化。泰勒二阶近似把目标变成关于 $f_t$ 的二次函数，可直接闭式求解。

**泰勒二阶展开**（对 $\hat{y}_i^{(t-1)}$ 处展开）：

$$L(y_i, \hat{y}_i^{(t-1)} + f_t(x_i)) \approx L(y_i, \hat{y}_i^{(t-1)}) + g_i f_t(x_i) + \frac{1}{2} h_i f_t^2(x_i)$$

其中：

$$g_i = \partial_{\hat{y}^{(t-1)}} L, \quad h_i = \partial^2_{\hat{y}^{(t-1)}} L$$

**去常数项**后的目标：

$$\mathrm{Obj}^{(t)} \approx \sum_{i=1}^{n} \left[g_i f_t(x_i) + \frac{1}{2} h_i f_t^2(x_i)\right] + \gamma T + \frac{1}{2}\lambda \sum_{j=1}^{T} w_j^2$$

**视角变换**（按样本 → 按叶子）：设 $I_j = \{i \mid q(x_i) = j\}$ 为落到叶子 $j$ 的样本集合，$f_t(x_i) = w_{q(x_i)}$：

$$\mathrm{Obj}^{(t)} = \sum_{j=1}^{T} \left[\left(\sum_{i \in I_j} g_i\right) w_j + \frac{1}{2}\left(\sum_{i \in I_j} h_i + \lambda\right) w_j^2\right] + \gamma T$$

记 $G_j = \sum_{i \in I_j} g_i$，$H_j = \sum_{i \in I_j} h_i$：

$$\mathrm{Obj}^{(t)} = \sum_{j=1}^{T} \left[G_j w_j + \frac{1}{2}(H_j + \lambda) w_j^2\right] + \gamma T$$

**对 $w_j$ 求导得最优叶子值**：

$$w_j^* = -\frac{G_j}{H_j + \lambda}$$

**代回得目标最小值**（打分函数 / scoring function）：

$$\mathrm{Obj}^* = -\frac{1}{2} \sum_{j=1}^{T} \frac{G_j^2}{H_j + \lambda} + \gamma T$$

**分裂增益**（树构建时选最佳分裂点）：

$$\mathrm{Gain} = \frac{1}{2}\left[\frac{G_L^2}{H_L + \lambda} + \frac{G_R^2}{H_R + \lambda} - \frac{(G_L + G_R)^2}{H_L + H_R + \lambda}\right] - \gamma$$

- $\mathrm{Gain} > 0$ 才分裂（公式中已减去 $\gamma$，等价于增益 > $\gamma$ 阈值）
- 停止条件：达到最大深度 / 叶子样本数过低 / 增益不足

## ━━━━━━━━ 讲解 ━━━━━━━━

### 生活锚 · 给医生写诊疗手册：怎么算"该写多重的方"和"该不该多分一类患者"

延续 [`./01-算法思想.md`](./01-算法思想.md) 升级版接力医生。这一节做的事是：**把医生的诊疗手册从"凭经验"改成"按公式"**。

举两个具体动作：

- **每位接力医生写处方时该开多少剂量**（叶子值 leaf weight $w_j$）：上一棒留下的"差距 + 趋势"是已知数，本棒怎么给一个最优剂量？
- **要不要把患者再细分一类**（节点是否分裂 split）：分类后两组分得更准的"收益"，要超过"开新处方扣的分"才划算

GBDT 这两件事靠经验：剂量 = 残差均值，分裂靠 max_depth 拍脑袋。XGBoost 把目标函数 objective function（损失 + 正则）写出来后，**两件事都能用一个公式解出来**——连"何时停"都内置在公式里。

本节用 5 步推导走完：怎么把目标函数从「按样本求和」化成「按叶子求和」的二次函数，再对叶子值求导拿到最优解、推出打分函数和分裂 Gain。

### 业务问题：为什么不直接对损失求导

工程师听到「直接对损失最小化」会觉得简单——求导令 0 就行。卡点在哪？

- **损失 $L(y_i, \hat{y}_i^{(t-1)} + f_t(x_i))$ 的形式不固定**：分类用 logloss、回归用 MSE、排序用 pairwise，每种损失的导数形式都不一样
- **每写一种损失就重写一套求解器**，代码维护爆炸
- **加了正则项后**，闭式解更难求

XGBoost 的解法两步：

1. 不管 $L$ 是什么形式，**用泰勒二阶展开打平成关于 $f_t$ 的二次多项式**——一阶导 $g_i$、二阶导 $h_i$ 是接口，每种损失只要提供这两个数即可
2. 二次函数对叶子值 $w_j$ 求导，**闭式解析解**直接拿到最优叶子值

这个设计让 XGBoost 支持任意可二阶导损失——分类回归排序统一一套求解器。下面把推导拆 5 步走。

### 【数学】数学符号扫盲

在推导前先把符号捋清楚：

| 符号 | 读法 | 含义 | 例子 |
|---|---|---|---|
| $\hat{y}_i^{(t)}$ | y-hat i at t | 第 $i$ 个样本在第 $t$ 轮后的预测值 | 前 3 棵树累加的分数 |
| $f_t(x_i)$ | f-t of x-i | 第 $t$ 棵树对样本 $i$ 的输出 | 第 4 棵树的叶子值 |
| $g_i$ | g-i | 损失对 $\hat{y}_i^{(t-1)}$ 的一阶导（gradient） | 残差方向（斜率） |
| $h_i$ | h-i | 损失对 $\hat{y}_i^{(t-1)}$ 的二阶导（Hessian） | 曲率（弯曲程度） |
| $T$ | 大写 T | 当前树的叶子总数 | 深度 3 二叉树 → 最多 8 叶 |
| $w_j$ | w-j | 第 $j$ 个叶子的输出值（权重 weight） | 叶子 D 输出 +2 |
| $I_j$ | I-j | 落到叶子 $j$ 的样本下标集合 | $I_D = \{1, 4, 7\}$ |
| $G_j$ | G-j | $\sum_{i \in I_j} g_i$，叶子 $j$ 内一阶导之和 | 3 个样本 $g$ 求和 |
| $H_j$ | H-j | $\sum_{i \in I_j} h_i$，叶子 $j$ 内二阶导之和 | 3 个样本 $h$ 求和 |
| $\lambda$ | lambda | L2 正则系数（超参数） | 控制叶子值不能太大 |
| $\gamma$ | gamma | 叶子数惩罚系数（超参数） | 控制树不能太多叶子 |

### 【数学】推导流程（5 步走）

```
Step 1  写出第 t 轮目标函数
Step 2  对损失做二阶泰勒展开（近似化简）
Step 3  去掉常数项
Step 4  把"按样本求和"换成"按叶子求和"
Step 5  对叶子值 wⱼ 求导 → 最优叶子值 → 代回得打分函数
```

#### Step 1：第 t 轮目标

前 $t-1$ 棵树已固定，第 $t$ 棵树 $f_t$ 是待学习的：

$$\text{Obj}^{(t)} = \sum_{i=1}^{n} L\!\left(y_i,\; \underbrace{\hat{y}_i^{(t-1)}}_{\text{已知常数}} + f_t(x_i)\right) + \Omega(f_t) + \text{const}$$

直接对 $f_t$ 求最优，损失项形式复杂（取决于 $L$ 的具体形式）。

#### Step 2：二阶泰勒展开——化繁为简

把损失看作关于 $f_t(x_i)$ 的函数，在 $f_t = 0$（即当前预测值 $\hat{y}^{(t-1)}$）处展开：

$$L(y_i, \hat{y}_i^{(t-1)} + f_t) \approx L(y_i, \hat{y}_i^{(t-1)}) + g_i \cdot f_t(x_i) + \frac{1}{2} h_i \cdot f_t^2(x_i)$$

其中：
- $g_i = \left.\frac{\partial L}{\partial \hat{y}}\right|_{\hat{y} = \hat{y}_i^{(t-1)}}$（一阶导，损失的"坡度"）
- $h_i = \left.\frac{\partial^2 L}{\partial \hat{y}^2}\right|_{\hat{y} = \hat{y}_i^{(t-1)}}$（二阶导，损失的"弯曲"）

**平方损失的具体例子**（$L = \frac{1}{2}(y_i - \hat{y}_i)^2$）：

```
g_i = ∂L/∂ŷ = -(y_i - ŷ_i)   = ŷ_i - y_i    （负残差）
h_i = ∂²L/∂ŷ² = 1                              （常数！）
```

$h_i = 1$ 意味着：平方损失情况下，二阶导恒为 1，每个样本"权重"相同。

#### Step 3：去常数项

展开后目标函数（对所有样本求和）：

$$\text{Obj}^{(t)} \approx \sum_{i=1}^n \left[\underbrace{L(y_i, \hat{y}_i^{(t-1)})}_{\text{常数，去掉}} + g_i f_t(x_i) + \frac{1}{2} h_i f_t^2(x_i)\right] + \Omega(f_t)$$

去掉与 $f_t$ 无关的常数项：

$$\text{Obj}^{(t)} \approx \sum_{i=1}^n \left[g_i f_t(x_i) + \frac{1}{2} h_i f_t^2(x_i)\right] + \gamma T + \frac{1}{2}\lambda \sum_{j=1}^T w_j^2$$

#### Step 4：从"按样本"换成"按叶子"

每个样本 $x_i$ 被树 $f_t$ 分到某个叶子 $j$，在那个叶子输出 $w_j$，即 $f_t(x_i) = w_j$。

设 $I_j$ = 落到叶子 $j$ 的所有样本下标，把求和重组：

```
按样本求和                       按叶子求和
────────────────────────────────────────────────────
Σᵢ [gᵢ·f_t(xᵢ) + ½hᵢ·f_t²(xᵢ)]
= Σⱼ Σᵢ∈Iⱼ [gᵢ·wⱼ + ½hᵢ·wⱼ²]   ← 同一叶子内 wⱼ 相同，提出
= Σⱼ [(Σᵢ∈Iⱼ gᵢ)·wⱼ + ½(Σᵢ∈Iⱼ hᵢ)·wⱼ²]
= Σⱼ [Gⱼ·wⱼ + ½Hⱼ·wⱼ²]          ← 令 Gⱼ = Σgᵢ, Hⱼ = Σhᵢ
```

加上正则项，目标函数变为：

$$\text{Obj}^{(t)} = \sum_{j=1}^T \left[G_j w_j + \frac{1}{2}(H_j + \lambda) w_j^2\right] + \gamma T$$

这是 $T$ 个**独立**的二次函数之和（每个叶子 $j$ 独立贡献），可以对每个 $w_j$ 分别求最优。

#### Step 5：求最优叶子值 → 打分函数

对 $w_j$ 求导，令导数 = 0：

$$\frac{d}{dw_j}\left[G_j w_j + \frac{1}{2}(H_j + \lambda) w_j^2\right] = G_j + (H_j + \lambda) w_j = 0$$

$$\boxed{w_j^* = -\frac{G_j}{H_j + \lambda}}$$

**直觉**：$G_j$ 是该叶子内所有样本的一阶导之和（"合力方向"），$H_j + \lambda$ 是"阻力"（二阶导总和 + 正则）。最优叶子值 = 合力 / 阻力，方向取反（负号）。

对比 GBDT：叶子值 = 残差均值（仅用一阶信息）；XGBoost 叶子值 $w_j^*$ 同时考虑一阶导、二阶导和正则，更稳健。

代回 $w_j^*$ 到目标函数，得到**打分函数（scoring function）**——衡量一棵树有多好：

$$\boxed{\text{Obj}^* = -\frac{1}{2}\sum_{j=1}^T \frac{G_j^2}{H_j + \lambda} + \gamma T}$$

第一项越负越好（损失小），第二项 $\gamma T$ 是叶子数惩罚（越多叶子扣越多分）。

### 【数学】分裂增益：如何用打分函数选最优切分

有了打分函数，分裂增益就是"切前得分 - 切后得分"：

```
父节点（未切分）
  Obj_parent = -½ · (GL+GR)²/(HL+HR+λ) + γ

分裂为左右子节点
  Obj_left  = -½ · GL²/(HL+λ) + γ
  Obj_right = -½ · GR²/(HR+λ) + γ
  Obj_after = Obj_left + Obj_right = -½[GL²/(HL+λ) + GR²/(HR+λ)] + 2γ

Gain = Obj_parent - Obj_after
     = ½[GL²/(HL+λ) + GR²/(HR+λ) - (GL+GR)²/(HL+HR+λ)] - γ
```

$$\text{Gain} = \frac{1}{2}\left[\frac{G_L^2}{H_L + \lambda} + \frac{G_R^2}{H_R + \lambda} - \frac{(G_L + G_R)^2}{H_L + H_R + \lambda}\right] - \gamma$$

- $\text{Gain} > 0$：切完之后目标更小，值得分裂
- $\text{Gain} \leq 0$：分裂后反而因为新增叶子被 $\gamma$ 惩罚，**不分裂**（自动停止）

**$\gamma$ 的角色**：Gain 公式里直接减了 $\gamma$，等价于"分裂的收益必须大于开一个叶子的成本"。这个停止条件是数学内置的，不需要人为设阈值。

### 与 GBDT 的核心差异汇总

```
对比维度            GBDT                    XGBoost
──────────────────────────────────────────────────────────
优化依据         负梯度（一阶 gᵢ）        一阶 gᵢ + 二阶 hᵢ
叶子值           残差均值                 -Gⱼ/(Hⱼ+λ)（由导数和正则决定）
分裂准则         降低残差平方和           Gain > 0（从目标函数推导）
停止条件         手动设 max_depth 等      Gain ≤ 0 自动停（γ 内置）
正则化           无内置正则              γT + ½λ‖w‖²（目标函数级别）
通用损失函数     难（需实现各自负梯度）  容易（gᵢ/hᵢ 通用接口）
```

### 【数学】平方损失下一个具体走查（虚构示意）

把推导落到数字上感受一遍。设 $L = \frac{1}{2}(y - \hat{y})^2$，$\lambda = 1$，叶子 $j$ 内有 3 个样本，残差分别 $-2, +1, +3$（即 $\hat{y} - y$）：

```
平方损失下：
  gᵢ = ŷᵢ - yᵢ = 残差     hᵢ = 1

叶子 j：
  Gⱼ = (-2) + 1 + 3 = 2
  Hⱼ = 1 + 1 + 1 = 3

最优叶子值：
  wⱼ* = -Gⱼ / (Hⱼ + λ) = -2 / (3 + 1) = -0.5

对比 GBDT：残差均值 = (2 + (-1) + (-3)) / 3 ≈ -0.67
（XGBoost 的 -0.5 比 GBDT 的 -0.67 收得更紧——λ=1 把叶子值往 0 拉，正则起作用了）
```

λ 越大，叶子值被拉回 0 的力越强；这正是「开重药扣分」生活锚在数字上的样子。
