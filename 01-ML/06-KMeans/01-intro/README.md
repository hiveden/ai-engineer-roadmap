> 01 · 聚类算法简介

**学习目标**：

2. 了解聚类算法的应用场景

> 【知道】聚类算法介绍

**聚类**（Clustering）：典型的**无监督学习**算法，根据样本之间的相似性自动把相似样本归到同一类别。

与分类的核心区别：分类有标签（监督），聚类只有特征、没有标签——簇的语义靠人事后解读。

相似性度量决定聚类结果——常用**欧氏距离**（Euclidean distance）：距离越小越相似。

> 【了解】聚类算法在现实中的应用

- 用户画像 / 广告推荐 / Data Segmentation / 搜索流量推荐 / 恶意流量识别
- 基于位置信息的商业推送 / 新闻聚类 / 筛选排序
- 图像分割 / 降维 / 识别；离群点检测；信用卡异常消费；功能相似的基因片段挖掘

工程视角：聚类是**探索性分析**（EDA）的常规手段——拿到无标签数据先聚一把，看数据是否自然分群。

> 【知道】分类

聚类算法按**簇的形成方式**分两大类：

| 类别 | 代表算法 | 思路 |
|---|---|---|
| **划分式聚类** | K-Means / K-Medoids | 预先指定簇数 K，迭代优化划分 |
| **层次聚类** | Agglomerative / Divisive | 自底向上合并 / 自顶向下分裂，形成树状结构（dendrogram） |
| **密度聚类** | DBSCAN / OPTICS | 按样本密度连通性划簇，能发现任意形状簇 + 自动识别噪声 |
| **谱聚类** | Spectral Clustering | 基于图拉普拉斯矩阵特征向量降维后再聚 |

本柱聚焦 **K-Means**（划分式 + 欧氏距离 + 迭代质心更新）作为聚类算法入门基线。
