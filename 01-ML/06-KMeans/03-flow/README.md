# 第 3 章 · KMeans 算法流程

> 维度：**算法**

## 底稿

> 03 · KMeans 算法流程

**学习目标**：

1. 理解 KMeans 算法的执行过程

> 【掌握】K-Means 聚类流程

**4 步迭代**：

1. **初始化**：在特征空间随机选 K 个点作为初始聚类中心（质心）
2. **分配**：对每个样本计算到 K 个质心的距离，归到**最近**的质心所属簇
3. **更新**：对每个簇，重新计算簇内所有样本的**均值**作为新质心
4. **判停**：若新质心 == 旧质心（不再移动）→ 结束；否则回到第 2 步

**目标函数**（簇内平方误差和 SSE，下一章详述）：

$$J = \sum_{i=1}^{K} \sum_{p \in C_i} \lVert p - m_i \rVert^2$$

每次迭代都使 $J$ 单调下降。**K-Means 一定会停**——质心只可能在有限多种划分下取值，且 $J$ 单调减不会循环。

**收敛性 vs 最优性**：保证收敛到**局部最优**，不保证全局最优——结果依赖初始质心选择。工程实践用 `n_init` 多次随机初始化取最优，或用 K-Means++ 智能初始化（sklearn 默认）。

> 【练习】案例练习

**场景**：5 个二维点 + K=2，初始质心 P1、P2。

**第 1 轮迭代**：

1. 设初始质心 P1、P2
2. 对每个点算到 P1、P2 的欧氏距离 $d = \sqrt{(x_1 - x_2)^2 + (y_1 - y_2)^2}$，归到最近质心的簇
3. 重新计算每簇的均值作为新质心。例：簇 2 重算后 P2′ = (2.3, 3.3)
4. 新 ≠ 旧 → 继续

**第 2 轮迭代**：重复 2 ~ 3 步，再次得到新质心。

**收敛**：当某一轮新质心 == 上一轮质心时，算法终止。

**核心结论**：每轮迭代 SSE 单调下降，所以一定会停，不会陷入"无限选质心"循环。
