# 拉普拉斯平滑

> 维度：**数学**

## 底稿（PPT 原文 + 笔记）

> 【知道】拉普拉斯平滑

**问题**：训练集里某个 $P(w_i \mid C) = 0$（词没出现过），整个连乘归零，分类器直接挂掉。

**解法**：分子 +α，分母 +αm。

$$P(w_i \mid C) = \frac{N_i + \alpha}{N + \alpha m}$$

- $\alpha$：平滑系数，通常取 1（拉普拉斯平滑），$0 < \alpha < 1$ 称 Lidstone 平滑
- $N_i$：类别 $C$ 中特征 $w_i$ 出现次数
- $N$：类别 $C$ 下所有特征出现总次数
- $m$：特征取值数（文本场景下即词表大小）

**骰子案例**

你有一个六面骰子，但不知道是否公平。你掷了 5 次，结果是： [1, 1, 2, 3, 1] 你想估计每个点数出现的概率。

朴素估计：
- 点数 1: 3 次 → P(1) = 3/5 = 0.6
- 点数 2: 1 次 → P(2) = 0.2
- 点数 3: 1 次 → P(3) = 0.2
- 点数 4、5、6: 0 次 → P(4) = P(5) = P(6) = 0

骰子并非不能扔出4、5、6点 仅仅是因为样本少而已。

加拉普拉斯平滑（α=1）：

$$P(x) = \frac{\text{count}(x) + 1}{N + M}$$

总观测次数 N = 5；所有可能得特征数 M = 6 (1~6)；P(x) = (count(x)+1)/(5+6)

$$P(1) = (3+1)/11 \approx 0.364$$
$$P(2) = (1+1)/11 \approx 0.182$$
$$P(3) = (1+1)/11 \approx 0.182$$
$$P(4) = (0+1)/11 \approx 0.091$$
$$P(5) = 0.091$$
$$P(6) = 0.091$$

## 讲解

### 业务问题：一个没见过的词 → 整条评论被压成 0

第 3 节末尾埋了个坑：连乘 $\prod_{i=1}^{n} P(w_i \mid C_k)$ 中**任何一项是 0，整个乘积归零**。

实战里这个坑被触发的频率高到离谱。设想垃圾邮件分类器训练时，训练集的 spam 邮件里**碰巧**没有「报销」这个词。新邮件「请报销机票」来了：

$$P(\text{spam} \mid \text{请},\text{报销},\text{机票}) \;\propto\; P(\text{请}\mid\text{spam}) \cdot \underbrace{P(\text{报销}\mid\text{spam})}_{=\,0} \cdot P(\text{机票}\mid\text{spam}) \;=\; 0$$

不管其他词多么强烈地指向 spam，**一个没见过的词就直接一票否决**。这显然不合理——「没见过」不等于「不可能」，只是训练集样本不够。

```
   训练集：spam 邮件里没出现 "报销"
                      │
                      ▼
        P(报销|spam) = 0/N = 0
                      │
                      ▼
   连乘归零 ── 整条新邮件被判 P(spam) = 0
                      │
                      ▼
        分类器瘫痪：没人见过的词 = 神圣不可侵犯？
```

**问题本质**：用样本频率直接估概率，**频率为 0 ≠ 概率为 0**。需要给「没见过」留一点可能性。

### 数学方法

#### 公式：分子加 α，分母加 αm

$$P(w_i \mid C) \;=\; \frac{N_i + \alpha}{N + \alpha m}$$

| 符号 | 含义 |
|---|---|
| $\alpha$ | 平滑系数。**通常取 1**，称拉普拉斯平滑 |
| $N_i$ | 类别 $C$ 中特征 $w_i$ 出现次数 |
| $N$ | 类别 $C$ 下所有特征出现的总次数 |
| $m$ | 特征取值数（文本场景下 = 词表大小） |

**分子加 $\alpha$ 是关键**：哪怕 $N_i = 0$，分子也变成 $\alpha > 0$，概率不再是零。

**分母加 $\alpha m$ 是配套**：保证所有取值的概率加起来仍然等于 1（归一化没破）。

> 直觉口号：「**每个特征预先虚构 $\alpha$ 次出现**」。$\alpha=1$ 时就是「每个词假装都先见过 1 次」，相当于给词表里每个词送 1 张「曾经出现」的入场券。

#### 骰子案例：5 次掷骰能下结论 P(4)=0 吗？

底稿那个例子非常对题。掷 6 面骰子 5 次，结果 [1, 1, 2, 3, 1]。

**朴素估计**（直接按频率）：

```
点数：    1     2     3     4     5     6
次数：    3     1     1     0     0     0
P：      0.6   0.2   0.2   0.0   0.0   0.0
```

ASCII bar：

```
P(1) ████████████████████████████████████  0.600
P(2) ████████████                          0.200
P(3) ████████████                          0.200
P(4)                                       0.000   ← 真的不可能吗？
P(5)                                       0.000
P(6)                                       0.000
```

**问题**：你只掷了 5 次，怎么就敢说 4、5、6 永远不会出现？只是没轮到，不是不能。

**加拉普拉斯平滑**（$\alpha=1$，$N=5$，$m=6$）：

$$P(x) = \frac{\text{count}(x) + 1}{5 + 6} = \frac{\text{count}(x) + 1}{11}$$

```
点数：    1       2       3       4       5       6
分子：   3+1=4   1+1=2   1+1=2   0+1=1   0+1=1   0+1=1
P：      4/11    2/11    2/11    1/11    1/11    1/11
        ≈0.364  ≈0.182  ≈0.182  ≈0.091  ≈0.091  ≈0.091
```

ASCII 对比：

```
              朴素                          拉普拉斯平滑（α=1）
P(1) ████████████████████████  0.600   ████████████████  0.364
P(2) ████████              0.200   ████████              0.182
P(3) ████████              0.200   ████████              0.182
P(4)                       0.000   ████                  0.091   ← 救活了
P(5)                       0.000   ████                  0.091
P(6)                       0.000   ████                  0.091
                                   ─────────────────────────
                                   合计 = 1.000（归一化保持）
```

**变化的两个方向**：

1. **零概率被救活**：4、5、6 不再是 0，给它们留了 $\approx 9\%$ 的可能性
2. **高频被压低**：P(1) 从 0.6 降到 0.36——把概率「分」给了未见过的取值

直觉是「**贝叶斯学派对极端估计的不信任**」：5 次样本不足以下定论，所以在频率估计上做一次「向均匀分布靠拢」的调和。$\alpha$ 越大，调和力度越强，越向均匀分布靠近。

#### α 的取舍

| α 值 | 名称 | 适用 |
|---|---|---|
| $\alpha = 1$ | 拉普拉斯平滑 | 默认选择，工程友好 |
| $0 < \alpha < 1$ | Lidstone 平滑 | 训练集大、想少干预频率估计时 |

> $\alpha$ 越小越「相信数据」，越大越「向均匀分布平滑」。$\alpha=0$ 退化为朴素估计（零概率回来了）；$\alpha\to\infty$ 退化为完全均匀分布（数据被淹没）。1 是个在「救零」和「不太歪」之间的工程默认。

#### 回到朴素贝叶斯

把平滑后的 $P(w_i \mid C)$ 代回第 3 节的连乘式，再也不会出现「整条样本归零」的崩溃：

$$\hat{C} \;=\; \arg\max_{C_k} \; P(C_k) \prod_{i=1}^{n} \underbrace{\frac{N_i^{(k)} + \alpha}{N^{(k)} + \alpha m}}_{\text{平滑后的}\,P(w_i\mid C_k)}$$

至此，朴素贝叶斯的全部数学基础齐了：**条件 / 联合 / 独立 → 贝叶斯翻转 → 朴素简化 → 平滑救零**。下一章用这套机器做「书 垃圾」情感分类的全流程实算。
