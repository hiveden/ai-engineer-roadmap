## 术语处理

首次出现的 ML / 工程专业词必须**中英双标**（例：`加载 load` / `探索性数据分析 EDA` / `精确率 precision` / `召回率 recall`）——中文翻译常"低分辨率"（一个词覆盖多个英文动词），工程师要对齐英文原词才能读文档、和社区对话。

具体的概念展开方式（案例 / 公式 / 锚点 / 二元判停结构）以《机器学习学习框架.md》子框架 B（循环框架）的内循环步骤为准，本文不预设。

## 工具库处理原则

**numpy / pandas**：不单独投入时间学。Alex 的 SQL / Lodash / Array 直觉已覆盖 80% API。

**遇到时主动做**：
- 就地讲解涉及的基础知识点（1-3 句）
- 首次出现的 API 给中英双标 + 工程类比
- 标注坑点（`SettingWithCopyWarning` / `dtype=object` / 链式索引 / 向量化 vs 循环）

**不做**：暂停主线去过章节、出专题练习。
