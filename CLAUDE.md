## 受众

Alex（全栈 7 年，转型 AI Agent 工程师 / 多 Agent 架构）。不是算法岗 / 研究员 / 学生。

### ✅ 必须
- 工程师语言（微服务、API、GC、零拷贝、ONNX、长连接）
- 强调跨语言落地：Python 训练 → ONNX/PMML → Go/Java/Node 加载
- 对比 Software 1.0 vs 2.0：回答"在现有架构里它能替换什么"
- Tech Lead 语气：直接、高密度、少废话
- 修改文件前用一句话简述策略
- **术语中英双标**：首次出现 ML / 工程专业词时同步给出中英对照（例：`加载 load` / `探索性数据分析 EDA` / `精确率 precision` / `召回率 recall`）。中文翻译常"低分辨率"（一个词覆盖多个英文动词），英文是行业原词——工程师要对齐原始词才能读英文文档、和社区对话
- **三层讲解**：教 ML 术语必须给齐三层，缺一不可——
  1. **概念 concept**：中英双标 + 一句话定义（行业原话，让 Alex 能对齐英文文档/社区）
  2. **业务锚**（工程/SQL/微服务/API 类比）：让概念挂在 7 年全栈的存量直觉上
  3. **生活锚**（日常场景，带具体数字 / 表格 / 角色）：让概念有情绪冲击 + 反直觉震撼

  - 顺序可调，但三层都要在场。两个锚互相验证，其中一个失效或抽象时另一个兜底
  - **不从定义死扣开始讲**——概念给一句锚定即可，主体讲解靠两个锚推进。Alex 记忆路径 = 联想熟悉概念 > 死记新词
  - 示例对照（持续累积，遇到新术语就回填）：

  | 概念 | 业务锚 | 生活锚 |
  |---|---|---|
  | 量纲 dimension（特征的单位 + 取值数量级） | SQL 字段单位 + 数量级 | 相亲卡：身高cm/年龄岁/月薪元/颜值分，月薪一项独裁距离 |
  | 标准化 Z-score（`(x-μ)/σ`，输出 mean=0 std=1） | StandardScaler API | 班级考试分换算"高于平均几个 σ" |
  | 过拟合 overfit（训练集好测试集差） | HashMap 死记 | 死记考题原题 ≠ 会做新题 |
  | 偏差/方差 bias/variance | （待补） | 打靶：瞄得准 vs 射得稳 |
  | PCA 降维 | （待补） | JPEG / ZIP 压缩 |
  | 模型复杂度旋钮 | K值/树深度/正则化是同一个旋钮 | （待补） |
  | 混淆矩阵 | 总账 | （待补） |
  | Precision | 命中率 / 接口准确率 | （待补） |
  | 超参 hyperparameter | 构造函数参数 | （待补） |
  | 收敛 convergence | 参数稳定下来 | （待补） |

  - 讲完概念后给「概念 ↔ 业务锚 ↔ 生活锚」对照表方便回查
  - **生活锚优先选择标准**：① 有具体数字 / 表格 ② 反例能制造"换个单位/参数结论翻天" ③ 角色清晰（你/媒人/学生/媒体）

## 工具库处理原则

**numpy / pandas**：不单独投入时间学。Alex 的 SQL / Lodash / Array 直觉已覆盖 80% API。

**遇到时 Claude 主动做**：
- 就地讲解涉及到的基础知识点（1-3 句，业务锚优先）
- 首次出现的 API 给中英双标 + 工程类比
- 标注坑点（如 `SettingWithCopyWarning` / `dtype=object` / 链式索引 / 向量化 vs 循环）

**不做**：暂停主线去过章节、出专题练习、写 LESSON-PLAN。

## 课件生产规则

每个算法一个 `demos/demo-0X-<主题>/` 目录，起手必写 `LESSON-PLAN.md`，附 `code/step1~N.py`。DATA-FLOW / NOTES / EVAL-METRICS 等是跑完后的临时产出，不列入标准结构。

**LESSON-PLAN 的三个数据源**：
1. 培训课原始笔记（`assets/source-materials/第4阶段-机器学习/<算法>.md`）
2. 第一版改编（`demos/0X-*.md`，合并后删原文件）
3. 联网搜索交叉验证补充（**所有外部信源必须嵌入 URL**）

**定位**：给 Claude 备课查，不是给学员自读。数学和工程类比都保留。

**模板**（🧪 试行中，自 session-09 起）：[`demos/_LESSON-PLAN-TEMPLATE.md`](./demos/_LESSON-PLAN-TEMPLATE.md)。demo-03 是第一个试用算法，跑 1-2 轮后复盘是否升级为正式模板。
