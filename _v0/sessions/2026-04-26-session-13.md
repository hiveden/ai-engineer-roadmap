### 3.1 站 00 · sklearn KNN 基础 API

**sklearn 三件套**（所有算法通用）：
```python
model = KNeighborsClassifier(n_neighbors=1)  # 构造（设超参）
model.fit(X, y)                              # 训练
model.predict([[4]])                         # 预测
```

**lazy learner 钩子**：fit 几乎 O(1) 只存数据 / predict 才算距离投票。对比 eager learner（决策树/NN 等）fit 时算完所有参数。**工程后果：训练快 / 预测慢，线上不友好——ANN/向量数据库的存在动因**。

**K=1 教学原因**：让示例输出可口算验证（`predict([[4]])` 最近邻 = 3 → 标签 1）。生产用 3/5/7 奇数。

### 3.2 站 01 · 暴力计算 timing

**实验目的**：用秒表证明站 00 的 lazy learner 钩子。

**load_digits**：1797 × 64（mini MNIST，真 MNIST 是 28×28），`return_X_y=True` 直接拆 (X, y)。

**漏讲（待补）**：

- `model._fit_method` 可查实际选了哪个算法
- `time.perf_counter()` > `time.time()` 精度
- **代码没量 accuracy**（`model.score()` vs `accuracy_score()`）

### 3.3 站 02 · 特征缩放

**量纲（dimension / scale）三层焊**：
- [概念] 特征的单位 + 取值数量级
- [业务锚] SQL 字段单位 + 数量级
- [生活锚] **相亲卡：身高cm/年龄岁/月薪元/颜值分**——月薪一项独裁距离
  ```
  d² = 5² + 2² + 10000² + 2² ≈ 1 亿  ← 月薪贡献 99.99999%
  ```

**反例震撼**：把月薪从"元"改成"万元"，结果天翻地覆——**机器对单位无感，全靠数字大小**。

**两把尺子对照**：
| | MinMax | Standard (Z-score) |
|---|---|---|
| 公式 | `(x-min)/(max-min)` | `(x-μ)/σ` |
| 范围 | `[0,1]` | mean=0 std=1 |
| 异常值 | 极敏感 | 较鲁棒 |
| 默认 | NN / 图像像素 | KNN/SVM 工业首选 |

**新增第三把尺子**：`RobustScaler`（中位数 + IQR）——异常值最鲁棒。

**fit/transform 铁律**：
```python
scaler.fit_transform(X_train)   # ✅
scaler.transform(X_test)        # ✅
scaler.fit_transform(X_test)    # ❌ data leakage
```

**Pipeline 解药**：强制契约 = "训练集 fit，其他只 transform"，无法写错——站 05 的主题铺垫。

**命名坑**：sklearn 存 `var_`（方差），不是 `std_`；`transformer.scale_` ≈ `sqrt(var_)` 才是真正用来除的值（处理零方差列）。

**库间默认值差异**（工程坑）：
| 库 | 函数 | 默认分母 |
|---|---|---|
| sklearn | `var_` | n |
| numpy | `np.var/std` | n |
| pandas | `.std()` | **n-1** |

### 3.4 站 03 · Iris 数据集探索（进行中）

**Bunch 对象三层焊**：
- [概念] `sklearn.utils.Bunch` = dict 子类，同时支持 `obj.key` 和 `obj['key']`
- [业务锚] JS 对象字面量 / Python SimpleNamespace + dict
- [生活锚] **宜家家具的纸盒**——零件（data）+ 零件名贴纸（feature_names）+ 标签盘（target/target_names）+ **说明书（DESCR）**

**字符串 key 不报错三层焊**（Alex 追问）：
- [概念] 写对了不报错，写错了 runtime 才炸（KeyError / AttributeError）
- [业务锚] JS `obj['key']` 拼错只 `undefined`；TS/Java/Go 编译时报错；Python = JS 同款宽松
- [生活锚] **字典查单词**——拼错"appel"字典也不喊你"拼错了"，只默默"找不到"

**data / target shape & dtype 三层焊**：
- [概念] `data.shape=(150,4) float64` / `target.shape=(150,) int64` —— X 2D / y 1D 铁律
- [业务锚] `target` 整数 0/1/2 ≈ enum ordinal；`target_names` ≈ enum name 数组
- [生活锚] **快递分拣**——包裹标签写数字 `1/2/3` 不是写城市名（机器读数字快、占内存少）
