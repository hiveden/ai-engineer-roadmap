# learning-sessions

学习会话元认知日志。每次会话产生一份结构化报告，捕捉：

- 涉及概念
- 暴露盲区
- 下次起点
- 元反思（用户亲笔填写，Claude 不代笔）

## 命名

`YYYY-MM-DD-session-NN.md`，编号从 `session-01` 起。

## 当前阶段

v1 试行从 `session-01` 重启。v0 阶段的 13 个 session 已归档至 [`../_v0/sessions/`](../_v0/sessions/)。
